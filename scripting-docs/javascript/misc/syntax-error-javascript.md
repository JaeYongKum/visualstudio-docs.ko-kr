# 구문 오류(JavaScript)
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]의 문법 규칙을 하나 이상 위반하는 문을 만들었습니다.  
  
### 이 오류를 해결하려면  
  
-   표시된 줄 번호에서 프로그램의 구문을 다시 한 번 확인합니다.  
  
-   방향이 잘못된 괄호나 중괄호를 찾습니다.
