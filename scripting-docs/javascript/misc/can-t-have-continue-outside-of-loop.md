# 루프 외부에서 &#39;continue&#39;를 사용할 수 없습니다.
루프 외부에서 **continue** 문을 사용하려고 했습니다.  **continue** 문은 다음 루프의 본문 내에서만 사용할 수 있습니다.  
  
-   `do-while` 루프  
  
-   `while` 루프  
  
-   **for** 루프  
  
-   **for\/in** 루프  
  
### 이 오류를 해결하려면  
  
-   **continue** 문이 다음 루프의 본문 내에 표시되는지 확인합니다.  
  
    -   `do-while` 루프  
  
    -   `while` 루프  
  
    -   **for** 루프  
  
    -   **for\/in** 루프
