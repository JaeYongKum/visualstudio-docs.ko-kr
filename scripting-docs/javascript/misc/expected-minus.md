# &#39;/&#39;가 필요합니다.
정규식 리터럴을 만들려고 했지만 슬래시\(\/\) 중 하나를 포함하지 않았습니다.  물음표 쌍 안의 문자로 작성되는 문자열 리터럴과 마찬가지로 정규식 리터럴은 슬래시\(\/\) 문자 쌍 안의 문자로 표현됩니다.  
  
### 이 오류를 해결하려면  
  
-   정규식 끝을 표시하기 위해 슬래시를 삽입하세요.
