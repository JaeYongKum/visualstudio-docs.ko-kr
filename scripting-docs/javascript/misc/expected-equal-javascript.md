# &#39;=&#39;가 필요합니다.(JavaScript)
조건부 컴파일 문에 사용할 변수를 만들려고 했지만 변수와 변수에 할당하려는 값 사이에 등호를 삽입하지 않았습니다.  
  
### 이 오류를 해결하려면  
  
-   등호를 추가하세요.  예를 들면 다음과 같습니다.  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
