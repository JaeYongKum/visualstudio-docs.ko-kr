# 숫자가 잘못되었습니다.
잘못된 숫자\(ErrbadNumber 1013\)  
  
 JavaScript에서 올바르지 않은 숫자를 사용하려고 했습니다.  
  
 올바른 숫자에 대한 내용은 [Number 개체](../../javascript/reference/number-object-javascript.md)를 참조하세요.
