# 문자열이 필요합니다.
`String` 이외의 형식으로 된 개체에서 **String.prototype.toString** 또는 **String.prototype.valueOf** 메서드를 호출하려고 했습니다.  이 호출 형식의 개체는 `String` 형식이어야 합니다.  
  
### 이 오류를 해결하려면  
  
-   `String` 형식의 개체에서 **String.prototype.toString** 또는 **String.prototype.valueOf** 메서드만 호출하세요.
