# 레이블이 다시 정의되었습니다.
새 레이블을 만들었지만 기존 레이블의 이름을 지정했습니다.  레이블은 코드 블록을 표시하는 데 사용할 수 있지만 지정된 범위 내에서 고유해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   프로그램에 사용하는 모든 레이블이 해당 범위 내에서 고유한지 확인하세요.
