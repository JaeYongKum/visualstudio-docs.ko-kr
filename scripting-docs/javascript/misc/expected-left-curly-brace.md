# <a name="expected-3939"></a>예상 &#39; {&#39;
함수 본문의 시작을 표시 하는 왼쪽된 중괄호를 포함 되지 않았습니다. 코드를 구성 하는 함수 본문을 한 줄 경우에 왼쪽 및 오른쪽 중괄호 안에 포함 되어야 합니다. 함수 본문에는 중괄호의 사용 보다 더 엄격한 루프에서 확인 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   함수 본문의 시작을 표시 하는 왼쪽된 중괄호를 추가 합니다.
