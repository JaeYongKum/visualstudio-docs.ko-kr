# &#39;{&#39;가 필요합니다.
함수 본문의 시작을 표시하는 왼쪽 중괄호를 포함하지 않았습니다.  함수의 본문을 구성하는 코드는 한 줄이더라도 왼쪽 중괄호와 오른쪽 중괄호 안에 넣어야 합니다.  함수 본문에서의 중괄호 사용은 루프의 경우보다 더 엄격합니다.  
  
### 이 오류를 해결하려면  
  
-   함수 본문의 시작을 표시하는 왼쪽 중괄호를 추가합니다.
