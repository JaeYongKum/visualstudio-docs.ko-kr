# <a name="expected-39while39"></a>예상 &#39; 동안 &#39;
포함 되지 않은 `while` 조건이 `do...while` 루프입니다. A **않습니다** 문에 해당 있어야 `while` 코드 블록의 끝을 테스트 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   포함 된 `while` 닫는 중괄호 뒤에 문을 테스트 합니다.
