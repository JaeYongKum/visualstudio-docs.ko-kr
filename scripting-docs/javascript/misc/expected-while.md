# &#39;while&#39;이 필요합니다.
`do...while` 루프에 `while` 조건을 포함하지 않았습니다.  **do** 문은 코드 블록의 끝에 해당 `while` 테스트를 포함해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   닫는 중괄호 다음에 `while` 테스트 문을 포함하세요.
