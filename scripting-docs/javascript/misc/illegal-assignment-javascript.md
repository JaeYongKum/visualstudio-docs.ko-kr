# 잘못된 할당입니다.(JavaScript)
값을 읽기 전용 식별자에 할당하려고 했습니다.  읽기 전용 식별자에 값을 할당할 수 없습니다.  예를 들면 호스트에서 정의한 개체 및 외부 COM 개체가 있습니다.  
  
### 이 오류를 해결하려면  
  
-   읽기 전용 식별자에는 값을 할당하지 마십시오.
