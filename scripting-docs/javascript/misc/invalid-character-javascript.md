# 잘못된 문자입니다.(JavaScript)
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 컴파일러에서 유효한 것으로 인식되지 않는 문자를 사용하여 식별자를 구성했습니다.  유효한 문자는 다음의 규칙을 따릅니다.  
  
-   첫 번째 문자는 ASCII 문자\(대문자 또는 소문자\) 또는 밑줄\(\_\)이어야 합니다.  
  
-   그 다음 문자는 ASCII 문자, 숫자 또는 밑줄이어야 합니다.  
  
-   예약어를 식별자 이름으로 사용할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 언어 정의의 일부가 아닌 문자는 사용하지 마십시오.
