# &#39;default&#39;는 &#39;switch&#39; 문에 한 번만 사용할 수 있습니다.
switch 문 내에서 **default** 문을 두 번 이상 사용하려고 했습니다.  기본 사례는 switch 문에서 항상 마지막 case 문이어야 합니다\(제어를 이동하는 사례\).  
  
### 이 오류를 해결하려면  
  
-   `switch` 문에서 여분의 **default** 사례를 제거하세요\(switch 문에는 기본 사례 문을 하나만 사용\).
