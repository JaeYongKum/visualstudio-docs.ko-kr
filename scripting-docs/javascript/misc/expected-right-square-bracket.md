# <a name="expected-3939"></a>예상 &#39;] &#39;
배열 요소에 대 한 참조를 했지만 오른쪽 대괄호를 포함 하지 않았습니다. 배열 요소를 참조 하는 모든 식 열고 닫는 대괄호에 포함 해야 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   배열 요소를 참조 하는 식에 오른쪽 대괄호를 추가 합니다.
