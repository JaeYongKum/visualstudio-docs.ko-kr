# &#39;]&#39;가 필요합니다.
배열 요소에 대한 참조를 만들었지만 오른쪽 대괄호를 포함하지 않았습니다.  배열 요소를 참조하는 식은 여는 대괄호와 닫는 대괄호를 모두 포함해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   배열 요소를 참조하는 식에 오른쪽 대괄호를 추가하세요.
