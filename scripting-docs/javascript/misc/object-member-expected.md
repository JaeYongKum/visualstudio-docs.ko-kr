# 개체 멤버가 필요합니다.
개체의 데이터 멤버에서 작업을 수행하려고 했지만 멤버가 존재하지 않습니다.  
  
### 이 오류를 해결하려면  
  
-   데이터 멤버의 이름을 올바르게 입력했는지 확인하세요.
