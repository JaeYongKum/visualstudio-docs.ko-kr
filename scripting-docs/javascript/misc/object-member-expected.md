# <a name="object-member-expected"></a>개체 멤버가 필요합니다.
개체의 데이터 멤버에 대 한 작업을 수행 하려고 시도 하지만 멤버가 존재 하지 않습니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   데이터 멤버의 이름을 올바르게 입력 했는지 확인 합니다.
