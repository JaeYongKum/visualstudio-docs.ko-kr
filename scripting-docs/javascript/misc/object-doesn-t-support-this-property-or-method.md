# <a name="object-doesn39t-support-this-property-or-method"></a>이 속성 또는 메서드를 지원 하지 않으면 개체 대상이 & #39
이 개체에 대해 없는 속성이나 메서드를 지정했습니다. 모든 개체에서 모든 속성과 메서드를 지원하는 것은 아닙니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   속성과 메서드에 오타가 없는지 확인합니다.  
  
-   개체 런타임 형식이 필요한 개체 형식인지 확인합니다.  
  
-   자세한 내용은 개체 설명서를 참조하세요.
