# 식별자가 필요합니다.(JavaScript)
식별자가 필요한 컨텍스트에 식별자가 아닌 다른 항목을 사용했습니다.  다음을 식별자로 사용할 수 있습니다.  
  
-   변수,  
  
-   속성,  
  
-   배열,  
  
-   또는 함수 이름  
  
### 이 오류를 해결하려면  
  
-   식별자가 등호 왼쪽에 오도록 식을 변경합니다.
