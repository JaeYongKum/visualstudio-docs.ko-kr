# 레이블을 찾을 수 없습니다.
존재하지 않는 레이블에 대한 참조를 만들었습니다.  레이블은 코드 블록을 표시하는 데 사용할 수 있지만 지정된 범위 내에서 고유해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   레이블 이름의 철자가 올바른지 확인하세요.  
  
-   모든 레이블 참조가 현재 범위에 정의된 레이블에 대한 참조인지 확인하세요\(전달 정의 포함\).
