# &#39;var&#39;이 필요합니다.
예기치 않은 'var'\(ERRnoVar 1034\)  
  
 필수인 var 키워드를 추가하지 못했습니다.  
  
 var 키워드에 대한 자세한 내용은 [var 문](../../javascript/reference/var-statement-javascript.md)을 참조하세요.
