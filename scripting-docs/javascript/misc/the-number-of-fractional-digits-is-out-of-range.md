# 소수 자릿수가 범위를 벗어났습니다.
**Number.prototype.toExponential** 함수에 잘못된 인수를 전달하려고 했습니다.  **toExponential\(\)** 함수의 인수는 0에서 20\(포함\) 사이여야 합니다.  
  
### 이 오류를 해결하려면  
  
-   **toExponential\(\)**의 인수가 너무 크거나 너무 작지 않은지 확인합니다.
