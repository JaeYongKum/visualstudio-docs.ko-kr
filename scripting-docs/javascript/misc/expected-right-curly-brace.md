# <a name="expected-3939"></a>예상 &#39;을 (를) &#39;
함수 본문, 루프, 코드 블록의 끝을 표시 하는 오른쪽 중괄호가 포함 하지 않았거나 이니셜라이저 개체 수도 있습니다. 이 오류의 예로 **에 대 한** 루프의 본문을 표시 하는 왼쪽된 중괄호만 있는 루프입니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   함수, 루프, 블록 또는 개체 이니셜라이저의 끝을 표시 하는 오른쪽 중괄호가 추가 합니다.
