# 정의되지 않은 식별자입니다.
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 컴파일러가 인식하지 않는 식별자를 사용하려고 했습니다.  다음 항목을 사용할 때마다 undefined 값이 반환됩니다.  
  
-   존재하지 않는 변수  
  
-   선언되었지만 값이 할당되지 않은 변수  
  
-   존재하지 않는 개체 속성  
  
### 이 오류를 해결하려면  
  
-   `var` x;와 같이 변수를 **var** 문으로 선언합니다
