# &#39;:&#39;이 필요합니다.
세 부분으로 구성된 조건 연산자를 사용하여 식을 만들려고 하면서 두 번째 피연산자와 세 번째 피연산자 사이에 콜론을 포함하지 않았습니다.  세 부분\(피연산자 3개\)으로 구성된 조건 연산자를 사용하려면 두 번째\(true\) 피연산자와 세 번째\(false\) 피연산자 사이에 콜론이 필요합니다.  
  
### 이 오류를 해결하려면  
  
-   두 번째 피연산자와 세 번째 피연산자 사이에 콜론을 삽입합니다.
