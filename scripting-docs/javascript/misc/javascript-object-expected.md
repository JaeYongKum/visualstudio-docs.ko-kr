# JavaScript 개체가 필요합니다.
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 개체가 필요한 기본 제공 함수에 비\-[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 개체를 전달하려고 했습니다.  호스트가 정의한 개체나 컨트롤 등의 외부 구성 요소와 달리 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]에서 정의된 개체가 필요한 기본 제공 함수가 많습니다.  
  
### 이 오류를 해결하려면  
  
-   매개 변수로 전달하는 개체가 올바른 형식인지 확인합니다.
