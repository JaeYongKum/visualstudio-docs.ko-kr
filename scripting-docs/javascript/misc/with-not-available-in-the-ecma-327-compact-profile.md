# &#39;with&#39;는 ECMA 327 Compact Profile에서 사용할 수 없습니다.
ECMA 327 Compact Profile\(ErrWithNotInCP\)에서 'with'를 사용할 수 없음  
  
 Compact Profile에서는 with 키워드를 사용할 수 없습니다.  
  
 with 키워드에 대한 자세한 내용은 [with 문](../../javascript/reference/with-statement-javascript.md)을 참조하세요.
