# 전체 자릿수가 범위를 벗어났습니다.
잘못된 인수를 **Number.prototype.toPrecision** 함수에 전달하려고 했습니다.  **toPrecision**에 대한 인수는 1에서 21 사이\(포함\)여야 합니다.  
  
### 이 오류를 해결하려면  
  
-   `toPrecision`에 대한 인수가 너무 크거나 너무 작지 않은지 확인하세요.
