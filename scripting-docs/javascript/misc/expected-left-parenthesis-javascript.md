# &#39;(&#39;가 필요합니다.(JavaScript)
괄호 집합 안에 식을 포함하려고 했지만 여는 괄호를 포함하지 않았습니다.  일부 식은 여는 괄호와 닫는 괄호의 집합으로 묶어야 합니다.  다음 예제에서 괄호 사용 방법을 확인하세요.  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### 이 오류를 해결하려면  
  
-   평가 식에 왼쪽 괄호를 추가하세요.
