# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>식별자에 대해 키워드를 잘못 사용했습니다.
1048: 키워드를 사용 하는 식별자에 대해 올바르지 않습니다.  
  
 변수, 함수 또는 다른 사용자 정의 식별자의 이름으로는 키워드를 사용할 수 없습니다.  
  
 키워드에 대 한 자세한 내용은 참조 [JavaScript 예약어](../../javascript/reference/javascript-reserved-words.md) 및 [JavaScript 미래 예약어](../../javascript/reference/javascript-future-reserved-words.md)합니다.
