# &#39;@end&#39;가 필요합니다.
조건부로 컴파일된 코드 블록을 만들려고 했지만 끝에 **@end** 문을 포함하지 않았습니다.  [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 문은 **@if\/@end** 블록으로 묶어서 조건부로 컴파일할 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   해당 **@end** 문을 추가하세요.
