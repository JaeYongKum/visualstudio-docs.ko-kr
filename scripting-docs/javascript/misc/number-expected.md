# <a name="number-expected"></a>숫자가 필요합니다.
호출 하려고는 **Number.prototype.toString** 또는 **Number.prototype.valueOf** 아닌 다른 형식의 개체에서 메서드 **번호**합니다. 이 형식의 호출 개체 유형 이어야 **번호**합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   호출할는 **Number.prototype.toString** 또는 **Number.prototype.valueOf** 형식의 개체에 대 한 메서드 **번호**합니다.
