# <a name="unterminated-string-constant-javascript"></a>종결되지 않은 문자열 상수입니다.(JavaScript)
닫는 따옴표를 사용 하면 문자열 상수를 종료 하지 않았습니다. 문자열 상수는 따옴표의 쌍 묶어야 합니다.  
  
> [!NOTE]
>  일치 하는 작은따옴표 또는 큰따옴표 쌍을 사용할 수 있습니다. 작은따옴표를로 둘러싸인 문자열에 큰따옴표를 포함할 수 있는 및 작은따옴표는 큰따옴표로 묶인 문자열 안에 포함 될 수 있습니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   문자열의 끝에 닫는 따옴표를 추가 합니다.
