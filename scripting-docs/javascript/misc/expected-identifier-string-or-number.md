# <a name="expected-identifier-string-or-number"></a>식별자, 문자열 또는 숫자가 필요합니다.
개체 리터럴 선언에 잘못 된 리터럴 구문을 사용 했습니다. 개체 리터럴 속성 식별자, 문자열 또는 숫자 여야 합니다. 개체 리터럴 ("개체 이니셜라이저"이 라고도 함) 대괄호 안에 속성: 값 쌍의 쉼표로 구분 된 목록으로 구성 합니다. 예:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   적절 한 리터럴 구문을 사용 하 여 확인 합니다.
