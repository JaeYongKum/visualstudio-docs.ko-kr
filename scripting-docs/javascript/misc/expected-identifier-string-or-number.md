# 식별자, 문자열 또는 숫자가 필요합니다.
잘못된 리터럴 구문을 사용하여 개체 리터럴을 선언했습니다.  개체 리터럴의 속성은 식별자, 문자열 또는 숫자여야 합니다.  "개체 이니셜라이저"라고도 하는 개체 리터럴은 대괄호 안에 들어 있는 쉼표로 구분된 속성 목록:값의 쌍으로 구성됩니다.  예를 들면 다음과 같습니다.  
  
```javascript  
var point = {x:1.2, y:-3.4};  
```  
  
### 이 오류를 해결하려면  
  
-   올바른 리터럴 구문을 사용해야 합니다.
