# 배열 또는 인수 개체가 필요합니다.
배열을 인수로 제공하지 않았습니다.  이 오류는 **Function.prototype.apply** 메서드에만 적용됩니다.  지정된 경우 이 함수에 대한 두 번째 인수가 `Array` 개체 또는 **Arguments** 개체여야 합니다.  
  
### 이 오류를 해결하려면  
  
-   `Array` 또는 **Arguments** 개체를 두 번째 인수로 지정하세요.
