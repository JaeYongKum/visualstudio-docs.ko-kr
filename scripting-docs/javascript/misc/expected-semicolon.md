# <a name="expected-3939"></a>예상 &#39; &#39;
둘 이상의 문을 한 줄에 배치 하려고 하지만 세미콜론으로 구분 되지 않았습니다. 세미콜론은 문을 종료 하는 데 사용 됩니다. 사용할 수 있지만 여러 문을 한 줄에, 각각을 세미콜론으로 다음 구분 되어야 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   각 문 세미콜론으로의 끝을 표시 합니다.  
  
-   함수 호출에서 괄호를 제대로 사용 해야 합니다.  
  
-   헤더 안에 세미콜론 잊지가 있는지 확인 한 **에 대 한** 루프입니다.
