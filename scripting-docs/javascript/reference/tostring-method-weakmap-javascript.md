# toString 메서드(WeakMap)(JavaScript)
`WeakMap` 개체를 나타내는 문자열을 반환합니다.  
  
## 구문  
  
```javascript  
weakmapObj.toString()  
```  
  
#### 매개 변수  
 `weakmapObj`  
 필수 요소.  `WeakMap` 개체  
  
## 속성 값\/반환 값  
 `WeakMap`의 문자열 표현입니다.
