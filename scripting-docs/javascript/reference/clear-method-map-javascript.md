# clear 메서드(Map)(JavaScript)
맵에서 요소를 모두 제거합니다.  
  
## 구문  
  
```javascript  
mapObj.clear()  
```  
  
#### 매개 변수  
 `mapObj`  
 필수 요소.  제거할 맵입니다.
