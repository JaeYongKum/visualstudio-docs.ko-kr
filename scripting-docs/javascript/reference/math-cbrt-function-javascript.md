# Math.cbrt 함수(JavaScript)
숫자의 세제곱근을 반환합니다.  
  
## 구문  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## 설명  
 `number`가 `NaN`인 경우, 결과는  `NaN`이 됩니다.
