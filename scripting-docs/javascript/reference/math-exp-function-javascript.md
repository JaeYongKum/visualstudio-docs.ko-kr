# <a name="mathexp-function-javascript"></a>Math.exp 함수(JavaScript)
E (자연 로그의 밑) 거듭제곱을 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>설명  
 필요한 `number` 인수는 숫자 식의 거듭제곱을 나타내는 *e*합니다.  
  
 반환 값은 숫자입니다. 상수 *e* 오일러의 숫자, 약 2.71828 같음 및 `number` 은 제공 된 인수입니다.
