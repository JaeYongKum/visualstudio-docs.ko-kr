# Math.exp 함수(JavaScript)
승수로 거듭제곱하는 자연 로그의 밑인 e를 반환합니다.  
  
## 구문  
  
```  
  
Math.exp(number)   
```  
  
## 설명  
 필수 `number` 인수는 *e*의 승수를 나타내는 숫자 식입니다.  
  
 반환 값이 숫자입니다.  Euler의 상수인 상수 *e*는 약 2.71828이며 `number`는 제공된 인수입니다.
