# 나머지 대입 연산자[JavaScript]
변수의 값을 식의 값으로 나누고 나머지를 변수에 할당합니다.  
  
## 구문  
  
```  
  
result %= expression  
```  
  
## 인수  
 `result`  
 모든 변수입니다.  
  
 `expression`  
 임의의 숫자 식입니다.  
  
## 설명  
 `%=` 연산자를 사용하는 것은 다음을 지정하는 것과 동일합니다.  
  
```  
result = result % expression  
```
