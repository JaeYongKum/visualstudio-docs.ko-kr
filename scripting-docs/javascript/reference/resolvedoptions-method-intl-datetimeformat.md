# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions 메서드(Intl.DateTimeFormat)
날짜/시간 포맷터 개체의 속성 및 값이 포함된 개체를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `dateTimeFormatObj`  
 `DateTimeFormat` 개체에서 정보를 검색 합니다.  
  
## <a name="remarks"></a>설명  
 반환된 된 개체의 속성의 계산 된 속성에 해당는 `DateTimeFormat` 개체입니다.
