# resolvedOptions 메서드(Intl.DateTimeFormat)
날짜\/시간 포맷터 개체의 속성 및 값이 포함된 개체를 반환합니다.  
  
## 구문  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### 매개 변수  
 `dateTimeFormatObj`  
 정보를 검색할 `DateTimeFormat` 개체입니다.  
  
## 설명  
 반환된 개체의 속성은 `DateTimeFormat` 개체의 계산 속성에 해당됩니다.
