# <a name="mathatan2-function-javascript"></a>Math.atan2 함수(JavaScript)
지점에 X 축에서 각도 (라디안)를 반환 합니다 (*y*,*x*).  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>매개 변수  
 *x*  
 필수 요소. 데카르트 x 좌표를 나타내는 숫자 식입니다.  
  
 *y*  
 필수 요소. 데카르트 y 좌표를 나타내는 숫자 식입니다.  
  
## <a name="remarks"></a>설명  
 반환 값은 사이의 *-pi* 및 *pi*합니다. 제공된 된 각도 나타냅니다 (*y*,*x*) 라디안으로 지정 합니다.
