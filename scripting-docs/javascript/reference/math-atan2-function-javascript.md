# Math.atan2 함수(JavaScript)
X축에서 점\(*y*,*x*\)까지의 각도를 라디안 값으로 반환합니다.  
  
## 구문  
  
```  
  
Math.atan2(y, x)  
```  
  
## 매개 변수  
 *x*  
 필수 요소.  x 좌표\(데카르트식\)를 나타내는 숫자 식입니다.  
  
 *y*  
 필수 요소.  y 좌표\(데카르트식\)를 나타내는 숫자 식입니다.  
  
## 설명  
 *\-pi*에서 *pi* 사이의 값이 반환됩니다.  제공된 \(*y*,*x*\) 점의 각도를 라디안 값으로 나타냅니다.
