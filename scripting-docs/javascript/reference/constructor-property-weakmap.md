# <a name="constructor-property-weakmap"></a>constructor 속성(WeakMap)
만드는 함수를 지정 된 `WeakMap` 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
weakmap.constructor  
```  
  
## <a name="remarks"></a>설명  
 필요한 `weakmap` 의 이름인는 `WeakMap` 개체입니다.  
  
 `constructor` 속성은 프로토타입이 있는 모든 개체의 프로토타입 멤버입니다. 여기에는 `Global` 및 `Math` 개체를 제외한 모든 내장 JavaScript 개체가 포함되어 있습니다. `constructor` 속성은 해당 특정 개체의 인스턴스를 구성하는 함수에 대한 참조를 포함합니다.
