# <a name="mathcosh-function-javascript"></a>Math.cosh 함수(JavaScript)
숫자의 쌍곡 코사인을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>설명  
 반환 값은 `(exp(x) + exp(-x))/2`와 동일합니다. `x`가 `NaN`인 경우, 결과는 `NaN`입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
