# <a name="mathfround-function-javascript"></a>Math.fround 함수(JavaScript)
숫자의 가장 가까운 단정밀도 부동 소수점 형식을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>설명  
 `number`가 `NaN`인 경우, 결과는 `NaN`입니다.
