# Global 개체(JavaScript)
전역 함수 및 상수를 하나의 개체로 모으는 내장 개체입니다.  
  
## 설명  
 `Global` 개체는 직접 사용할 수 없으며 `new` 연산자를 사용하여 만들 수도 없습니다.  이 개체는 스크립팅 엔진이 초기화되면 만들어지므로 이 개체의 함수 및 상수는 즉시 만들 수 있습니다.
