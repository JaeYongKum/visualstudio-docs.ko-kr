# <a name="size-property-set-javascript"></a>size 속성(Set)(JavaScript)
Set에서 요소 수를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>매개 변수  
 `sizeVar`  
 필수 요소. 모든 숫자입니다.  
  
 `setObj`  
 필수 요소. 모든 `Set` 개체입니다.
