# size 속성(Set)(JavaScript)
집합에 있는 요소 수를 반환합니다.  
  
## 구문  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### 매개 변수  
 `sizeVar`  
 필수 요소.  모든 숫자.  
  
 `setObj`  
 필수 요소.  임의의 `Set` 개체입니다.
