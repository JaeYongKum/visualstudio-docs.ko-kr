# <a name="mathfloor-function-javascript"></a>Math.floor 함수(JavaScript)
숫자 인수 보다 작거나 같은 최대 정수를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>설명  
 필요한 *번호* 인수는 숫자 식입니다.  
  
 반환 값은 숫자 인수 보다 작거나 가장 큰 정수는 정수 값입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
