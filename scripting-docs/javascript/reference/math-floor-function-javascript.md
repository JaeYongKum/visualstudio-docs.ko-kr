# Math.floor 함수(JavaScript)
숫자 인수보다 작거나 같은 가장 큰 정수를 반환합니다.  
  
## 구문  
  
```  
  
Math.floor(number)   
```  
  
## 설명  
 필수 *number*  인수는 숫자 식입니다.  
  
 반환 값은 숫자 인수보다 작거나 같은 가장 큰 정수와 같은 정수 값입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
