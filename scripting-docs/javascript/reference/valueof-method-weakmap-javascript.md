# <a name="valueof-method-weakmap-javascript"></a>valueOf 메서드(WeakMap)(JavaScript)
지정된 개체의 기본 값을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
weakmapObj.valueOf()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `weakmapObj`  
 필수 요소. `WeakMap` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 반환 된 `WeakMap`인스턴스.
