# clear 메서드(WeakMap)(JavaScript)
`WeakMap` 개체에서 요소를 모두 제거합니다.  
  
## 구문  
  
```javascript  
weakmapObj.clear()  
```  
  
#### 매개 변수  
 `weakmapObj`  
 필수 요소.  삭제할 `WeakMap` 개체입니다.
