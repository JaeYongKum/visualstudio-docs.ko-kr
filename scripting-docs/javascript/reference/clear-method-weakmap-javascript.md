# <a name="clear-method-weakmap-javascript"></a>clear 메서드(WeakMap)(JavaScript)
`WeakMap` 개체에서 요소를 모두 제거합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `weakmapObj`  
 필수 요소. `WeakMap` 지울 개체입니다.
