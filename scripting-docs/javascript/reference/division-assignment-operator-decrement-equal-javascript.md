# 나누기 할당 연산자(/=)(JavaScript)
변수 값을 식의 값으로 나누고 결과를 변수에 할당합니다.  
  
## 구문  
  
```  
  
result /= expression   
```  
  
## 매개 변수  
 `result`  
 임의의 숫자 변수입니다.  
  
 `expression`  
 임의의 숫자 식입니다.  
  
## 설명  
 `/=` 연산자를 사용하는 것은 `result = result / expression`을 지정하는 것과 동일합니다.
