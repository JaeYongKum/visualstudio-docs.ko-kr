# 할당 연산자(=)(JavaScript)
변수에 값을 할당합니다.  
  
## 구문  
  
```  
  
result = expression  
```  
  
## 매개 변수  
 `result`  
 임의의 변수입니다.  
  
 `expression`  
 임의의 숫자 식입니다.  
  
## 설명  
 \= 연산자는 다른 연산자와 비슷하게 동작하므로 이 연산자가 포함된 식에는 값이 있습니다.  따라서 `j = k = l = 0`과 같이 할당 연산자들을 연결할 수 있습니다.  이 경우 `j`, `k` 및 `l`는 0입니다.
