# <a name="valueof-method-map-javascript"></a>valueOf 메서드(Map)(JavaScript)
지정된 개체의 기본 값을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
mapObj.valueOf()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `mapObj`  
 필수 요소. `Map` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 맵 인스턴스를 반환합니다.
