# <a name="mathlog1p-function-javascript"></a>Math.log1p 함수(JavaScript)
1 + 숫자의 자연 로그를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>반환 값  
 `x` `NaN`이면, 이 함수는 `NaN`을 반환합니다. `x`가 영(0)보다 작으면 이 함수는 `NaN`을 반환합니다.
