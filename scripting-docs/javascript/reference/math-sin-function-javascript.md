# Math.sin 함수(JavaScript)
숫자의 사인 값을 반환합니다.  
  
## 구문  
  
```  
  
Math.sin(radians)   
```  
  
## 설명  
 `radians` 인수는 라디안으로 측정된 각도를 포함하는 숫자 식입니다.  
  
 반환 값은 숫자 인수의 사인 값입니다.
