# <a name="mathsin-function-javascript"></a>Math.sin 함수(JavaScript)
숫자의 사인을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>설명  
 `radians` 인수는 라디안 단위의 각도 포함 하는 숫자 식입니다.  
  
 반환 값은 숫자 인수의 사인 값.
