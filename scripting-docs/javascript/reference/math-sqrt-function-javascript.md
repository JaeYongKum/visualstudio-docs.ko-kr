# <a name="mathsqrt-function-javascript"></a>Math.sqrt 함수(JavaScript)
숫자의 제곱근을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>설명  
 필수 `number` 인수는 숫자 식입니다.  
  
 경우 `number` 가 음수 이면 반환 값은 `NaN`합니다.
