# Math.sqrt 함수(JavaScript)
숫자의 제곱근을 반환합니다.  
  
## 구문  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## 설명  
 필수 `number` 인수는 숫자 식입니다.  
  
 `number`가 음수이면 반환 값은 `NaN`입니다.
