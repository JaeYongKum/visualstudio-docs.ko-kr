# <a name="javascript-future-reserved-words"></a>JavaScript 이후 버전 예약어
JavaScript에서 미래의 예약 된 키워드 수 사용 안 식별자로 하지만 현재 버전에서 특별 한 의미를 갖습니다.  
  
 Javascript에서 예약어 목록은 참조 [JavaScript 예약어](../../javascript/reference/javascript-reserved-words.md)합니다.  
  
## <a name="future-reserved-words"></a>이후 버전 예약어  
 다음은 이후 버전 예약어입니다.  
  
1.  클래스  
  
2.  enum  
  
3.  내보내기  
  
4.  확장  
  
5.  import  
  
6.  슈퍼  
  
## <a name="future-reserved-words-in-strict-mode"></a>Strict 모드에서 이후 버전 예약어  
 다음은 strict 모드에만 이후 버전 예약어입니다. Strict 모드에 대 한 자세한 내용은 참조 [use strict 지시문](../../javascript/reference/use-strict-directive.md)합니다.  
  
1.  구현  
  
2.  interface(인터페이스)  
  
3.  패키지  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
