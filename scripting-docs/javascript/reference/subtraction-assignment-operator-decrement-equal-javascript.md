# <a name="subtraction-assignment-operator---javascript"></a>빼기 할당 연산자(-=)(JavaScript)
변수 값에서 식 값을 빼고 결과를 변수에 할당합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>매개 변수  
 *결과*  
 모든 숫자 변수입니다.  
  
 *식*  
 임의의 숫자 식입니다.  
  
## <a name="remarks"></a>설명  
 사용 하 여는 `-=` 연산자는 정확 하 게 다음과 같은 방법으로 동일 합니다.  
  
```JavaScript  
result = result - expression  
```
