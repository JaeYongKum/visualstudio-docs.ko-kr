# 빼기 할당 연산자(-=)(JavaScript)
변수의 값에서 식의 값을 빼고 결과를 변수에 할당합니다.  
  
## 구문  
  
```  
  
result -= expression  
```  
  
## 매개 변수  
 *결과*  
 임의의 숫자 변수입니다.  
  
 *expression*  
 임의의 숫자 식입니다.  
  
## 설명  
 `-=` 연산자를 사용하는 것은 다음을 수행하는 것과 동일합니다.  
  
```javascript  
result = result – expression  
```
