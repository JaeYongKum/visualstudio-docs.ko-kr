# constructor 속성(Set)
집합을 만드는 함수를 지정합니다.  
  
## 구문  
  
```javascript  
set.constructor  
```  
  
## 설명  
 필수 `set`는 설정의 이름입니다.  
  
 `constructor` 속성은 초기 설정을 가지는 모든 초기 설정 개체의 멤버입니다.  여기에는 `Global` 및 `Math` 개체를 제외한 모든 내장 JavaScript 개체가 포함됩니다.  `constructor` 속성은 특정 개체의 인스턴스를 구성하는 함수에 대한 참조를 포함합니다.
