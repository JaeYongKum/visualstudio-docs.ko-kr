# <a name="void-operator-javascript"></a>void 연산자(JavaScript)
식이 값을 반환하지 않도록 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
void expression   
```  
  
## <a name="remarks"></a>설명  
 `expression` 인수는 유효한 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 식입니다.  
  
 `void` 연산자의 식을 계산 하 고 반환 `undefined`합니다. 식을 계산할 수 있고 원하지 않는 결과 스크립트의 나머지 부분에 볼 수 있는 경우에 유용 합니다.
