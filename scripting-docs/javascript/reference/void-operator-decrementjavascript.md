# void 연산자(JavaScript)
식이 값을 반환하지 않게 합니다.  
  
## 구문  
  
```  
void expression   
```  
  
## 설명  
 `expression` 인수는 임의의 유효한 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 식입니다.  
  
 `void` 연산자는 해당 식을 계산하고 `undefined`를 반환합니다.  식을 계산해야 하지만 결과를 스크립트의 나머지 부분에 보이지 않도록 하려는 경우에 유용합니다.
