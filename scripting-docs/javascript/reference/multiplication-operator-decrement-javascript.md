# <a name="multiplication-operator--javascript"></a>곱하기 연산자(*)(JavaScript)
두 식의 값을 곱합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>매개 변수  
 *결과*  
 모든 변수입니다.  
  
 `number1`  
 임의의 식입니다.  
  
 `number2`  
 임의의 식입니다.
