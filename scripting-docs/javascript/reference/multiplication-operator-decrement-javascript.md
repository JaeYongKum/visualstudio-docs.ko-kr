# 곱하기 연산자(*)(JavaScript)
두 식의 값을 곱합니다.  
  
## 구문  
  
```  
  
result = number1*number2  
```  
  
## 매개 변수  
 *결과*  
 임의의 변수입니다.  
  
 `number1`  
 임의의 식입니다.  
  
 `number2`  
 임의의 식입니다.
