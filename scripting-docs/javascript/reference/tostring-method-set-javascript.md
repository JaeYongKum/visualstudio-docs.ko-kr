# toString 메서드(Set)(JavaScript)
집합의 문자열 표현을 반환합니다.  
  
## 구문  
  
```javascript  
setObj.toString()  
```  
  
#### 매개 변수  
 `setObj`  
 필수 요소.  `Set` 개체  
  
## 속성 값\/반환 값  
 집합의 문자열 표현입니다.
