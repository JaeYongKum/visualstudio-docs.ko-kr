# <a name="tostring-method-set-javascript"></a>toString 메서드(Set)(JavaScript)
집합의 문자열 표현을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `setObj`  
 필수 요소. `Set` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 집합의 문자열 표현입니다.
