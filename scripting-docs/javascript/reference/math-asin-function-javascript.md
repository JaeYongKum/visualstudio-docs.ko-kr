# <a name="mathasin-function-javascript"></a>Math.asin 함수(JavaScript)
숫자의 아크사인을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>설명  
 필요한 `number` 인수 아크사인 필요한 숫자 식입니다.  
  
 반환 값의 아크사인 값은 고 `number` 라디안에서의 인수입니다.
