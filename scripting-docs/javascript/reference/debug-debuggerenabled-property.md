# Debug.debuggerEnabled 속성
스크립트 컨텍스트에 대해 디버깅이 설정되었는지 확인합니다.  디버깅 사용 여부는 디버거 연결 여부에 따라 다릅니다.  
  
## 구문  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
