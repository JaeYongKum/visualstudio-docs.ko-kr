# Math.pow 함수(JavaScript)
기본 식의 값을 지정한 거듭제곱으로 반환합니다.  
  
## 구문  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## 매개 변수  
 *base*  
 필수 요소.  식의 기준 값입니다.  
  
 *exponent*  
 필수 요소.  식의 지수 값입니다.  
  
## 예제  
 다음 예제에서는 기준 지수와 같은 숫자 식이 1000을 반환합니다.  
  
```javascript  
Math.pow(10,3);  
```
