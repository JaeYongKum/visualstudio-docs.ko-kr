# resolvedOptions 메서드(Intl.Collator)
병합기 개체의 속성 및 값이 포함된 개체를 반환합니다.  
  
## 구문  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### 매개 변수  
 `collatorObj`  
 정보를 검색할 `Collator` 개체입니다.  
  
## 설명  
 반환된 개체의 속성은 `Collator` 개체의 계산된 속성에 해당됩니다.
