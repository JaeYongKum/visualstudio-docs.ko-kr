# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions 메서드(Intl.Collator)
속성 및 병합기 개체의 값을 포함 하는 개체를 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `collatorObj`  
 `Collator` 개체에서 정보를 검색 합니다.  
  
## <a name="remarks"></a>설명  
 반환된 된 개체의 속성의 계산 된 속성에 해당는 `Collator` 개체입니다.
