# Math.ceil 함수(JavaScript)
숫자 인수보다 크거나 같은 가장 작은 정수를 반환합니다.  
  
## 구문  
  
```  
  
Math.ceil(number)   
```  
  
## 설명  
 필수 `number` 인수는 숫자 식입니다.  
  
 반환 값은 숫자 인수보다 크거나 같은 가장 작은 정수와 동일한 정수 값입니다.
