# <a name="mathceil-function-javascript"></a>Math.ceil 함수(JavaScript)
숫자 인수 보다 크거나 같은 최소 정수를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>설명  
 필수 `number` 인수는 숫자 식입니다.  
  
 반환 값은 가장 작은 정수 숫자 인수 보다 크거나 같은 정수 값입니다.
