# 곱하기 할당 연산자(*=)(JavaScript)
변수 값을 식의 값으로 곱하고 결과를 변수에 할당합니다.  
  
## 구문  
  
```  
  
result *= expression  
```  
  
## 매개 변수  
 *결과*  
 임의의 변수입니다.  
  
 *expression*  
 임의의 식입니다.  
  
## 설명  
 **\*\=** 연산자를 사용하는 것은 다음을 지정하는 것과 동일합니다.  
  
```javascript  
result = result * expression  
```
