# <a name="mathatanh-function-javascript"></a>Math.atanh 함수(JavaScript)
숫자의 역쌍곡 탄젠트를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>설명  
 `x`가 `NaN`인 경우, 결과는 `NaN`입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
