# <a name="nan-constant-javascript"></a>NaN 상수(JavaScript)
ThatNot 식을 지정 하는 특수 상수 숫자가 아닙니다.  
  
## <a name="syntax"></a>구문  
  
```  
NaN   
```  
  
## <a name="remarks"></a>설명
