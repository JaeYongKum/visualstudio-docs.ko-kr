# NaN 상수(JavaScript)
식이 숫자가 아님을 지정하는 특수 상수입니다.  
  
## 구문  
  
```  
NaN   
```  
  
## 설명  
 `NaN` 상수\(숫자가 아님\)는 `Global` 개체의 멤버이고 스크립팅 엔진이 초기화되면 사용할 수 있습니다.  
  
## 요구 사항  
 `NaN` 속성은 [!INCLUDE[jsv3text](../../javascript/reference/includes/jsv3text-md.md)]에 도입되었고 [!INCLUDE[jsv9textspecific](../../javascript/reference/includes/jsv9textspecific-md.md)]에서는 읽기 전용으로 만들어졌습니다.  
  
 **적용 대상**: [Global 개체](../../javascript/reference/global-object-javascript.md)
