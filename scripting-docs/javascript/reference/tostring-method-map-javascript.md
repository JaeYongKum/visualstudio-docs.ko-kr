# toString 메서드(Map)(JavaScript)
맵의 문자열 표현을 반환합니다.  
  
## 구문  
  
```javascript  
mapObj.toString()  
```  
  
#### 매개 변수  
 `mapObj`  
 필수 요소.  `Map` 개체  
  
## 속성 값\/반환 값  
 맵의 문자열 표현입니다.
