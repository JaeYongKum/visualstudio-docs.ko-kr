# <a name="size-property-map-javascript"></a>size 속성(Map)(JavaScript)
Map에서 요소 수를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>매개 변수  
 `sizeVar`  
 필수 요소. 모든 숫자입니다.  
  
 `mapObj`  
 필수 요소. 모든 `Map` 개체입니다.
