# size 속성(Map)(JavaScript)
맵의 요소 수를 반환합니다.  
  
## 구문  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### 매개 변수  
 `sizeVar`  
 필수 요소.  모든 숫자.  
  
 `mapObj`  
 필수 요소.  임의의 `Map` 개체입니다.
