# Math.expm1 함수(JavaScript)
e\(자연 로그의 밑\) 거듭제곱에서 1을 뺀 결과를 반환합니다.\)  
  
## 구문  
  
```  
  
Math.expm1(number)   
```  
  
## 설명  
 필요한 `number` 인수는 e의 거듭제곱을 나타내는 숫자 식입니다.  
  
 반환 값은 숫자입니다.  상수 e는 약 2.71828인 오일러의 수이고 `number`은\(는\) 제공된 인수입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
