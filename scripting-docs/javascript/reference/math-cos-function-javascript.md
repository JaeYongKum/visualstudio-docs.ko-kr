# Math.cos 함수(JavaScript)
숫자의 코사인 값을 반환합니다.  
  
## 구문  
  
```  
  
Math.cos(radians)   
```  
  
## 설명  
 필수 `radians` 인수는 라디안으로 측정된 각도를 포함하는 숫자 식입니다.  
  
 반환 값은 `radians`의 숫자 인수에 대한 코사인 값입니다.
