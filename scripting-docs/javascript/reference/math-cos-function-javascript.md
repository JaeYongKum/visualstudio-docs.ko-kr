# <a name="mathcos-function-javascript"></a>Math.cos 함수(JavaScript)
숫자의 코사인을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>설명  
 필요한 `radians` 인수는 라디안 단위의 각도 포함 하는 숫자 식입니다.  
  
 반환 값의 숫자 인수의 코사인 값은 `radians`합니다.
