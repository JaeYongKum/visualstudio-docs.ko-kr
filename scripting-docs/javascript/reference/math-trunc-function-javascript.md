# <a name="mathtrunc-function-javascript"></a>Math.trunc 함수(JavaScript)
소수 자릿수를 제거하고 숫자의 정수 부분을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>설명  
 필수 `number` 인수는 잘린 정수가 필요한 숫자 식입니다.  
  
 `number`가 `NaN`이면 `NaN`을 반환합니다.
