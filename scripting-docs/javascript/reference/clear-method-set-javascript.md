# clear 메서드(Set)(JavaScript)
집합에서 요소를 모두 제거합니다.  
  
## 구문  
  
```javascript  
setObj.clear()  
```  
  
#### 매개 변수  
 `setObj`  
 필수 요소.  삭제할 집합입니다.
