# <a name="clear-method-set-javascript"></a>clear 메서드(Set)(JavaScript)
집합에서 요소를 모두 제거합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `setObj`  
 필수 요소. 설정의 선택을 취소 합니다.
