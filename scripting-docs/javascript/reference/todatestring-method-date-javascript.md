# toDateString 메서드(Date)(JavaScript)
날짜를 문자열 값으로 반환합니다.  
  
## 구문  
  
```  
  
objDate.toDateString( )  
```  
  
## 설명  
 필수 *objDate* 참조는 `Date` 개체입니다.  
  
 `toDateString` 메서드는 현재 표준 시간대의 날짜가 들어 있는 문자열 값을 편리하고 읽기 쉬운 형식으로 반환합니다.
