# <a name="todatestring-method-date-javascript"></a>toDateString 메서드(Date)(JavaScript)
문자열 값으로 날짜를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>설명  
 필요한 *objDate* 참조가 `Date` 개체입니다.  
  
 `toDateString` 읽기 쉬운 형식 메서드는 편리 하 고 현재 표준 시간대의 날짜를 포함 하는 문자열 값을 반환 합니다.
