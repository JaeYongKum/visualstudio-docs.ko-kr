# <a name="mathrandom-function-javascript"></a>Math.random 함수(JavaScript)
0에서 1 사이의 의사 난수를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>설명  
 생성 된 난수를 0 (포함)에서 1 (제외), 즉, 반환 될 수 있는 0이 것은 아니지만 항상 다 낮아야 합니다. 난수 생성기 때 시드 자동으로 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 처음 로드 합니다.
