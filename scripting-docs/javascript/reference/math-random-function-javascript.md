# Math.random 함수(JavaScript)
0과 1 사이의 의사 난수 값을 반환합니다.  
  
## 구문  
  
```  
  
Math.random( )  
  
```  
  
## 설명  
 생성되는 의사 난수 값은 0\(포함\)부터 1\(제외\)까지입니다. 따라서 반환되는 숫자에 0은 허용되지만 항상 1보다는 작습니다.  난수 생성기는 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]가 처음 로드될 때 자동으로 시드됩니다.
