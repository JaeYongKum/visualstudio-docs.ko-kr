# <a name="valueof-method-set-javascript"></a>valueOf 메서드(Set)(JavaScript)
지정된 개체의 기본 값을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
setObj.valueOf()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `setObj`  
 필수 요소. `Set` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 집합 인스턴스를 반환 합니다.
