# valueOf 메서드(Set)(JavaScript)
지정한 개체의 원시 값을 반환합니다.  
  
## 구문  
  
```javascript  
setObj.valueOf()  
```  
  
#### 매개 변수  
 `setObj`  
 필수 요소.  `Set` 개체  
  
## 속성 값\/반환 값  
 집합 인스턴스를 반환합니다.
