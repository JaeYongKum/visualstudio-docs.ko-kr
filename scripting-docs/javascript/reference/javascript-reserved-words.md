# <a name="javascript-reserved-words"></a>JavaScript 예약어
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]다양 한 식별자로 사용할 수 없는 예약 된 단어에 있습니다. 예약 된 단어에 특정 의미를 지니는 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 언어, 언어 구문의 일부인 것 처럼 합니다. 예약어를 사용 하 여 스크립트를 로드 하는 경우 컴파일 오류가 발생 합니다.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]이후 예약 된 단어 목록을 있습니다. 이러한 단어 않습니다의 일부로 현재는 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 언어에서 하지만 나중에 사용 하도록 예약 합니다. Javascript에서 이후 예약 된 키워드에 대 한 자세한 내용은 참조 [JavaScript 이후 버전 예약어](../../javascript/reference/javascript-future-reserved-words.md)합니다.  
  
 식별자를 선택할 때 것도 중요 내장 형식의 이름을 이미 있는 단어를 방지 하기 위해 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 개체 또는 함수 같은 `String` 또는 `parseInt`합니다.
