# Math.atan 함수(JavaScript)
숫자의 아크탄젠트 값을 반환합니다.  
  
## 구문  
  
```  
  
Math.atan(number)   
```  
  
## 설명  
 필수 `number` 인수는 아크탄젠트가 필요한 숫자 식입니다.  
  
 반환 값은 `number` 인수의 아크탄젠트\(라디안\)입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
