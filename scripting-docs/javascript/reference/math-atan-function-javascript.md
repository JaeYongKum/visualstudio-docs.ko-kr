# <a name="mathatan-function-javascript"></a>Math.atan 함수(JavaScript)
숫자의 아크탄젠트를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>설명  
 필요한 `number` 인수 아크탄젠트 필요한 숫자 식입니다.  
  
 반환 값은의 아크탄젠트는 `number` 라디안에서의 인수입니다.
