# 나누기 연산자(/)(JavaScript)
두 식의 값을 나눕니다.  
  
## 구문  
  
```  
  
result = number1 / number2   
```  
  
## 매개 변수  
 `result`  
 임의의 숫자 변수입니다.  
  
 `number1`  
 임의의 숫자 식입니다.  
  
 `number2`  
 임의의 숫자 식입니다.
