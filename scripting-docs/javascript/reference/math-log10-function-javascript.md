# <a name="mathlog10-function-javascript"></a>Math.log10 함수(JavaScript)
숫자의 밑이 10인 로그 값을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>반환 값  
 `x` `NaN`이면, 이 함수는 `NaN`을 반환합니다. `x`가 영(0)보다 작으면 이 함수는 `NaN`을 반환합니다.
