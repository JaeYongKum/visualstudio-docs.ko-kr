# IActiveScriptTraceInfo::StopScriptTracing 메서드
스크립트 추적을 중지 합니다.  
  
## 구문  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### 매개 변수  
 이 메서드는 매개 변수가 없습니다.  
  
## 반환 값  
 이 메서드의 가능한 반환 값은 다음과 같습니다.  
  
1.  S\_OK: 성공.  
  
2.  E\_NOTIMPL: 구현 되지 않았습니다.
