# <a name="ivariantchangetype-interface"></a>IVariantChangeType 인터페이스

## <a name="methods"></a>메서드  
 상속 된 메서드 외에도 `IUnknown`, `IVariantChangeType` 인터페이스는 다음 메서드를 노출 합니다.  
  
|메서드|설명|  
|------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Variant 값을 사용 하 고 지정 된 유형의 새 variant를 만듭니다.|
