# <a name="ivariantchangetype-interface"></a>IVariantChangeType 인터페이스
호출자를를 다른 형식에서 VARIANT 값을 변환할 수 있습니다.  
  
## <a name="methods"></a>메서드  
 상속 된 메서드 외에도 `IUnknown`, `IVariantChangeType` 인터페이스는 다음 메서드를 노출 합니다.  
  
|메서드|설명|  
|------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Variant 값을 사용 하 고 지정된 된 형식의 새 변형을 만듭니다.|
