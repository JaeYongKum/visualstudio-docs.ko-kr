# IVariantChangeType 인터페이스
VARIANT 값 형식을 변환 하 여 호출자를 허용 합니다.  
  
## 메서드  
 `IUnknown`에서 상속되는 메서드 외에 `IVariantChangeType` 인터페이스는 다음 메서드를 노출합니다.  
  
|메서드|설명|  
|---------|--------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Variant 값을 가져와 새로운 지정 된 형식으로 만듭니다.|
