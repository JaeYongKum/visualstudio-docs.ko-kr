# IDebugDocumentTextEvents::onDestroy
기본 문서 파손 되었고 더 이상 유효 하지 나타냅니다.  
  
## 구문  
  
```  
HRESULT onDestroy();  
```  
  
#### 매개 변수  
 이 메서드는 매개 변수를 사용하지 않습니다.  
  
## 반환 값  
 이 메서드는 `HRESULT`를 반환합니다.  가능한 값 포함 되지만, 다음 테이블에 제한 되지는지 않습니다.  
  
|값|설명|  
|-------|--------|  
|`S_OK`|메서드가 성공했으며|  
  
## 설명  
 이 메서드는 기본 문서 소멸 된 및 더 이상 유효 하지 나타냅니다.
