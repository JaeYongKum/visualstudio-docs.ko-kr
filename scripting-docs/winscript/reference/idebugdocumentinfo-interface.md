# IDebugDocumentInfo 인터페이스
인스턴스화할 수 있고 문서에 정보를 제공 합니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IDebugDocumentInfo` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|지정 된 문서 이름을 반환합니다.|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|문서 형식을 식별 하는 CLSID를 반환 합니다.|
