# <a name="idebugdocumentinfo-interface"></a>IDebugDocumentInfo 인터페이스
인스턴스화될 수도 있고 그렇지 않을 수도 있는 문서에 대한 정보를 제공합니다.  
  
 상속 된 메서드 외에도 `IUnknown`, `IDebugDocumentInfo` 인터페이스는 다음 메서드를 노출 합니다.  
  
## <a name="methods-in-vtable-order"></a>Vtable 순서의 메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|지정 된 문서 이름을 반환합니다.|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|문서 유형을 식별 하는 CLSID를 반환 합니다.|
