# IDebugExtendedProperty 인터페이스
확장 `IDebugProperty` 인터페이스.  
  
## Vtable 순서의 메서드  
 상속 된 메서드 외에도 `IDebugProperty`,이 인터페이스는 다음 메서드를 노출 합니다.  
  
|메서드|설명|  
|---------|--------|  
|[IDebugExtendedProperty::GetExtendedPropertyInfo](../../winscript/reference/idebugextendedproperty-getextendedpropertyinfo.md)|가져옵니다의 `ExtendedDebugPropertyInfo` 는이 설명`IDebugExtendedProperty``.`|  
|[IDebugExtendedProperty::EnumExtendedMembers](../../winscript/reference/idebugextendedproperty-enumextendedmembers.md)|확장된 속성의 구성원을 열거합니다.|
