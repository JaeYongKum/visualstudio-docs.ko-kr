# IEnumDebugExpressionContexts 인터페이스
컬렉션의 열거 `IDebugExpressionContexts` 개체입니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IEnumDebugExpressionContexts` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IEnumDebugExpressionContexts::Next](../../winscript/reference/ienumdebugexpressioncontexts-next.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 검색합니다.|  
|[IEnumDebugExpressionContexts::Skip](../../winscript/reference/ienumdebugexpressioncontexts-skip.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 건너뜁니다.|  
|[IEnumDebugExpressionContexts::Reset](../../winscript/reference/ienumdebugexpressioncontexts-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumDebugExpressionContexts::Clone](../../winscript/reference/ienumdebugexpressioncontexts-clone.md)|현재 열거자와 상태가 같은 열거자를 만듭니다.|
