# IEnumRemoteDebugApplications 인터페이스
응용 프로그램 개체를 열거합니다.  "응용 프로그램에 연결" 대화 상자는 컴퓨터에서 실행 중인 응용 프로그램을 열거 합니다.이 인터페이스를 사용할 수 있습니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IEnumRemoteDebugApplications` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IEnumRemoteDebugApplications::Clone](../../winscript/reference/ienumremotedebugapplications-clone.md)|현재 열거자와 상태가 같은 열거자를 만듭니다.|  
|[IEnumRemoteDebugApplications::Next](../../winscript/reference/ienumremotedebugapplications-next.md)|세그먼트는 열거 시퀀스에서 지정 된 횟수를 검색합니다.|  
|[IEnumRemoteDebugApplications::Reset](../../winscript/reference/ienumremotedebugapplications-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumRemoteDebugApplications::Skip](../../winscript/reference/ienumremotedebugapplications-skip.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 건너뜁니다.|
