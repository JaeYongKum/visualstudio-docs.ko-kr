# <a name="ibindeventhandler-interface"></a>IBindEventHandler 인터페이스
지정 된 이벤트 처리기를 처리 하는 개체를 제공 하는 호출자 수 있습니다.  
  
## <a name="methods"></a>메서드  
 상속 된 메서드 외에도 `IUnknown`, `IBindEventHandler` 인터페이스는 다음 메서드를 노출 합니다.  
  
|메서드|설명|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|개체에 이벤트를 바인딩합니다.|
