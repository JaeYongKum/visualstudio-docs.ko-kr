 상속 된 메서드 외에도 `IUnknown`, `IRemoteDebugApplicationThreadEx` 인터페이스는 다음 메서드를 노출 합니다.  
  
## <a name="methods-in-vtable-order"></a>Vtable 순서의 메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|이 스레드에서 실행 중인 모든 언어에 대 한 전역 식 컨텍스트를 열거 합니다.|
