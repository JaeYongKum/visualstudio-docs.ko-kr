# IRemoteDebugApplicationThreadEx 인터페이스
특정 응용 프로그램 내의 실행 스레드를 나타냅니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IRemoteDebugApplicationThreadEx` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|이 스레드에서 실행 되는 모든 언어에 대 한 전역 식 컨텍스트를 열거 합니다.|
