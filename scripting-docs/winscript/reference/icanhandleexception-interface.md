# ICanHandleException 인터페이스
스크립트 엔진의 호출자는 예외를 지정 하 여 호출자가 핸들입니다.  
  
## 메서드  
 `IUnknown`에서 상속되는 메서드 외에 `ICanHandleException` 인터페이스는 다음 메서드를 노출합니다.  
  
|메서드|설명|  
|---------|--------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|스크립트 엔진의 호출자는 지정 된 예외를 처리할 수 있는지 확인 합니다.|
