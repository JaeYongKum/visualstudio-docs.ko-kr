# <a name="icanhandleexception-interface"></a>ICanHandleException 인터페이스
예외를 지정 하는 스크립트 엔진의 호출자가 호출자가 있도록 처리 합니다.  
  
## <a name="methods"></a>메서드  
 상속 된 메서드 외에도 `IUnknown`, `ICanHandleException` 인터페이스는 다음 메서드를 노출 합니다.  
  
|메서드|설명|  
|------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|스크립트 엔진의 호출자가 지정 된 예외를 처리할 수 있는지 확인 합니다.|
