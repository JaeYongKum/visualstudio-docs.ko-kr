# SCRIPT_E_RECORDED 오류 코드
스크립트 엔진에서 호스트 사이의 오류가 전달 되었습니다.  호스트 오류 코드 호출자에 게 전달 해야 합니다.  
  
## 구문  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
