# IEnumDebugExpressionContexts::Reset
열거형 시퀀스를 처음으로 다시 설정합니다.  
  
## 구문  
  
```  
HRESULT Reset();  
```  
  
#### 매개 변수  
 이 메서드는 매개 변수를 사용하지 않습니다.  
  
## 반환 값  
 이 메서드는 `HRESULT`를 반환합니다.  가능한 값 포함 되지만, 다음 테이블에 제한 되지는지 않습니다.  
  
|값|설명|  
|-------|--------|  
|`S_OK`|메서드가 성공했으며|  
  
## 설명  
 이 메서드는 열거 시퀀스의 시작 부분으로 다시 설정합니다.
