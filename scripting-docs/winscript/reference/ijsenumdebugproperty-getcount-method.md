# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount 메서드

## <a name="syntax"></a>구문  
  
```cpp
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pCount`  
 [out] 속성의 수입니다.
