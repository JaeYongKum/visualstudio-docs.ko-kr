# IActiveScriptParseProcedure
소스 코드 텍스트 스크립트에 추가 하는 절차에 대 한 Windows 스크립트 엔진을 사용할 수 있으면 구현에서 `IActiveScriptParseProcedure` 인터페이스.  이 제공 하는 대체 메커니즘 없음 VBScript 같은 독립적인 제작 환경 해석 스크립팅 언어 \(이외의 `IActiveScriptParse` 또는 `IPersist`\*\) 프로시저 스크립트 네임 스페이스에 추가 합니다.
