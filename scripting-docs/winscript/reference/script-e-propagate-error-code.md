# SCRIPT_E_PROPAGATE 오류 코드
스크립트 오류는 다른 스레드를 호출자에 게 전파 됩니다.  호스트에서 오류 코드를 호출자에 게 전달 해야 합니다.  
  
## 구문  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
