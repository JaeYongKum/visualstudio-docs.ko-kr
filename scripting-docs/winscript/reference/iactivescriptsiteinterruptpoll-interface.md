# IActiveScriptSiteInterruptPoll 인터페이스
`IActiveScriptSiteInterruptPoll` 인터페이스는 스크립트 종료 됨을 지정 하는 데 있습니다.  
  
## 메서드  
 `IUnknown`에서 상속되는 메서드 외에도 `IActiveScriptSiteInterruptPoll` 인터페이스는 다음 메서드를 노출합니다.  
  
|메서드|설명|  
|---------|--------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|스크립트 종료 됨을 지정 하는 데가 있습니다.|
