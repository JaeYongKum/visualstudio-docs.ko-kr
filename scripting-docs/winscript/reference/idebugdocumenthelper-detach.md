# IDebugDocumentHelper::Detach
이 문서는 문서 트리에서 제거 됩니다.  
  
## 구문  
  
```  
HRESULT Detach();  
```  
  
#### 매개 변수  
 이 메서드는 매개 변수를 사용하지 않습니다.  
  
## 반환 값  
 이 메서드는 `HRESULT`를 반환합니다.  가능한 값 포함 되지만, 다음 테이블에 제한 되지는지 않습니다.  
  
|값|설명|  
|-------|--------|  
|`S_OK`|메서드가 성공했으며|  
  
## 설명  
 이 메서드는 문서 트리에서이 문서를 제거합니다.
