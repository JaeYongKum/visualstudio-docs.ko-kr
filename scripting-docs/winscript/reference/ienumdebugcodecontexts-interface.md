# IEnumDebugCodeContexts 인터페이스
문서 컨텍스트를 해당 코드 컨텍스트를 열거 합니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IEnumDebugCodeContexts` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IEnumDebugCodeContexts::Next](../../winscript/reference/ienumdebugcodecontexts-next.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 검색합니다.|  
|[IEnumDebugCodeContexts::Skip](../../winscript/reference/ienumdebugcodecontexts-skip.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 건너뜁니다.|  
|[IEnumDebugCodeContexts::Reset](../../winscript/reference/ienumdebugcodecontexts-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumDebugCodeContexts::Clone](../../winscript/reference/ienumdebugcodecontexts-clone.md)|현재 열거자와 상태가 같은 열거자를 만듭니다.|
