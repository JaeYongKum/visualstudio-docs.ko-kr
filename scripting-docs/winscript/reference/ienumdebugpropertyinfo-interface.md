# IEnumDebugPropertyInfo 인터페이스
열거 `DebugPropertyInfo` 구조.  
  
## Vtable 순서의 메서드  
 다음 표에서는 `IEnumDebugPropertyInfo`의 메서드를 보여 줍니다.  
  
|메서드|설명|  
|---------|--------|  
|[IEnumDebugPropertyInfo::Next](../../winscript/reference/ienumdebugpropertyinfo-next.md)|지정 된 수의 검색 `DebugPropertyInfo` 는 열거 시퀀스에서 구조.|  
|[IEnumDebugPropertyInfo::Skip](../../winscript/reference/ienumdebugpropertyinfo-skip.md)|지정 된 개수의 생략 `DebugPropertyInfo` 는 열거 시퀀스에서 구조.|  
|[IEnumDebugPropertyInfo::Reset](../../winscript/reference/ienumdebugpropertyinfo-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumDebugPropertyInfo::Clone](../../winscript/reference/ienumdebugpropertyinfo-clone.md)|현재 열거자와 열거 상태가 같은 열거자를 만듭니다.|  
|[IEnumDebugPropertyInfo::GetCount](../../winscript/reference/ienumdebugpropertyinfo-getcount.md)|매핑되는 `DebugPropertyInfo` 열거자에 대 한 구조.|
