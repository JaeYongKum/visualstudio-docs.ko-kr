# IEnumDebugApplicationNodes 인터페이스
응용 프로그램에 연결 된 노드의 자식 노드를 열거 합니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IEnumDebugApplicationNodes` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IEnumDebugApplicationNodes::Next](../../winscript/reference/ienumdebugapplicationnodes-next.md)|세그먼트는 열거 시퀀스에서 지정 된 횟수를 검색합니다.|  
|[IEnumDebugApplicationNodes::Skip](../../winscript/reference/ienumdebugapplicationnodes-skip.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 건너뜁니다.|  
|[IEnumDebugApplicationNodes::Reset](../../winscript/reference/ienumdebugapplicationnodes-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumDebugApplicationNodes::Clone](../../winscript/reference/ienumdebugapplicationnodes-clone.md)|현재 열거자와 상태가 같은 열거자를 만듭니다.|
