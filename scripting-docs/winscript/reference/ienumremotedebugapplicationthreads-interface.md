# IEnumRemoteDebugApplicationThreads 인터페이스
응용 프로그램에서 실행 중인 스레드를 열거합니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IEnumRemoteDebugApplicationThreads` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IEnumRemoteDebugApplicationThreads::Next](../../winscript/reference/ienumremotedebugapplicationthreads-next.md)|세그먼트는 열거 시퀀스에서 지정 된 횟수를 검색합니다.|  
|[IEnumRemoteDebugApplicationThreads::Skip](../../winscript/reference/ienumremotedebugapplicationthreads-skip.md)|열거형 시퀀스에서 지정된 수의 세그먼트를 건너뜁니다.|  
|[IEnumRemoteDebugApplicationThreads::Reset](../../winscript/reference/ienumremotedebugapplicationthreads-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumRemoteDebugApplicationThreads::Clone](../../winscript/reference/ienumremotedebugapplicationthreads-clone.md)|현재 열거자와 상태가 같은 열거자를 만듭니다.|
