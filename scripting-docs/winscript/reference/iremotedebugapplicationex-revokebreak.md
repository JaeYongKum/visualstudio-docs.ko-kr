# IRemoteDebugApplicationEx:RevokeBreak
Break 명령을 취소합니다.  
  
## 구문  
  
```  
HRESULT RevokeBreak( );  
```  
  
#### 매개 변수  
 없음  
  
## 반환 값  
 이 메서드는 `HRESULT`를 반환합니다.  가능한 값 포함 되지만, 다음 테이블에 제한 되지는지 않습니다.  
  
|값|설명|  
|-------|--------|  
|`S_OK`|메서드가 성공했으며|
