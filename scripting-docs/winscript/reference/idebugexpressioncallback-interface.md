# <a name="idebugexpressioncallback-interface"></a>IDebugExpressionCallBack 인터페이스
진행률과 관련 된 상태 이벤트를 제공 하는 `IDebugExpression` 평가 합니다.  
  
## <a name="methods"></a>메서드  
 상속 된 메서드 외에도 `IUnknown`, `IDebugExpressionCallBack` 인터페이스는 다음 메서드를 노출 합니다.  
  
|메서드|설명|  
|------------|-----------------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|식 계산 완료 되었음을 나타냅니다.|
