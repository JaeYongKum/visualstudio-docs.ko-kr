# IDebugExpressionCallBack 인터페이스
상태 이벤트의 진행에 관련 된 제공 된 `IDebugExpression` 평가.  
  
## 메서드  
 `IUnknown`에서 상속되는 메서드 외에 `IDebugExpressionCallBack` 인터페이스는 다음 메서드를 노출합니다.  
  
|메서드|설명|  
|---------|--------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|식 평가 완료 되었음을 나타냅니다.|
