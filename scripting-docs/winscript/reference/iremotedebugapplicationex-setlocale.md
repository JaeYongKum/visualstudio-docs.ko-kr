# IRemoteDebugApplicationEx:SetLocale
디버거 지역화 된 언어를 설정합니다.  
  
## 구문  
  
```  
HRESULT SetLocale(  
   DWORD  dwLangID  
);  
```  
  
#### 매개 변수  
 `dwLangID`  
 \[in\] 언어 id입니다.  
  
## 반환 값  
 이 메서드는 `HRESULT`를 반환합니다.  가능한 값 포함 되지만, 다음 테이블에 제한 되지는지 않습니다.  
  
|값|설명|  
|-------|--------|  
|`S_OK`|메서드가 성공했으며|
