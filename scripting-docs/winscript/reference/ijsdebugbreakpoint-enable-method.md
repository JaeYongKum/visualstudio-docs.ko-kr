# IJsDebugBreakPoint::Enable 메서드
중단점을 사용하도록 설정합니다.  
  
## 구문  
  
```  
HRESULT Enable(void);  
```  
  
## 반환 값  
  
## 설명  
 삭제된 중단점에서 E\_UNEXPECTED를 반환합니다.
