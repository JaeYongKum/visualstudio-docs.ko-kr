# IDebugAsyncOperationCallBack 인터페이스
상태 이벤트의 진행에 관련 된 제공 된 `IDebugAsyncOperation` 인터페이스 평가.  
  
## 메서드  
 `IUnknown`에서 상속되는 메서드 외에 `IDebugAsyncOperationCallBack` 인터페이스는 다음 메서드를 노출합니다.  
  
|메서드|설명|  
|---------|--------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|신호 결과 디버그를 비동기 작업에서 사용할 수 있습니다.|
