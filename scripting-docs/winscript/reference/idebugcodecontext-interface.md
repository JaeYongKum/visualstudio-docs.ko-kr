# IDebugCodeContext 인터페이스
실행 코드의 위치를 나타내는 추상화입니다.  
  
 `IUnknown`에서 상속되는 메서드 외에 `IDebugCodeContext` 인터페이스는 다음 메서드를 노출합니다.  
  
## Vtable 순서의 메서드  
  
|메서드|설명|  
|---------|--------|  
|[IDebugCodeContext::GetDocumentContext](../../winscript/reference/idebugcodecontext-getdocumentcontext.md)|이 코드의 컨텍스트와 관련 된 문서의 컨텍스트를 반환 합니다.|  
|[IDebugCodeContext::SetBreakPoint](../../winscript/reference/idebugcodecontext-setbreakpoint.md)|설정 하거나 여기서 코드에서 중단점을 지웁니다.|
