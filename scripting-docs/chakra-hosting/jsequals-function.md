# JsEquals 함수
두 JavaScript 값이 같은지를 비교합니다.  
  
## 구문  
  
```  
STDAPI_(JsErrorCode) JsEquals(  
   _In_ JsValueRef object1,  
   _In_ JsValueRef object2,  
   _Out_ bool *result  
);  
```  
  
#### 매개 변수  
 `object1`  
 비교할 첫 번째 개체입니다.  
  
 `object2`  
 비교할 두 번째 개체입니다.  
  
 `result`  
 값이 같은지를 나타냅니다.  
  
## 반환 값  
 작업에 성공한 경우 코드 `JsNoError`이고, 그렇지 않은 경우 오류 코드입니다.  
  
## 설명  
 이 함수는 Javascript의 `==` 연산자와 같습니다.  
  
 활성 스크립트 컨텍스트가 필요합니다.
