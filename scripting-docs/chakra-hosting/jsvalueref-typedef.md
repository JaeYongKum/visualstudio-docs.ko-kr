# <a name="jsvalueref-typedef"></a>JsValueRef Typedef
JavaScript 값에 대한 참조입니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>설명  
 JavaScript 값은 정의되지 않음, Null, 부울, 문자열, 숫자 또는 개체 형식의 값 중 하나입니다.  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** jsrt.h
