# JsPropertyIdType 열거형
JavaScript 속성의 형식 열거형입니다.  
  
## 구문  
  
```vb  
enum JsPropertyIdType;  
  
```  
  
## 멤버  
  
### 값  
  
|이름|설명|  
|--------|--------|  
|JsPropertyIdTypeString|JavaScript 문자열 속성의 형식 열거형입니다.|  
|JsPropertyIdTypeSymbol|JavaScript 기호 속성의 형식 열거형입니다.|
