# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion 열거형
런타임 버전입니다.  
  
## <a name="syntax"></a>구문  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>멤버  
  
### <a name="values"></a>값  
  
|이름|설명|  
|----------|-----------------|  
|`JsRuntimeVersion10`|IE10 버전으로 런타임을 만듭니다.|  
|`JsRuntimeVersion11`|IE11 버전으로 런타임을 만듭니다.|  
|`JsRuntimeVersionEdge`|런타임에 컴퓨터에서 가장 높은 버전으로 런타임을 만듭니다.|
