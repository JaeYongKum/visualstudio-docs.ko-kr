# <a name="jspropertyidref-typedef"></a>JsPropertyIdRef Typedef
속성 식별자입니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>설명  
 속성 식별자는 문자열을 사용하는 대신 JavaScript 개체의 속성을 참조하는 데 사용됩니다.
