# <a name="razor-support"></a>Razor 지원

Mac용 Visual Studio는 이제 IntelliSense 및 `.cshtml` 파일의 구문 강조 표시를 비롯한 Razor 편집을 공식적으로 지원합니다.

![Mac용 Visual Studio의 Razor 편집](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Mac용 Visual Studio에서 Razor 시작

Mac용 Visual Studio의 Razor를 시작할 때는 ASP.NET Core와 ASP.NET Core MVC의 Razor 페이지를 고려해야 합니다. 두 옵션에 대한 자습서와 자세한 정보를 보려면 아래 가이드 중 하나를 참조하세요.

- [Mac용 Visual Studio를 사용하여 macOS에서 ASP.NET Core의 Razor 페이지 시작](https://docs.microsoft.com/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [ASP.NET Core MVC 및 Mac용 Visual Studio 시작](https://docs.microsoft.com/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
