# <a name="editor-themes"></a>편집기 테마
**Visual Studio > 기본 설정... > 텍스트 편집기 > 색 테마** 메뉴 옵션을 통해 편집기 테마를 Mac용 Visual Studio에서 기본적으로 제공되는 색 구성표 10개 중 하나로 변경할 수 있습니다.

 ![색 테마 선택](media/source-editor-image17.png)

## <a name="adding-new-themes"></a>새 테마 추가

지원되는 형식 중 하나인 경우 새로운 색 테마를 추가할 수 있습니다. 이러한 항목은 다음과 같습니다.

* Visual Studio(`.vssettings`)
* Xamarin Studio(`.json`)
* TextMate(`.tmTheme`)

테마를 만들고 지원되는 형식으로 파일을 생성하는 데 사용할 수 있는 많은 리소스가 온라인에 제공됩니다. Visual Studio 및 TextMate 테마에 대한 무료 액세스를 제공하는 많은 갤러리도 온라인에 있습니다.

새 테마를 찾았거나 만든 후 로컬 컴퓨터로 다운로드했으면, 다음을 수행하여 추가합니다.

1. 색 테마 옵션 대화 상자에서 **추가** 단추를 선택합니다.   
    ![색 테마 추가](media/source-editor-image20.png)
2. 로컬 컴퓨터에서 테마를 찾은 다음 **열기**를 누릅니다.
3. 사전 설치된 테마 목록의 맨 아래에 테마가 추가됩니다. 미리 보려면 테마를 선택합니다. 그러면 편집기가 백그라운드 ![색 테마 미리 보기](media/source-editor-image21.png)에서 업데이트됩니다.
4. 적용하려면 **확인** 단추를 누릅니다.
