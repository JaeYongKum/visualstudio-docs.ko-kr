# <a name="get-started-on-connected-environment-with-net-core-and-visual-studio"></a>.NET Core 및 Visual Studio를 사용하여 연결된 환경에서 시작

이전 단계: [도구 설치](get-started-netcore-visualstudio-01.md)

## <a name="create-an-aspnet-web-app"></a>ASP.NET 웹앱 만들기
Visual Studio 2017 내에서 새 프로젝트 만들기에서 현재 이는 **ASP.NET Core 웹 응용 프로그램**이어야 합니다. 프로젝트 이름을 '**webfrontend**'으로 합니다.

![](images/NewProjectDialog1.png)

**웹 응용 프로그램(모델-뷰-컨트롤러)** 템플릿을 선택하고 대화 상자의 맨 위에 있는 두 개의 드롭다운 목록에서 **.NET Core** 및 **ASP.NET 코어 2.0**를 대상으로 하는지 확인합니다. **확인**을 클릭하여 프로젝트를 만듭니다.

![](images/NewProjectDialog2.png)
