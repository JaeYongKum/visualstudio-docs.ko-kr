# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>경고: 변경 사항이 적용 되지 않은 동작 구성 대화 상자

경고. 동작 구성 대화 상자에 적용되지 않은 변경 내용이 있습니다. 변경 내용을 적용하시겠습니까?

**동작 구성** 대화 상자에서는 사용할 수 있는 모든 클래스에 대 한 Insert, Update 및 Delete 동작을 구성할 수 있습니다. 이 메시지는 새 선택 하면 나타납니다 **클래스** 및 **동작** 조합 하 고 이전 변경이 아직 적용 되지 않았습니다.

## <a name="to-apply-the-change-and-continue"></a>변경 내용을 적용하고 계속하려면

- 클릭 **예** 선택한에서 변경 내용을 적용할 **클래스** 및 **동작**합니다.

## <a name="to-cancel-the-previous-change-and-continue"></a>이전 변경 내용을 취소하고 계속하려면

- 클릭 **아니요** 선택한 변경 내용을 취소 **클래스** 및 **동작**합니다.
