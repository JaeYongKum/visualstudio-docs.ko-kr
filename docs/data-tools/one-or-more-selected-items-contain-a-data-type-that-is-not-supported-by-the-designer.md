# <a name="one-or-more-selected-items-contain-a-data-type-that-is-not-supported-by-the-designer"></a>선택한 항목 중 하나 이상이 디자이너에서 지원되지 않는 데이터 형식을 포함하고 있습니다.
하나 이상의 항목을 끌 **서버 탐색기**/**데이터베이스 탐색기** 에 [!INCLUDE[vs_ordesigner_short](../data-tools/includes/vs_ordesigner_short_md.md)] 에서 지원 되지 않는 데이터 형식이 포함 되어는 [!INCLUDE[vs_ordesigner_short](../data-tools/includes/vs_ordesigner_short_md.md)] (예: [CLR 사용자 정의 형식](http://msdn.microsoft.com/Library/9f70e0b0-3a0d-4eb1-b914-07a5d0c167c2)).  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  지원되지 않는 데이터 형식이 들어 있지 않은 테이블을 기초로 뷰를 만듭니다.  
  
2.  뷰를 끌어 놓을 **서버 탐색기**/**데이터베이스 탐색기** 디자이너에 끌어 합니다.
