# DataTable 디자인
[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]에서는 데이터 테이블\(**DataTable**\)을 만들고 편집하는 데 사용하는 디자인 타임 도구를 제공합니다.  **DataTable**에 대한 개요를 보려면 [DataTables](../Topic/DataTables.md)을 참조하십시오.  
  
 다음 항목에서는 데이터 테이블을 만드는 방법, **데이터 집합 디자이너**를 사용하여 데이터 테이블을 형식화된 데이터 집합에 추가하는 방법 및 데이터 테이블을 정의하는 데이터 열\(**DataColumn**\)을 만들고 편집하는 방법에 대해 설명합니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.
