# <a name="cannot-create-an-association-ltassociation-namegt---property-listed-twice"></a>연결을 만들 수 없습니다 &lt;association 이름이&gt; -속성이 두 번 나열
연결을 만들 수 없습니다 \<연결 이름 >. 동일한 속성이 두 번 이상 나열 되었습니다: \<속성 이름 >.  
  
 연결 정의 된 선택한 **연결 속성** 에 **연결 편집기** 대화 상자. 속성은 연결의 각 클래스에 대해 한 번만 나열될 수 있습니다.  
  
 메시지의 속성은 부모 또는 자식 클래스에 대 한 번 이상 나타납니다 **연관 속성**합니다.  
  
### <a name="to-resolve-this-condition"></a>이 문제를 해결하려면  
  
-   메시지를 검사하여 메시지에 지정된 속성을 기록합니다.  
  
-   클릭 **확인** 메시지 상자를 닫습니다.  
  
-   검사는 **연관 속성** 중복 항목을 제거 하 고 있습니다.  
  
-   **확인**을 클릭합니다.
