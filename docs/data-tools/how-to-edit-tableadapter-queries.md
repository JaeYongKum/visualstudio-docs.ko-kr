# 방법: TableAdapter 쿼리 편집
**데이터 집합 디자이너**에서 [TableAdapter 쿼리 구성 마법사](../data-tools/editing-tableadapters.md)를 사용하여 TableAdapter 쿼리를 편집합니다.  TableAdapter 쿼리가 응용 프로그램의 요구에 더 이상 적합하지 않을 경우 이 쿼리를 편집하거나  TableAdapter에 추가 쿼리를 만들 수 있습니다.  새 쿼리 추가에 대한 자세한 내용은 [방법: TableAdapter 쿼리 만들기](../data-tools/how-to-create-tableadapter-queries.md)를 참조하십시오.  
  
> [!NOTE]
>  **TableAdapter 쿼리 구성 마법사** 대신 [TableAdapter 구성 마법사](../Topic/TableAdapter%20Configuration%20Wizard.md)가 열리면 TableAdapter의 추가 쿼리 중 하나를 선택하지 않고 TableAdapter의 주 `Fill` 쿼리를 선택한 것입니다.  TableAdapter의 주 `Fill` 쿼리 편집에 대한 자세한 내용은 [방법: TableAdapter 편집](../Topic/How%20to:%20Edit%20TableAdapters.md)을 참조하십시오.  
  
 ![쿼리가 여러 개 포함된 TableAdapter](../data-tools/media/tableadapter.gif "TableAdapter")  
  
### TableAdapter 쿼리를 편집하려면  
  
1.  **데이터 집합 디자이너**에서 데이터 집합을 엽니다.  자세한 내용은 [방법: 데이터 집합 디자이너에서 데이터 집합 열기](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md)를 참조하십시오.  
  
2.  편집할 TableAdapter 쿼리를 선택합니다.  
  
3.  TableAdapter 쿼리를 마우스 오른쪽 단추로 클릭하고 **구성**을 선택합니다.  
  
     **TableAdapter 쿼리 구성 마법사**가 열리고 쿼리 또는 해당 쿼리에 대한 저장 프로시저를 수정할 수 있게 됩니다.  
  
4.  원하는 변경 내용을 적용하여 **TableAdapter 쿼리 구성 마법사**를 완료합니다.  자세한 내용은 [TableAdapter 쿼리 구성 마법사](../data-tools/editing-tableadapters.md)를 참조하십시오.
