# Visual Studio에서 데이터 액세스
대부분 응용 프로그램의 주요 작업은 데이터베이스에서 정보를 읽고 업데이트하는 것입니다.  확장 가능한 분산 응용 프로그램에 데이터를 통합할 수 있도록 Visual Studio에서는 응용 프로그램에 데이터를 통합하는 기능을 지원합니다.  
  
## 단원 내용  
 [데이터 응용 프로그램 만들기](../data-tools/creating-data-applications.md)  
 응용 프로그램에 데이터 액세스 기능을 통합하는 방법에 대해 설명합니다.  
  
 [Windows Communication Foundation Services and WCF Data Services in Visual Studio](../data-tools/windows-communication-foundation-services-and-wcf-data-services-in-visual-studio.md)  
 에서 WCF\(Windows Communication Foundation\) 및 WCF Data Services 응용 프로그램을 만드는 데 사용 가능한 도구에 대해 설명합니다.  
  
 [Visual Database Tools](http://msdn.microsoft.com/ko-kr/6b145922-2f00-47db-befc-bf351b4809a1)  
 Microsoft SQL Server 및 Oracle 데이터베이스를 만들고 유지 관리하는 데 사용할 수 있는 도구에 대해 설명합니다.  
  
 [Creating and Managing Databases and Data\-tier Applications in Visual Studio](../data-tools/creating-and-managing-databases-and-data-tier-applications-in-visual-studio.md)  
 데이터 계층 응용 프로그램 및 Visual Studio의 데이터베이스 기능에 대해 설명합니다.  
  
 [차트 컨트롤](../Topic/Chart%20Controls%20for%20Integrating%20Data.md)  
 차트를 응용 프로그램에 통합하는 방법을 설명하는 정보와 항목에 대한 링크를 제공합니다.  
  
 [Microsoft Reports](../Topic/Microsoft%20Reports.md)  
 ReportViewer 컨트롤의 사용 방법을 설명하는 항목에 대한 링크를 제공합니다.  
  
## 관련 단원  
 [Windows Forms 데이터 아키텍처](../Topic/Windows%20Forms%20Data%20Binding.md)  
 Windows Forms에서 데이터 바인딩의 작동 방식과 컨트롤을 데이터 소스에 바인딩하는 방법에 대한 항목을 제공합니다.  
  
 [ADO.NET](../Topic/ADO.NET.md)  
 ADO.NET 아키텍처에 대해 설명하고, ADO.NET 클래스를 사용하여 응용 프로그램 데이터를 관리하는 방법과 데이터 소스 및 XML과 상호 작용하는 방법에 대해 설명합니다.  
  
 [ADO.NET Entity Framework](../Topic/ADO.NET%20Entity%20Framework.md)  
 개발자가 관계형 데이터베이스에 대해 직접 프로그래밍하는 대신 개념 모델에 대해 프로그래밍할 수 있도록 하는 데이터 응용 프로그램을 만드는 방법에 대해 설명합니다.  
  
 [ADO.NET Entity Data Model  Tools](http://msdn.microsoft.com/ko-kr/91076853-0881-421b-837a-f582f36be527)  
 Entity Framework에 제공되는 엔터티 데이터 모델 도구를 사용하여 응용 프로그램을 만드는 방법에 대해 설명합니다.  
  
 [WCF Data Services 4.5](../Topic/WCF%20Data%20Services%204.5.md)  
 이 [!INCLUDE[ssAstoria](../data-tools/includes/ssastoria_md.md)] 를 사용하여 [Open Data Protocol \(OData\)](http://go.microsoft.com/fwlink/?LinkID=182204)을 구현하는 인트라넷 또는 웹에 데이터 서비스를 배포하는 방법에 대해 설명합니다.  
  
 [XML 문서 및 데이터](../Topic/XML%20Documents%20and%20Data.md)  
 .NET Framework에서 XML 문서 및 데이터로 작업하는 종합적이고 통합된 클래스 집합에 대한 개요를 제공합니다.  
  
 [ASP.NET Data Access Content Map](http://msdn.microsoft.com/ko-kr/f9219396-a0fa-481f-894d-e3d9c67d64f2)  
 Web Forms 페이지에서 데이터 바인딩과 관련된 문제와 컨트롤을 데이터 소스에 바인딩하는 방법에 대한 항목을 제공합니다.  
  
 [Office 솔루션의 데이터](/office-dev/office-dev/data-in-office-solutions)

 [응용 프로그램 및 구성 요소 배포](../deployment/deploying-applications-services-and-components.md)  
 구성 요소 배포에 대한 절차, 개념 및 단계별 항목에 대한 링크를 제공합니다.  
  
 [LINQ \(Language\-Integrated Query\)](../Topic/LINQ%20\(Language-Integrated%20Query\).md)  
 C\# 및 Visual Basic에서 기본 제공되는 쿼리 기능과 관계형 데이터베이스, XML 문서, 데이터 집합 및 메모리 내 컬렉션을 쿼리하기 위한 일반적인 모델에 대해 설명합니다.  
  
 [Visual Studio의 XML 도구](../xml-tools/xml-tools-in-visual-studio.md)
