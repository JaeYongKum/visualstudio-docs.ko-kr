# <a name="how-to-add-comments-while-analyzing-a-completed-load-test"></a>방법: 완료된 부하 테스트를 분석하는 동안 주석 추가

부하 테스트가 완료된 후나 테스트 결과를 로드한 후에 부하 테스트 분석기를 사용하여 설명과 분석 주석을 부하 테스트에 추가할 수 있습니다. 설명과 분석 주석은 부하 테스트 결과에 영구적으로 저장됩니다. 자세한 내용은 [방법: 분석을 위한 부하 테스트 결과 액세스](../test/how-to-access-load-test-results-for-analysis.md)를 참조하세요.

입력하는 설명은 **부하 테스트 편집기**에서 **부하 테스트 결과 열기 및 관리** 대화 상자의 부하 테스트에 연결된 **설명** 열에도 표시됩니다. 자세한 내용은 [방법: 분석을 위한 부하 테스트 결과 액세스](../test/how-to-access-load-test-results-for-analysis.md)를 참조하세요. 또한 주석은 부하 테스트 결과에 대한 Excel 보고서를 만들 때도 표시됩니다. 자세한 내용은 [테스트 비교 또는 추세 분석을 위한 부하 테스트 결과 보고](../test/compare-load-test-results.md)를 참조하세요.

![부하 테스트 메모를 추가하기 위한 분석 대화 상자](../test/media/ltest_ananotes.png)

## <a name="to-add-a-comment-to-a-load-test"></a>부하 테스트에 주석을 추가하려면

1.  부하 테스트가 완료된 후나 테스트 결과를 로드한 후에 부하 테스트 분석기의 도구 모음에서 **분석 참고 사항 추가**를 선택합니다.

     **분석** 대화 상자가 표시됩니다.

2.  **설명** 텍스트 상자에 설명을 입력합니다.

3.  **분석** 텍스트 상자에 분석 주석을 입력합니다.

    자세한 내용은 [방법: 분석을 위한 부하 테스트 결과 액세스](../test/how-to-access-load-test-results-for-analysis.md)를 참조하세요.
