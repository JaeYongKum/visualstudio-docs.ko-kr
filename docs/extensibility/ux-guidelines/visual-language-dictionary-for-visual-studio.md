# Visual Studio에 대 한 비주얼 언어 사전
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Visual Studio의 해당도 해 및 다른 이미지의 구성 요소를 되는 개념을 설명 하기 위해 시각적인 공통 언어를 사용 합니다. 새 아이콘을 디자인할 때이 사전을 사용 하 여 사용자 고유의 이미지에 전달 하려는 개념 식별.  
  
 **일반 지침**  
  
-   해당 사전 정의 된 이러한 시각적 개념을 일관 되 게 사용 되어야 합니다. 그렇지 않은 경우, 일관 된 의미가 하기 위해 공통 요소를 예상 하는 사용자의 혼동 발생할 수 있습니다.  
  
-   이미지를 선명 하 고 쉽게 구분할 수 있도록 사용자 아이콘에서 사용 하기 위해 가장 관련이 있거나 중요 한 개념을 선택 합니다.  
  
-   각 이미지의 하나 또는 두 가지 개념을 사용 합니다.  
  
-   디자인 아이콘에 대 한 자세한 전략에 대 한 참조 [아이콘 디자인](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign)합니다.  
  
 **섹션으로 이동 합니다.**

2.  [제품 및 플랫폼](../../extensibility/ux-guidelines/visual-language-dictionary-for-visual-studio.md#BKMK_VLDProducts)  
  
3.  [화살표를 사용 하는 일반적인 개념](../../extensibility/ux-guidelines/visual-language-dictionary-for-visual-studio.md#BKMK_VLDArrows)
