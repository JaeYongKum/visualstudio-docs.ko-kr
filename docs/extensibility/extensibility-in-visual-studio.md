# <a name="extensibility-in-visual-studio"></a>Visual Studio의 확장성
Visual Studio 확장성 옵션을 많이 제공합니다. 수 고유한 Sdk, Windows, Windows Phone 및 (Visual Studio의 일부로 설치 되어 있음)는 Azure Sdk를 사용 하 여 만들고 자체는 Visual Studio를 확장 합니다.  
  
## <a name="extend-visual-studio"></a>Visual Studio 확장  
 Visual Studio의 거의 모든 부분을 확장 하는 Visual Studio SDK를 사용할 수 있습니다: 명령, 메뉴, windows, 편집자 및 프로젝트입니다. 자세한 내용을 참조 하십시오 [Visual Studio SDK](../extensibility/visual-studio-sdk.md)합니다.  
  
## <a name="create-your-own-sdks"></a>사용자 고유의 Sdk 만들기  
 패키지를 만들고, 고유한 플랫폼 및 확장 Sdk를 배포 하는 방법을 보려면: [소프트웨어 개발 키트를 만드는](../extensibility/creating-a-software-development-kit.md)합니다.  
  
## <a name="roslyn-extensibility"></a>Roslyn 확장성  
 Roslyn 확장성을 사용 하는 방법에 대해 알아봅니다: [.NET 컴파일러 플랫폼 ("Roslyn") 확장성](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md)합니다.
