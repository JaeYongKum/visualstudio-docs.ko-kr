# 도구 모음에 명령 추가
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

도구 모음은 명령에 연결 하는 단추가 포함 된 가로 또는 세로 줄무늬입니다. Visual Studio에서 여러 가지 창에 있는 도구 모음에 직접 명령을 추가할 수 있습니다: Visual Studio의 주 창, 솔루션 탐색기에서 솔루션 및 사용자 지정 도구 창을 직접.  
  
 이 섹션에서는 다양 한 도구 모음에 명령으로 수행할 수 있는 작업을 보여 줍니다.
