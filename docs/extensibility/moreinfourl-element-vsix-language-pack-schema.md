# MoreInfoURL 요소 (VSIX 언어 팩 스키마)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

선택적 요소. 링크는 확장에 대 한 지역화 된 정보입니다.  
  
## 구문  
  
```  
<MoreInfoURL>URL</MoreInfoURL>  
```  
  
## 특성 및 요소  
 다음 단원에서는 특성, 자식 요소 및 부모 요소에 대해 설명합니다.  
  
### 특성  
  
|특성|설명|  
|--------|--------|  
|없음||  
  
### 자식 요소  
  
|요소|설명|  
|--------|--------|  
|없음||  
  
### 부모 요소  
  
|요소|설명|  
|--------|--------|  
|[VSIX 언어 요소](../extensibility/vsixlanguagepack-element-vsix-language-pack-schema.md)|필수 요소. VSIX 언어 팩에 대 한 루트 요소를 제공 합니다.|  
  
## 텍스트 값  
 선택적 요소. 웹 사이트에 연결 합니다. 링크는 텍스트 문자열입니다.
