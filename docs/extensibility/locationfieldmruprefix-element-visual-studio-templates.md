# LocationFieldMRUPrefix 요소(Visual Studio 템플릿)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**새 프로젝트** 및 **새 항목 추가** 대화 상자에 MRU\(가장 최근에 사용한\) 경로를 지정합니다.  
  
## 구문  
  
```  
<LocationFieldMRUPrefix> ... </LocationFieldMRUPrefix>  
```  
  
## 특성 및 요소  
 다음 단원에서는 특성, 자식 요소 및 부모 요소에 대해 설명합니다.  
  
### 특성  
 없음  
  
### 자식 요소  
 없음  
  
### 부모 요소  
  
|요소|설명|  
|--------|--------|  
|[TemplateData](../extensibility/templatedata-element-visual-studio-templates.md)|템플릿을 분류하고 **새 프로젝트** 또는 **새 항목 추가** 대화 상자에 템플릿을 표시하는 방법을 정의합니다.|  
  
## 설명  
 이 요소는 [!INCLUDE[vsipprvsip](../extensibility/includes/vsipprvsip_md.md)]을 통해 생성되는 템플릿에만 사용해야 합니다.
