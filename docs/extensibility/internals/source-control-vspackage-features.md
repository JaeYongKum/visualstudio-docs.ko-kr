# 소스 제어 VSPackage 기능
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

이 섹션에서는 VSPackage 원본 컨트롤의 다양 한 기능에 설명합니다.  등록에 간략하게 설명 하 고 선택 영역이 이러한 Vspackage를 자세히 설명 하 고 3 주 소스 제어와 관련 된 기능에 설명 합니다: 소스 제어 기능에 대 한 사용자 지정 사용자 인터페이스 \(UI\), 문자 모양 대체, 및 쿼리 편집 쿼리 저장 \(QEQS\) 이벤트를 처리 합니다.  
  
## 단원 내용  
 [등록 및 선택](../../extensibility/internals/registration-and-selection-source-control-vspackage.md)  
 패키지 등록 및 선택 메커니즘을 설명합니다.  
  
 [쿼리 저장 쿼리 편집](../../extensibility/internals/query-edit-query-save-source-control-vspackage.md)  
 역할을 편집 쿼리 쿼리 저장 이벤트와 VSPackage 소스 컨트롤에 의해 처리 되는 방법을 설명 합니다.  
  
 [문자 모양 제어](../../extensibility/internals/glyph-control-source-control-vspackage.md)  
 문자 모양 제어 수준과 구현 방법을 설명 합니다.  
  
 [사용자 지정 사용자 인터페이스](../../extensibility/internals/custom-user-interface-source-control-vspackage.md)  
 소스 제어 VSPackage 지정할 수 있는 UI 요소에 간략하게 설명 합니다.  
  
## 관련 단원
