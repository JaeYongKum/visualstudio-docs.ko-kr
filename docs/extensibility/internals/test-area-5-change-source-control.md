# <a name="test-area-5-change-source-control"></a>테스트 영역 5: 소스 제어 변경
이 소스 제어 플러그 인 테스트 기능은 통한 소스 제어 변경는 **소스 제어 변경** 명령입니다.  
  
 **소스 제어 변경** 명령은 사용자에 대 한 네 가지 기본 기능을 제공 합니다.  
  
-   **바인딩:**  
  
     설정 하거나 솔루션/프로젝트와 버전 저장소 사이 소스 제어 연결을 다시 설정 하는 사용자를 있습니다.  
  
-   **바인딩 해제 합니다.**  
  
     각 연결 단위로 소스 제어에서 프로젝트/솔루션을 제거합니다.  
  
-   **연결/연결 끊기:**  
  
 영역 3에서 제어 되는 솔루션의 설정/해제 연결 또는 오프 라인 상태입니다. 자세한 내용은 참조 [테스트 영역 3: 체크 아웃 / 체크 아웃 취소](../../extensibility/internals/test-area-3-check-out-undo-checkout.md)합니다.  
  
## <a name="command-menu-access"></a>명령 메뉴 액세스  
 다음 [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] 통합된 개발 환경 메뉴 경로 테스트 사례에서 사용 됩니다.  
  
 소스 제어 변경:**파일**, **소스 제어**, **소스 제어 변경**합니다.
