# 웹 프로젝트
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

웹 프로젝트 시스템 프로젝트 웹 프로젝트를 만드는 시스템이입니다.  웹 프로젝트에 웹 응용 프로그램을 만듭니다.  같은 기존의 프로그래밍 언어를 사용 하 여 웹 페이지를 프로그래밍할 수 있습니다 [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] 또는 [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)], 수집 하 고, 사용자에서 정보를 처리, 데이터베이스에 저장 하.  웹 페이지 관련 된 코드와 스마트 웹 페이지 라고 합니다.  
  
## 단원 내용  
 [Essentials](../../extensibility/internals/web-project-essentials.md)  
 중요 한 요소를 표시 합니다. [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] 웹 프로젝트입니다.  
  
 [웹 사이트 지원](../../extensibility/internals/web-site-support.md)  
 개요를 표시 합니다. [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] 웹 사이트 프로젝트.  웹 사이트 프로젝트를 각 스마트 웹 페이지에 대 한 하나의 실행 파일을 생성합니다.  추가 실행 파일 \/App\_Code 폴더의 소스 코드 파일에서 생성 됩니다.
