# 색 테마 도구
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

VS SDK 만들고 색 테마를 관리할 수 있도록 여러 가지 도구를 제공 합니다.  
  
-   [VSIX 색 편집기](../../extensibility/internals/vsix-color-editor.md) 도구를 만들어 Visual Studio에 대 한 사용자 지정 색을 편집 합니다.  
  
-   [VSIX 색 컴파일러](../../extensibility/internals/vsix-color-compiler.md) 도구.pkgdef 확장 프로그램에 대해를 기존 Visual Studio 테마.xml 파일을 변환 합니다.
