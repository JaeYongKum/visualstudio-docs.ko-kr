# Visual Studio 디버깅 샘플
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Visual Studio 디버거를 사용 하 여 확장 하는 두 가지 예제를 보여 줍니다 [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)].  
  
-   관리 되는 언어에 대 한 디버거 식 계산기 샘플에 포함 되어 있는 [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)]. 에 [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] 창 클릭의  **샘플** 탭에서 클릭  **모든** 또는  **C\#** 언어를 클릭 하 고 목록에서  **디버거**.  
  
-   Visual Studio 디버그 엔진 샘플, 기본 언어에 대 한 MSDN 코드 갤러리에서 사용할 수 있습니다.  에 대 한 자세한 내용은 Microsoft 웹 사이트에서이 페이지를 참조 하십시오: [Visual Studio 디버깅 엔진 샘플](란?LinkId%20=%20150236).
