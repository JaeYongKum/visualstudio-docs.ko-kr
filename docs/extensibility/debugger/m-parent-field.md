# <a name="mparent-field"></a>m_parent 필드
에 대 한 지원 필드를 <xref:System.Threading.Tasks.Task?displayProperty=fullName> `Parent` 속성입니다.  
  
 **네임스페이스:** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **어셈블리:** mscorlib (에서 *mscorlib.dll*)  
  
 .NET Framework에서이 내부 멤버에 액세스할 수 없는 때문에 다음 구문은 공통 중간 언어 (CIL) 제공 됩니다.  
  
## <a name="syntax"></a>구문  
  
```csharp  
.field private initonly class System.Threading.Tasks.Task m_parent  
```
