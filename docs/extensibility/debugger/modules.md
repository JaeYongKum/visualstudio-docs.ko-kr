# 모듈
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

디버거 아키텍처 측면에서  **모듈**:  
  
-   코드를 실행 파일 또는 DLL의 물리적 컨테이너가입니다.  
  
-   해당 기호를 다시 로드 하 고 자체 설명 있습니다.  모듈 설명 IDE의 모듈 창에 표시 됩니다.  
  
-   표시 되는  [IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md) 모듈을 설명 하는 디버그 엔진에서 만든 인터페이스를.
