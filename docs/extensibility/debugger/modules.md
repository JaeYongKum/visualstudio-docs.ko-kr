# <a name="modules"></a>모듈
디버거 아키텍처를 기준으로 한 **모듈**:  
  
-   실행 파일이 나 DLL 등의 물리적 컨테이너가입니다.  
  
-   수 해당 기호를 다시 로드 되 고 자체에 대해 설명 합니다. 모듈 설명 IDE의 모듈 창에 표시 됩니다.  
  
-   로 표시 됩니다는 [IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md) 인터페이스를 모듈을 설명 하기 위해 디버그 엔진에서 만들어집니다.
