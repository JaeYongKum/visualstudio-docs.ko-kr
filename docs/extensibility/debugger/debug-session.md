# <a name="debug-session"></a>디버그 세션
디버거 아키텍처를 기준으로 한 **세션**:  
  
-   관련 또는 관련 없는 모든 집합이 처리, 시작 또는 IDE의 단일 인스턴스에서 연결 된입니다.  
  
-   세션 디버그 관리자 (SDM)에 의해 구현 됩니다. 자세한 내용은 참조 [세션 디버그 관리자](../../extensibility/debugger/session-debug-manager.md)합니다.
