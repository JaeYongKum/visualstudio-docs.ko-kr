# <a name="stepunit"></a>STEPUNIT
단계별 코드 실행 단계 단위를 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
typedef DWORD STEPUNIT;  
```  
  
```csharp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
```  
  
## <a name="members"></a>멤버  
 STEP_STATEMENT  
 문에 의해 단계입니다.  
  
 STEP_LINE  
 줄이 단계를 수행 합니다.  
  
 STEP_INSTRUCTION  
 명령에 의해 단계입니다.  
  
## <a name="remarks"></a>설명  
 에 대 한 인수로 전달 되는 [단계](../../../extensibility/debugger/reference/idebugprocess3-step.md) 메서드.

 헤더: msdbg.h  
  
 Namespace: Microsoft.VisualStudio.Debugger.Interop  
  
 Assembly: Microsoft.VisualStudio.Debugger.Interop.dll
