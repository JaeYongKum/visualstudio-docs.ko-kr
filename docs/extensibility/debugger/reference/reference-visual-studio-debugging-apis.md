# 참조 (Visual Studio 디버깅 Api)
[!INCLUDE[vs2017banner](../../../code-quality/includes/vs2017banner.md)]

Visual Studio SDK 디버깅 참조 런타임 아키텍처와 기능을 디버거 및 해당 구성 요소에 대 한 기본 모델을 요약합니다.  설명서는 몇 가지 가능한 구성 요소와 인터페이스를 정의 하는 인터페이스를 통해 동작 별도로 설명 하는 방법에 대 한 가정 합니다.  
  
 참조 섹션의 목적은 가능한 집합 언어, 실행된 시간 및 운영 체제의 지원으로 다양 한 이름으로 제공 하는입니다.
