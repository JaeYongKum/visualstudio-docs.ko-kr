# <a name="namematch"></a>NAME_MATCH
이름 일치에 대 한 사례 옵션을 선택 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>멤버  
 nmNone  
 지정된 옵션이 없습니다.  
  
 nmCaseSensitive  
 일치 시킬 이름 대/소문자 구분 함을 나타냅니다.  
  
 nmCaseInsensitive  
 일치 시킬 이름이 대/소문자 구분 되지 않는지를 나타냅니다.
