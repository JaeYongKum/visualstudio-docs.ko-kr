# <a name="bplocationresolution"></a>BP_LOCATION_RESOLUTION
특정 위치에 있는 중단점의 해결 방법을 설명합니다.

## <a name="syntax"></a>구문

```cpp
typedef struct _BP_LOCATION_RESOLUTION {
    IDebugBreakpointResolution2* pResolution;
} BP_LOCATION_RESOLUTION;
```

## <a name="members"></a>멤버
