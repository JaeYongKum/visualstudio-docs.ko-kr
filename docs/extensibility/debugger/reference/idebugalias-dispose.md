# <a name="idebugaliasdispose"></a>IDebugAlias::Dispose
제거를 위해이 별칭을 표시합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Dispose();  
```  
  
```csharp  
int Dispose();  
```  
  
#### <a name="parameters"></a>매개 변수  
 없음  
  
## <a name="return-value"></a>반환 값  
 성공 하면 s_ok이 고; 반환 그렇지 않은 경우 오류 코드를 반환합니다.  
  
## <a name="remarks"></a>설명  
 이 메서드가 호출 되 면 별칭을 사용할 수 없습니다.
