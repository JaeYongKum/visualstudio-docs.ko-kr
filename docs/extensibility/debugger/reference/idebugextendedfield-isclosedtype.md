# <a name="idebugextendedfieldisclosedtype"></a>IDebugExtendedField::IsClosedType
필드 폐쇄형된 형식을 나타내는지 여부를 확인 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT IsClosedType(  
   void  
);  
```  
  
```csharp  
int IsClosedType();  
```  
  
## <a name="return-value"></a>반환 값  
 필드 닫힌 형식인 경우 반환 `S_OK`, 그렇지 않으면 반환 `S_FALSE`합니다.
