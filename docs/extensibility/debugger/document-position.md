# 문서 위치
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] 디버깅은  **문서 위치**:  
  
-   IDE로 알려진 소스 파일에서 위치를 추상화를 제공 합니다.  대부분의 언어에 대 한 오늘날, 문서 위치의 위치는 소스 파일의 이름으로 생각할 수 있습니다.  
  
-   디버그 엔진 소스 문서의 위치를 설명합니다.  
  
-   에 의해 구현 되는  [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md) 인터페이스입니다.
