# <a name="document-position"></a>문서 위치
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] 을 디버깅 하는 **위치 문서**:  
  
-   IDE에 알려진 소스 파일의 위치에 대 한 추상화를 제공 합니다. 대부분의 언어에 대 한 현재 문서 위치 수으로 생각할 수 소스 파일의 위치입니다.  
  
-   디버그 엔진에 소스 문서의 위치를 설명합니다.  
  
-   에 의해 구현 된 [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md) 인터페이스입니다.
