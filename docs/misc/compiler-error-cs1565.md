# 컴파일러 오류 CS1565
Win32 리소스 파일과 Win32 아이콘 옵션은 서로 충돌하므로 함께 지정할 수 없습니다.  
  
 동일한 컴파일에서 [\/win32res](/dotnet/csharp/language-reference/compiler-options/win32res-compiler-option) 컴파일러 옵션 및 [\/win32ico](/dotnet/csharp/language-reference/compiler-options/win32icon-compiler-option) 컴파일러 옵션 둘 다를 지정하는 것은 유효하지 않습니다.
