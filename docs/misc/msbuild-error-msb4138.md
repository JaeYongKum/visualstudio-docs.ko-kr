# MSBuild 오류 MSB4138
**MSB4138: 레지스트리 위치 "'\<key\>'"에 문자열이 아닌 데이터가 지정되었습니다.**  
  
 사용자 지정 도구 집합 구성에 지정된 레지스트리 위치가 잘못되었습니다.  
  
### 이 오류를 해결하려면  
  
-   도구 집합 정의가 올바르고 지정된 레지스트리 키가 있는지 확인합니다.
