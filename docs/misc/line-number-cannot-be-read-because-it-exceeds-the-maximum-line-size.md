# 최대 줄 크기를 초과하기 때문에 줄 &lt;number&gt;를 읽을 수 없습니다.
최대 줄 크기를 초과하기 때문에 줄을 읽을 수 없습니다. 일반적으로 파일에 여는 따옴표가 있지만 닫는 따옴표가 없음을 의미합니다.  
  
### 이 오류를 해결하려면  
  
-   파일에 닫는 따옴표를 삽입합니다.
