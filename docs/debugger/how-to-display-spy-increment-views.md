# How to: Display Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+를 시작하면 해당 컴퓨터에서 사용할 수 있는 모든 창 및 컨트롤의 트리 뷰인 [창 뷰](../debugger/windows-view.md)에서 Spy\+\+가 열립니다.  이 루트 창에는 "창 1"이라는 캡션이 지정됩니다. Spy\+\+에는 이외에도 [메시지 뷰](../debugger/messages-view.md), [프로세스 뷰](../debugger/processes-view.md) 및 [스레드 뷰](../debugger/threads-view.md)라는 세 개의 뷰가 더 있습니다.  **Spy** 메뉴에서 원하는 뷰를 표시할 수 있습니다.  자세한 내용은 [Spy\+\+ 뷰](../debugger/spy-increment-views.md)를 참조하십시오.
