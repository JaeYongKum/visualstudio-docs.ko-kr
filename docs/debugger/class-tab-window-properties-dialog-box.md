# 창 속성 대화 상자, 클래스 탭
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**클래스** 탭을 사용하여 선택된 창의 클래스에 대한 정보를 표시할 수 있습니다.  [창 속성 대화 상자](../debugger/window-properties-dialog-box.md)를 표시하려면 포커스를 [창 뷰](../debugger/windows-view.md) 창으로 이동합니다.  그런 다음 트리에서 창 노드를 선택하고 **보기** 메뉴에서 **속성**을 선택합니다.  
  
 **클래스** 탭에서 사용할 수 있는 설정은 다음과 같습니다.  
  
|Entry|설명|  
|-----------|--------|  
|**클래스 이름**|이 창 클래스의 이름\(또는 서수\)입니다.|  
|**클래스 스타일**|클래스 스타일 코드의 조합입니다.|  
|**클래스 바이트**|이 창 클래스와 연결된 응용 프로그램 관련 데이터입니다.|  
|**클래스 아톰**|**RegisterClass** 호출에 의해 반환되는 클래스의 아톰입니다.|  
|**인스턴스 핸들**|클래스를 등록한 모듈의 인스턴스 핸들입니다.  인스턴스 핸들은 고유하지 않습니다.|  
|**창 바이트**|이 클래스의 각 창과 연결된 추가 바이트 수입니다.  이러한 바이트의 의미는 응용 프로그램에 따라 달라집니다.  바이트 값을 DWORD 형식으로 표시하려면 목록 상자를 확장합니다.|  
|**창 프로시저**|이 클래스의 창에 대한 **WndProc** 함수의 현재 주소입니다.  창이 서브클래싱된 경우 이 프로시저는 **일반** 탭의 **창 프로시저**와 다릅니다.|  
|**메뉴 이름**|이 클래스의 창과 연결된 주 메뉴의 이름입니다. 메뉴가 없는 경우에는 해당되지 않습니다.|  
|**아이콘 핸들**|이 클래스의 창과 연결된 아이콘의 핸들입니다. 아이콘이 없는 경우에는 해당되지 않습니다.|  
|**커서 핸들**|이 클래스의 창과 연결된 커서의 핸들입니다. 커서가 없는 경우에는 해당되지 않습니다.|  
|**배경 브러시**|이 클래스의 창과 연결된 배경 브러시의 핸들이거나 창 배경을 칠하는 데 사용되는 미리 정의된 COLOR\_\* 색 중 하나입니다. 브러시가 없는 경우에는 해당되지 않습니다.|
