### <a name="to-open-the-window-properties-dialog-box"></a>창 속성 대화 상자를 열려면  
  
1.  항목을 선택한 후 선택 **속성** 에서 **보기** 메뉴.  
  
 속성 대화 상자 모달있지 않습니다. 보기 창에서 항목을 클릭 하면 대화 상자가 자동으로 새로 고쳐집니다 선택 된 각 항목에 대 한 정보입니다.
