# <a name="how-to-specify-debugger-settings"></a>방법: 디버거 설정 지정
[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]에서는 변수 표시 방법, 특정 경고 표시 여부, 중단점 설정 방법, 프로그램 중단이 실행 프로그램에 미치는 영향 등을 포함하여 디버거 작동에 대한 여러 가지 설정을 지정할 수 있습니다. 디버거 설정에 지정 된 **옵션** 대화 상자.  
  
### <a name="to-set-debugger-options"></a>디버거 옵션을 설정하려면  
  
1.  **도구** 메뉴에서 **옵션**을 클릭합니다.  
  
2.  에 **옵션** 열기 대화 상자는 **디버깅** 폴더입니다.  
  
3.  에 **디버깅** 폴더에서 원하는 옵션 범주를 선택 합니다.  
  
     가장 일반적인 옵션에는 **일반** 범주입니다. 자세한 내용은 참조 [옵션 대화 상자, 디버깅, 일반](../debugger/general-debugging-options-dialog-box.md)합니다.  
  
4.  원하는 옵션을 선택하거나 해제합니다. 옵션에 대한 도움말을 보려면 F1 키를 누르십시오.
