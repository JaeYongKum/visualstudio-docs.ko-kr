# <a name="find-window-dialog-box"></a>창 찾기 대화 상자
이 대화 상자를 사용 하 여 창을 선택 하 고 해당 속성이 나 메시지를 표시 합니다. 이 대화 상자를 표시 하려면 선택 **창 찾기** 에서 **Spy** 메뉴.  
  
 다음 설정을 사용할 수 있습니다.  
  
 **찾기 도구**  
 끌어서 놓기 도구 창을 선택 하는 데 사용 합니다.  
  
 **Spy + + 숨기기**  
 Spy + + 주 창을 제거는 **창 찾기** 대화 상자를 다른 응용 프로그램 위에 표시 합니다.  
  
 **핸들**  
 창의 고유 ID를 찾을 수 있습니다.  
  
 **캡션**  
 창 캡션의 텍스트입니다.  
  
 **클래스**  
 창 클래스입니다.  
  
 **스타일**  
 창 스타일입니다.  
  
 **Rect**  
 창에 대 한 경계 사각형입니다.  
  
 **속성 또는 메시지 표시**  
 확인을 누르면를 표시할지 여부를 결정은 [창 속성 대화 상자](../debugger/window-properties-dialog-box.md) 열거나는 [메시지 뷰](../debugger/messages-view.md)합니다.
