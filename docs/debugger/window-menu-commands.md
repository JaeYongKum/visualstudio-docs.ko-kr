# <a name="window-menu-commands"></a>창 메뉴 명령
**창** 메뉴는 Spy + + 창 관리 명령을 제공 합니다.  
  
 **계단식 배열**  
 모든 창을 계단식으로 다시 정렬 합니다.  
  
 **가로 바둑판식 배열**  
 모든 창을 가로 바둑판식으로 다시 정렬 합니다.  
  
 **세로 바둑판식 배열**  
 모든 창을 세로 바둑판식으로 다시 정렬합니다.  
  
 **아이콘 정렬**  
 창이 최소화 된 경우 뷰 아이콘, 다시 정렬 합니다.  
  
 **닫기 / 모두 닫기**  
 Spy + +에서 하나 또는 모든 창을 닫습니다.  
  
 **새로 고침**  
 현재 선택 된 보기를 새로 고칩니다.  
  
 **창**  
 특정 창을 선택할 수 있습니다.
