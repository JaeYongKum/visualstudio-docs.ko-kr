# <a name="idiasymbolgetuavslot"></a>IDiaSymbol::get_uavSlot
Uav 슬롯을 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```C++  
HRESULT get_uavSlot(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pRetVal`  
 [out] 에 대 한 포인터를 `DWORD` uav 슬롯을 보유 하는 합니다.  
  
## <a name="return-value"></a>반환 값  
 성공 하면 반환 `S_OK`이 고, 그렇지 않으면 반환 `S_FALSE` 또는 오류 코드입니다.
