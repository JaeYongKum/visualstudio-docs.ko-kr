# <a name="how-to-expand-and-collapse-spy-trees"></a>방법: Spy++ 트리 확장명 및 축소
확장 및 두 가지 방법으로 창, 프로세스 및 스레드 뷰를 축소할 수 있습니다: 창이 나 사용 하 여 아이콘을 클릭 하 여는 **트리** 메뉴. + 및-트리에서 아이콘 Visual c + + 프로젝트 창에서와 마찬가지로 작동 합니다.  
  
 **트리** 메뉴 4 개의 명령이 포함 됩니다.  
  
|메뉴 명령|설명|  
|------------------|-----------------|  
|**한 수준 확장**|다음 수준으로 현재 선택 된 항목을 확장합니다.|  
|**분기를 확장 합니다.**|현재 선택한 항목을 완전히 확장합니다.|  
|**모두 확장**|창에서 모든 항목이 완전히 확장합니다.|  
|**축소**|현재 선택한 항목을 완전히 축소합니다.|  
  
> [!TIP]
>  프로세스를 확장 하는 프로세스에서 소유 하 고 모든 스레드가 표시 됩니다. 스레드를 확장 하는 경우 소유 하는 모든 windows 목록이 표시 됩니다.  
  
### <a name="to-expand-or-collapse-spy-trees"></a>확장 또는 Spy + + 트리를 축소 하려면  
  
1.  Windows, 프로세스 또는 스레드 보기에서 항목 중 하나를 강조 표시 합니다.  
  
2.  **트리** 메뉴 확장 중 하나를 선택 하거나 명령을 축소 합니다.
