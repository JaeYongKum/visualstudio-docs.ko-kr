# 창 속성 대화 상자, 일반 탭
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**일반** 탭을 사용하여 선택된 창에 대한 정보를 표시할 수 있습니다.  [창 속성 대화 상자](../debugger/window-properties-dialog-box.md)를 표시하려면 포커스를 [창 뷰](../debugger/windows-view.md) 창으로 이동합니다.  그런 다음 트리에서 창 노드를 선택하고 **보기** 메뉴에서 **속성**을 선택합니다.  
  
 **일반** 탭에서 사용할 수 있는 설정은 다음과 같습니다.  
  
|Entry|설명|  
|-----------|--------|  
|**창 캡션**|컨트롤일 경우 창 캡션의 텍스트이거나 창에 포함된 텍스트입니다.|  
|**창 핸들**|이 창의 고유 ID입니다.  창 핸들 번호는 다시 사용되므로 해당 창의 수명 중에만 창을 식별합니다.|  
|**창 프로시저**|이 창에 대한 창 프로시저 함수의 가상 주소입니다.  또한 이 필드는 이 창이 유니코드 창인지 여부와 서브클래싱되었는지 여부를 나타냅니다.|  
|**사각형**|창의 경계 사각형입니다.  사각형의 크기도 표시됩니다.  화면 좌표의 단위는 픽셀입니다.|  
|**복원된 영역**|복원된 창의 경계 사각형입니다.  사각형의 크기도 표시됩니다.  복원된 영역은 창이 최대화 또는 최소화된 경우에만 사각형과 다릅니다.  화면 좌표의 단위는 픽셀입니다.|  
|**클라이언트 영역**|창 클라이언트 영역의 경계 사각형입니다.  사각형의 크기도 표시됩니다.  단위는 창 클라이언트 영역의 왼쪽 위를 기준으로 계산된 픽셀입니다.|  
|**인스턴스 핸들**|응용 프로그램의 인스턴스 핸들입니다.  인스턴스 핸들은 고유하지 않습니다.|  
|**컨트롤 ID 또는 메뉴 핸들**|표시되는 창이 자식 창일 경우 컨트롤 ID 레이블이 표시됩니다.  컨트롤 ID는 이 자식 창의 컨트롤 ID를 식별하는 정수입니다.  표시되는 창이 자식 창이 아닐 경우 메뉴 핸들 레이블이 표시됩니다.  메뉴 핸들은 이 창과 연결된 메뉴의 핸들을 식별하는 정수입니다.|  
|**사용자 데이터**|이 창 구조체에 연결된 응용 프로그램 관련 데이터입니다.|  
|**창 바이트**|이 창과 연결된 추가 바이트 수입니다.  이러한 바이트의 의미는 응용 프로그램에 따라 달라집니다.  바이트 값을 DWORD 형식으로 표시하려면 목록 상자를 확장합니다.|
