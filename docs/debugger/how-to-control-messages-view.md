# How to: Control Messages View
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+를 사용하면 [메시지 뷰](../debugger/messages-view.md) 창의 내용을 세부적으로 제어할 수 있습니다.  언제든지 메시지 표시를 [시작 및 중지](../debugger/how-to-start-and-stop-the-message-log-display.md)할 수 있으며 다음 사항을 지정할 수 있습니다.  
  
-   표시할 메시지 형식  
  
-   모니터링할 창  
  
-   메시지 줄의 표시 형식  
  
 이러한 설정은 [메시지 옵션 대화 상자](../debugger/message-options-dialog-box.md)에서 사용할 수 있으며 활성 [메시지 뷰](../debugger/messages-view.md) 창에만 적용됩니다.  이 대화 상자를 열려면 **Spy** 메뉴에서 **로그 메시지**를 선택합니다.  
  
 메시지 옵션 대화 상자는 세 개의 탭 창으로 구성되어 있습니다.  자세한 내용은 [메시지 옵션 대화 상자, 창 탭](../debugger/windows-tab-message-options-dialog-box.md), [메시지 옵션 대화 상자, 메시지 탭](../debugger/messages-tab-message-options-dialog-box.md) 및 [메시지 옵션 대화 상자, 출력 탭](../debugger/output-tab-message-options-dialog-box.md)을 참조하십시오.
