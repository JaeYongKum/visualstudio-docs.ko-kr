# <a name="vsgdbgvsgdbg-destructor"></a>VsgDbg::~VsgDbg(소멸자)
인스턴스를 제거는 `VsgDbg` 클래스입니다. 그래픽 정보를 적극적으로 기록 하 고 그래픽 로그 파일을 종료 하 고 닫은 및 적극적으로 그래픽 정보를 캡처하는 동안 사용 된 리소스를 해제 합니다.  
  
## <a name="syntax"></a>구문  
  
```C++  
~VsgDbg();  
```
