# Search Tools for Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

이러한 대화 상자를 사용하여 Spy\+\+ 뷰 창에 표시할 데이터를 지정할 수 있습니다.  
  
## 단원 내용  
 [창 찾기 대화 상자](../debugger/find-window-dialog-box.md)  
 특정 창에서 속성이나 메시지를 표시하는 데 사용됩니다.  
  
 [창 경로 대화 상자](../debugger/window-search-dialog-box.md)  
 창 뷰에서 특정 창의 노드를 찾는 데 사용됩니다.  
  
 [메시지 검색 대화 상자](../debugger/message-search-dialog-box.md)  
 메시지 뷰에서 특정 메시지의 노드를 찾는 데 사용됩니다.  
  
 [프로세스 검색 대화 상자](../debugger/process-search-dialog-box.md)  
 프로세스 뷰에서 특정 프로세스의 노드를 찾는 데 사용됩니다.  
  
 [스레드 검색 대화 상자](../debugger/thread-search-dialog-box.md)  
 스레드 뷰에서 특정 스레드의 노드를 찾는 데 사용됩니다.  
  
## 관련 단원  
 [Spy\+\+ 뷰](../debugger/spy-increment-views.md)  
 창, 메시지, 프로세스 및 스레드의 Spy\+\+ 트리 뷰에 대해 설명합니다.  
  
 [Spy\+\+ 사용](../debugger/using-spy-increment.md)  
 Spy\+\+ 도구를 소개하고 사용 방법에 대해 설명합니다.  
  
 [Spy\+\+ 참조](../debugger/spy-increment-reference.md)  
 각 Spy\+\+ 메뉴 및 대화 상자에 대해 설명하는 단원이 포함되어 있습니다.
