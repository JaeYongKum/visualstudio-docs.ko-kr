# 글꼴 대화 상자(Microsoft Spy++ 도움말)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

이 대화 상자를 사용하여 [Spy\+\+ 뷰](../debugger/spy-increment-views.md) 창에서 사용되는 문자 글꼴을 구성할 수 있습니다.  이 대화 상자를 표시하려면 **보기** 메뉴에서 **글꼴**을 선택합니다.  
  
 다음과 같은 설정을 사용할 수 있습니다.  
  
 **글꼴**  
 포커스가 있는 Spy\+\+ 뷰 창에서 사용되는 문자 글꼴입니다.  
  
 **글꼴 스타일**  
 포커스가 있는 Spy\+\+ 뷰 창에서 사용되는 글꼴 스타일입니다.  
  
 **크기**  
 포커스가 있는 Spy\+\+ 뷰 창에서 사용되는 글꼴 크기입니다.  
  
 **글꼴을 기본값으로 저장**  
 Spy\+\+ 뷰를 열 때 현재 글꼴 설정을 기본 설정으로 저장합니다.
