# How to: Start and Stop the Message Log Display
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[메시지 뷰](../debugger/messages-view.md) 창이 활성화되면 Spy\+\+ **메시지** 메뉴에 **로깅 시작** 또는 **로깅 중지** 옵션이 표시되고 해당 **로깅 시작** 또는 **로깅 중지** 도구 모음 단추도 활성화됩니다.  
  
### 메시지 로그 표시를 시작하려면  
  
1.  **메시지** 메뉴에서 **로깅 시작**을 선택합니다.  
  
### 메시지 로그 표시를 중지하려면  
  
1.  **메시지** 메뉴에서 **로깅 중지**를 선택합니다.
