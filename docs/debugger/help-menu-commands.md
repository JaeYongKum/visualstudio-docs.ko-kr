# <a name="help-menu-commands"></a>도움말 메뉴 명령
**도움말** 메뉴는 도움말 파일을 여는 명령을 제공 합니다.  
  
 **Index**  
 Spy + + 도움말에 대 한 내용을 화면을 엽니다.  
  
 **도움말 사용**  
 도움말을 사용 하는 방법을 설명 하는 도움말 파일을 엽니다.  
  
 **Spy + + 정보**  
 Spy + +에 대 한 소프트웨어 버전 및 저작권 정보를 보여 줍니다.
