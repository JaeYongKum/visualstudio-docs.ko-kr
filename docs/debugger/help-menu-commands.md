# 도움말 메뉴 명령
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**도움말** 메뉴는 도움말 파일을 열기 위한 명령을 제공합니다.  
  
 **Index**  
 Spy\+\+ 도움말의 목차 화면을 엽니다.  
  
 **도움말 사용**  
 도움말을 사용하는 방법을 설명하는 도움말 파일을 엽니다.  
  
 **Spy\+\+ 정보**  
 Spy\+\+에 대한 소프트웨어 버전 및 저작권 정보를 표시합니다.
