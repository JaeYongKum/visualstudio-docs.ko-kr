# <a name="spy-menu-commands"></a>Spy 메뉴 명령
**Spy** 메뉴는 Spy + + 창 및 뷰를 사용 하는 명령을 제공 합니다.  
  
 **Windows**  
 새 창 뷰를 엽니다.  
  
 **프로세스**  
 새 프로세스 뷰를 엽니다.  
  
 **스레드**  
 새 스레드 뷰를 엽니다.  
  
 **로그 메시지**  
 새 메시지 뷰를 열고 엽니다는 [메시지 옵션 대화 상자](../debugger/message-options-dialog-box.md) 뷰에 대 한 합니다.  
  
 **창 찾기**  
 표시는 **마우스로** 대화 상자는 창을 선택 합니다.  
  
 **끝내기**  
 Spy + + 유틸리티를 닫습니다.
