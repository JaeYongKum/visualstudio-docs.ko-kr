# 메시지 메뉴 명령
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**메시지** 메뉴는 메시지 뷰를 제어하기 위한 명령을 제공합니다.  
  
 **로깅 시작\/중지**  
 메시지 뷰를 시작하거나 중지합니다.  
  
 **로깅 옵션**  
 메시지 검색 조건을 지정할 수 있는 **메시지 옵션** 대화 상자를 엽니다.  
  
 **Clear Log**  
 활성 메시지 뷰의 내용을 지웁니다.
