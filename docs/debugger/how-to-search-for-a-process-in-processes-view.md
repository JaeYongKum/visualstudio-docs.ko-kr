# <a name="how-to-search-for-a-process-in-processes-view"></a>방법: 프로세스 뷰에서 프로세스 검색
프로세스 ID 또는 모듈 문자열을 검색 조건으로 사용 하 여 특정 프로세스 뷰에서 프로세스에 대 한 검색할 수 있습니다. 또한 검색의 초기 방향을 지정할 수 있습니다. 대화 상자의 필드에에서 프로세스 트리에서 선택한 프로세스의 특성을 표시 됩니다.  
  
### <a name="to-search-for-a-process-in-processes-view"></a>프로세스 뷰에서 프로세스 검색 하려면  
  
1.  따라서 정렬 해당 Spy + + 및 활성 [프로세스 뷰](../debugger/processes-view.md) 창이 표시 됩니다.  
  
2.  **검색** 메뉴 선택 **프로세스 찾기**  
  
     [프로세스 검색 대화 상자](../debugger/process-search-dialog-box.md) 열립니다.  
  
3.  검색 조건으로 프로세스 ID 또는 모듈 문자열을 입력 합니다.  
  
4.  모든 필드 값을 지정 하지 않을 선택을 취소 합니다.  
  
    > [!TIP]
    >  모듈에 의해 소유 하는 모든 프로세스를 확인 하려면 선택을 취소는 **프로세스** 에 모듈 이름을 입력 하 고 상자는 **모듈** 상자입니다. 다음 사용 하 여 **다음 찾기** 프로세스에 대 한 검색을 계속 하려면.  
  
5.  선택 **를** 또는 **아래로** 검색의 초기 방향에 대 한 합니다.  
  
6.  **확인**을 클릭합니다.  
  
 강조 표시 된 일치 하는 프로세스에 없는 경우는 **프로세스 뷰** 창.
