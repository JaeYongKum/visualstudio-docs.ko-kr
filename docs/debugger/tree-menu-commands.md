# <a name="tree-menu-commands"></a>트리 메뉴 명령
**트리** 메뉴는 확장 하거나 축소 뷰 명령을 제공 합니다.  
  
 **한 수준 확장**  
 한 수준 아래에 표시 하 여 시스템 뷰에서 현재 선택한 항목을 확장 합니다.  
  
 **분기를 확장 합니다.**  
 현재 선택 된 항목 아래에 있는 모든 레벨을 확장 합니다.  
  
 **모두 확장**  
 확장 하 여 시스템 뷰에서 모든 수준을 표시 합니다.  
  
 **축소**  
 현재 선택 된 항목 아래에 있는 모든 수준을 축소합니다.
