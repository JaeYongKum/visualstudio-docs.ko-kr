# <a name="thread-search-dialog-box"></a>스레드 검색 대화 상자
이 대화 상자를 사용 하 여 선택 합니다. 파일에 지정 된 스레드에 대 한 노드를 [스레드 뷰](../debugger/threads-view.md)합니다. 이 대화 상자를 표시 하려면 포커스를 이동 하는 **스레드** 뷰 창. 그런 다음 선택 **스레드 찾기** 에서 **검색** 메뉴.  
  
 다음 설정을 사용할 수 있습니다.  
  
 **스레드**  
 검색할 스레드 ID입니다.  
  
 **모듈**  
 검색할 모듈 문자열입니다.  
  
 **검색 방향을 위나 아래로 이동**  
 검색의 초기 방향입니다.
