# How to: Start Spy++
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Visual Studio 또는 명령 프롬프트에서 Spy\+\+를 시작할 수 있습니다.  
  
 Spy\+\+를 시작할 때 컴퓨터를 변경할 수 있는 권한을 묻는 메시지가 표시되면 **예**를 클릭합니다.  
  
> [!NOTE]
>  Spy\+\+ 인스턴스를 하나만 실행할 수 있습니다.  다른 인스턴스를 실행하려고 하면 현재 실행되는 인스턴스가 포커스를 얻게 됩니다.  
  
### Visual Studio에서 Spy\+\+를 시작하려면  
  
-   **도구** 메뉴에서 **Spy\+\+**를 클릭합니다.  
  
     Spy\+\+가 독립적으로 실행되기 때문에 Spy\+\+를 시작한 후 Visual Studio를 닫을 수 있습니다.  
  
    > [!NOTE]
    >  Spy\+\+를 사용하여 메시지를 기록하는 경우 운영 체제가 더 느리게 실행될 수 있습니다.  
  
### 명령 프롬프트에서 Spy\+\+를 시작하려면  
  
1.  명령 프롬프트 창에서 spyxx.exe가 포함된 폴더로 디렉터리를 변경합니다.  일반적으로 이 폴더의 경로는 ..  \\*Visual Studio installation folder*\\Common7\\Tools\\입니다.  
  
2.  **spyxx.exe**를 입력하고 Enter 키를 누릅니다.
