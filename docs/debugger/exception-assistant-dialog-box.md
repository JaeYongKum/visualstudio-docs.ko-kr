# 예외 도우미 대화 상자
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**예외 도우미** 대화 상자는 런타임 예외가 throw되면 표시됩니다.  **예외 도우미**는 예외의 형식을 표시하고, 추가 정보와 문제 해결 팁에 대한 링크를 제공하며, 온라인으로 추가 도움말을 검색할 수 있는 방법을 제공합니다. 사용자는 이 대화 상자에서 예외에 대한 세부 정보를 보는 등의 특정 작업을 수행할 수 있습니다.  
  
## 작업 목록  
 발생한 예외 형식을 해결하는 방법에 대한 항목을 보려면 **문제 해결 팁** 창에 표시된 팁 메시지 중 하나를 클릭합니다.  
  
 예외와 관련된 작업을 수행하려면 작업 창에 표시된 작업 중 하나를 클릭합니다.  
  
 예외 도우미를 사용하거나 사용하지 않도록 설정하는 방법에 대한 자세한 내용은 [옵션 대화 상자, 디버깅, 일반](../debugger/general-debugging-options-dialog-box.md)을 참조하십시오.  
  
## 사용자 인터페이스 요소 목록  
 **예외 형식**  
 throw된 예외의 형식을 표시합니다.  
  
 **추가 정보**  
 예외에 대한 추가 정보를 표시합니다.  
  
 **문제 해결 팁**  
 예외의 원인을 발견하는 데 도움이 될 수 있는 문제 해결 팁에 대한 링크를 표시합니다.  
  
 **작업**  
 예외 개체에 대한 추가 정보 표시를 비롯하여 수행할 수 있는 작업을 나열합니다.  
  
 **온라인 도움말 보기**  
 온라인으로 추가 도움말을 검색할 수 있습니다.
