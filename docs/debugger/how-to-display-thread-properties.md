# How to: Display Thread Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[스레드 뷰](../debugger/threads-view.md)에 표시된 스레드에 대한 자세한 내용을 보려면 [스레드 속성 대화 상자](../debugger/thread-properties-dialog-box.md)를 사용합니다.  
  
### 스레드 속성 대화 상자를 열려면  
  
1.  [스레드 뷰](../debugger/threads-view.md)에 표시된 트리에서 스레드를 선택합니다.  
  
2.  **보기** 메뉴에서 **속성**을 선택합니다.  
  
 속성 대화 상자는 모달 대화 상자가 아니므로 뷰 창에서 다른 항목을 클릭할 수 있으며 속성 대화 상자에는 선택한 항목에 대한 정보가 표시됩니다.
