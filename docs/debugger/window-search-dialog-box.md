# <a name="window-search-dialog-box"></a>창 경로 대화 상자
특정 창에 대 한 노드를 선택 합니다.를이 대화 상자를 사용 하 여 [창 뷰](../debugger/windows-view.md)합니다. 이 대화 상자를 표시 하려면 창 뷰 창에 포커스를 이동 합니다. 그런 다음 선택 **창 찾기** 에서 **검색** 메뉴.  
  
 사용할 수는 **찾기 도구** 창 핸들을 입력 하거나 캡션과 클래스 창을 식별할 수 하는 창 또는 있습니다를 선택 합니다.  
  
 다음 설정을 사용할 수 있습니다.  
  
 **찾기 도구**  
 끌어서 놓기 도구 창을 선택 하는 데 사용 합니다.  
  
 **Spy + + 숨기기**  
 Spy + + 주 창을 제거는 **창** 대화 상자를 다른 응용 프로그램 위에 표시 합니다.  
  
 **핸들**  
 검색할 창 고유 ID입니다.  
  
 **캡션**  
 검색할 창 캡션의 텍스트입니다.  
  
 **클래스**  
 검색할 창 클래스입니다.  
  
 **검색 방향을 위나 아래로 이동**  
 검색의 초기 방향입니다.
