# 창 속성 대화 상자, 프로세스 탭
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**프로세스** 탭을 사용하여 선택된 창의 프로세스를 식별할 수 있습니다.  [창 속성 대화 상자](../debugger/window-properties-dialog-box.md)를 표시하려면 포커스를 [창 뷰](../debugger/windows-view.md) 창으로 이동합니다.  그런 다음 트리에서 창 노드를 선택하고 **보기** 메뉴에서 **속성**을 선택합니다.  
  
 **프로세스** 탭에서 사용할 수 있는 설정은 다음과 같습니다.  
  
|Entry|설명|  
|-----------|--------|  
|**프로세스 ID**|이 창을 만든 스레드를 소유하고 있는 프로세스의 ID입니다.  이 프로세스의 속성을 보려면 이 값을 선택합니다.|  
|**스레드 ID**|이 창을 만든 스레드의 ID입니다.  이 스레드의 속성을 보려면 이 값을 선택합니다.|
