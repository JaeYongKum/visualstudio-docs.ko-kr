# 창 속성 대화 상자
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

이 대화 상자를 사용하여 특정 창에 대한 자세한 내용을 볼 수 있습니다.  이 대화 상자를 표시하려면 포커스를 [창 뷰](../debugger/windows-view.md) 창으로 이동합니다.  그런 다음 트리에서 창 노드를 선택하고 **보기** 메뉴에서 **속성**을 선택합니다.  
  
## 단원 내용  
 [일반 탭. 창 속성 대화 상자](../debugger/general-tab-window-properties-dialog-box.md)  
 선택된 창에 대한 정보를 표시합니다.  
  
 [스타일 탭. 창 속성 대화 상자](../debugger/styles-tab-window-properties-dialog-box.md)  
 선택된 창에 적용된 스타일을 표시합니다.  
  
 [창 탭. 창 속성 대화 상자](../debugger/windows-tab-window-properties-dialog-box.md)  
 선택된 창과 관련된 창에 대한 정보를 표시합니다.  
  
 [클래스 탭. 창 속성 대화 상자](../debugger/class-tab-window-properties-dialog-box.md)  
 선택된 창의 클래스에 대한 정보를 표시합니다.  
  
 [프로세스 탭. 창 속성 대화 상자](../debugger/process-tab-window-properties-dialog-box.md)  
 선택된 창의 프로세스를 식별합니다.
