# <a name="window-properties-dialog-box"></a>창 속성 대화 상자
이 대화 상자를 사용 하 여 특정 창에 대 한 자세한 내용을 찾을 수 있습니다. 이 대화 상자를 표시 하려면 포커스를 이동 하 여 [창 뷰](../debugger/windows-view.md) 창. 트리에서 창 노드를 선택한 후 선택 **속성** 에서 **보기** 메뉴.
