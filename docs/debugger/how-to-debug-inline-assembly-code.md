# <a name="how-to-debug-inline-assembly-code"></a>방법: 인라인 어셈블리 코드 디버깅
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 실제 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다. 설정을 변경하려면 도구 메뉴에서 설정 가져오기 및 내보내기를 선택합니다. 자세한 내용은 [Visual Studio IDE 개인 설정](../ide/personalizing-the-visual-studio-ide.md)을 참조하세요.  
  
 디버거는 인라인 어셈블리 코드 디버깅에 대 한 두 가지 창을 제공는 **디스어셈블리** 창 및 **등록** 창.  
  
## <a name="procedure"></a>프로시저  
  
#### <a name="to-debug-inline-assembly-code"></a>인라인 어셈블리 코드를 디버깅하려면  
  
1.  사용 하 여는 **디스어셈블리** 어셈블리 명령을 확인 하는 창입니다.  
  
2.  사용 하 여는 **등록** 창 레지스터의 내용을 볼 수 있습니다.
