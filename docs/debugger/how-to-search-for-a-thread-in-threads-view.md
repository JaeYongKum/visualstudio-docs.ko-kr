# <a name="how-to-search-for-a-thread-in-threads-view"></a>방법: 스레드 뷰에서 스레드 검색
스레드 뷰에서 특정 스레드에 대 한 검색 기준으로 스레드 ID 또는 모듈 문자열을 사용 하 여 검색할 수 있습니다. 또한 검색의 초기 방향을 지정할 수 있습니다. 대화 상자의 필드에에서는 스레드 트리에서 선택 된 스레드 특성 표시 됩니다.  
  
### <a name="to-search-for-a-thread-in-threads-view"></a>스레드 뷰에서 스레드 검색 하려면  
  
1.  따라서 정렬 해당 Spy + + 및 활성 [스레드 뷰](../debugger/threads-view.md) 창이 표시 됩니다.  
  
2.  **검색** 메뉴 선택 **스레드 찾기**합니다.  
  
     [스레드 검색 대화 상자](../debugger/thread-search-dialog-box.md) 열립니다.  
  
3.  검색 조건으로 스레드 ID 또는 모듈 문자열을 입력 합니다.  
  
4.  모든 필드 값을 지정 하지 않을 선택을 취소 합니다.  
  
    > [!TIP]
    >  모듈에서 소유한 모든 스레드를 확인 하려면 선택을 취소는 **스레드** 텍스트 상자 및 모듈 형식 이름에 **모듈** 상자입니다. 다음 사용 하 여 **다음 찾기** 스레드에 대 한 검색을 계속 하려면.  
  
5.  선택 **를** 또는 **아래로** 검색의 초기 방향에 대 한 합니다.  
  
6.  **확인**을 클릭합니다.  
  
 일치 하는 스레드 발견 되 면 스레드 보기 창에 강조 표시 됩니다.
