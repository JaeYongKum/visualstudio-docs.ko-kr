# 메시지 속성 대화 상자
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

이 대화 상자를 사용하여 특정 메시지에 대한 자세한 내용을 볼 수 있습니다.  이 대화 상자를 표시하려면 포커스를 [메시지 뷰](../debugger/messages-view.md) 창으로 이동합니다.  그런 다음 트리에서 메시지 노드를 선택하고 **보기** 메뉴에서 **속성**을 선택합니다.  
  
 이 대화 상자에는 유일하게 **일반** 탭만 표시됩니다.  다음과 같은 설정을 사용할 수 있습니다.  
  
 **창 핸들**  
 이 창의 고유 ID입니다.  창 핸들 번호는 다시 사용되므로 해당 창의 수명 중에만 창을 식별합니다.  이 창의 속성을 보려면 이 값을 클릭합니다.  
  
 **중첩 수준**  
 이 메시지의 중첩 수준입니다. 여기서 0은 중첩되지 않음을 나타냅니다.  
  
 **메시지**  
 선택된 창 메시지의 번호, 상태 및 이름입니다.  
  
 **lResult**  
 *lResult* 매개 변수의 값입니다\(있는 경우\).  
  
 **wParam**  
 *wParam* 매개 변수의 값입니다\(있는 경우\).  
  
 **lParam**  
 *lParam* 매개 변수의 값입니다\(있는 경우\).  이 값은 문자열이나 구조체에 대한 포인터일 경우 디코딩됩니다.
