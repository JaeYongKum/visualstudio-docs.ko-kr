# <a name="message-properties-dialog-box"></a>메시지 속성 대화 상자
이 대화 상자를 사용 하 여 특정 메시지에 대 한 자세한 내용을 찾을 수 있습니다. 이 대화 상자를 표시 하려면 포커스를 이동 하는 [메시지 뷰](../debugger/messages-view.md) 창. 트리에서 메시지 노드를 선택한 후 선택 **속성** 에서 **보기** 메뉴.  
  
 **일반** 탭에는 표시 합니다. 다음 설정을 사용할 수 있습니다.  
  
 **창 핸들**  
 이 창의 고유 ID입니다. 창 핸들 번호는 다시 사용 합니다. 해당 창의 사용 기간에만 창을 식별합니다. 이 창의 속성을 보려면이 값을 클릭 합니다.  
  
 **중첩 수준**  
 여기서 0은 중첩 없이이 메시지의 중첩 수준입니다.  
  
 **메시지**  
 번호, 상태 및 선택한 windows 메시지의 이름입니다.  
  
 **lResult**  
 값은 *lResult* 있는 경우 매개 변수입니다.  
  
 **wParam**  
 값은 *wParam* 있는 경우 매개 변수입니다.  
  
 **lParam**  
 값은 *lParam* 있는 경우 매개 변수입니다. 이 값은 문자열 또는 구조에 대 한 포인터 이면 디코딩됩니다.
