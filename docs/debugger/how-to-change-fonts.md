# How to: Change Fonts
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ 창의 글꼴, 글꼴 스타일 및 글꼴 크기를 변경할 수 있습니다.  
  
### 글꼴 옵션을 변경하려면  
  
1.  **보기** 메뉴에서 **글꼴**을 선택합니다.  
  
2.  글꼴, 글꼴 스타일 및 글꼴 크기를 선택합니다.  
  
3.  **확인**을 클릭합니다.  
  
 **글꼴을 기본값으로 저장**을 선택하면 이후의 모든 Spy\+\+ 창에서 이 글꼴이 사용됩니다.
