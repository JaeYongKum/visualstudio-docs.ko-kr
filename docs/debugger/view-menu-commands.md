# 보기 메뉴 명령
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**보기** 메뉴는 Spy\+\+ 사용자 인터페이스를 제어하기 위한 명령을 제공합니다.  
  
 **글꼴**  
 현재 Spy\+\+ 뷰에서 사용할 글꼴을 선택할 수 있는 **글꼴** 대화 상자를 엽니다.  
  
 **도구 모음**  
 도구 모음을 숨기거나 표시합니다.  
  
 **Status Bar**  
 상태 표시줄을 숨기거나 표시합니다.  
  
 **속성**  
 활성 뷰의 **속성** 대화 상자를 엽니다.
