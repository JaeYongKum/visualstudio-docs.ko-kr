# <a name="overview-of-deployment-in-visual-studio"></a>Visual Studio에서 배포의 개요

다른 컴퓨터, 장치, 서버 또는 클라우드에 설치할 수 있도록 응용 프로그램, 서비스 또는 구성 요소를 배포할 수 있습니다. Visual Studio에서 필요한 배포 유형에 적합한 방법을 선택할 수 있습니다. 

많은 공통 응용 프로그램 형식에 대 한 Visual Studio의 솔루션 탐색기에서 응용 프로그램 바로 프로그램을 배포할 수 있습니다.
