# <a name="uninstall-visual-studio"></a>Visual Studio 제거
이 페이지에서는 개발자를 위한 통합 생산성 도구 제품군인 Visual Studio를 제거하는 과정을 안내합니다.  

1.  컴퓨터에서 Visual Studio 설치 관리자를 찾습니다.  

     예를 들어 Windows 10 1주년 업데이트를 실행하는 컴퓨터에서 **시작**을 클릭한 다음 **M** 문자로 스크롤합니다. 여기서 **Microsoft Visual Studio 설치 관리자**를 찾을 수 있습니다.  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png)

2.  설치 관리자를 클릭하여 시작하고 ![세부 정보 아이콘](media/vs2017uninstall-UninstallIcon.png) 아이콘을 클릭한 다음 **제거**를 클릭합니다.  

     ![Visual Studio 2017 RC 수정: 시작 또는 수정](media/vs2017uninstall-ChooseUninstallFromInstaller.png "Visual Studio 2017 RC 복구 또는 제거")  

3.  **확인**을 클릭하여 선택을 확인합니다.

     ![Visual Studio 2017 RC 수정: 시작 또는 수정](media/vs2017uninstall-UninstallConfirm.png "Visual Studio 2017 RC 제거 확인")  

나중에 마음을 바꾸어 Visual Studio 2017 RC를 다시 설치하려는 경우 Visual Studio 설치 관리자를 다시 시작한 다음 선택 화면에서 **설치**를 클릭합니다.

> [!IMPORTANT]
> Visual Studio 2017 RC는 일반적으로 프로덕션 환경에서 사용할 수 있지만 설치 UI에서 "미리 보기"로 표시된 워크로드 및 구성 요소는 프로덕션 환경에서 사용할 수 없습니다.
