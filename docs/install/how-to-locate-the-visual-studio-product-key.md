# 방법: Visual Studio 제품 키 찾기
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

고객 지원 팀과 작업하거나, 평가판에서 업그레이드하거나, 사용자 고유의 레코드를 업데이트하는 경우 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]의 제품 키를 찾으려고 할 수 있습니다.  
  
> [!NOTE]
>  [!INCLUDE[esprtfs](../code-quality/includes/esprtfs_md.md)]의 제품 키를 찾으려면 [Team Foundation Server의 제품 키 찾기 또는 변경](../Topic/Locate%20or%20Change%20the%20Product%20Key%20for%20Team%20Foundation%20Server.md)의 단계를 수행합니다.  
  
### Visual Studio의 제품 키를 찾으려면  
  
-   다음 위치 중 한 곳에서 제품 키를 찾습니다.  
  
    -   설치 미디어\(해당하는 경우\)  
  
    -   Microsoft 웹 사이트의 [MSDN 구독](http://go.microsoft.com/fwlink/?LinkId=206363) 페이지  
  
    -   소프트웨어를 구입한 공급업체에서 제공한 설명서 또는 기타 통신
