# <a name="modify-visual-studio-2017-rc"></a>Visual Studio 2017 RC 수정
수행하려는 작업에 맞게 Visual Studio를 개인 설정하기 쉬울 뿐 아니라 Visual Studio 수정도 쉬워졌습니다. 더 이상 제어판에서 작업할 필요 없이 새로운 Visual Studio 설치 관리자를 시작하고 원하는 대로 변경하면 됩니다.  

 방법은 다음과 같습니다.  

## <a name="modify-workloads"></a>작업 수정  
 작업에는 사용 중인 프로그래밍 언어 또는 플랫폼에 필요한 기능이 포함되어 있습니다. 작업을 사용하여 원하는 시기에 원하는 작업을 지원하도록 Visual Studio를 수정할 수 있습니다.  

1.  컴퓨터에서 Visual Studio 설치 관리자를 찾습니다.  

     예를 들어 Windows 10 1주년 업데이트를 실행하는 컴퓨터에서 **시작**을 클릭한 다음 **M** 문자로 스크롤합니다. 여기서 **Microsoft Visual Studio 설치 관리자**를 찾을 수 있습니다.  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png "Microsoft Visual Studio 설치 관리자 찾기")

2.  설치 관리자를 클릭하여 시작한 다음 **수정**을 클릭합니다.  

     ![Visual Studio 2017 RC 수정: 시작 또는 수정](media/vs2017modify-ModifyOrLaunch.png "Visual Studio 2017 RC 시작 또는 수정")  

3.  **작업** 화면에서 설치하거나 제거할 작업을 클릭하여 선택하거나 선택 취소합니다.  

    ![Visual Studio 2017 RC 설정 대화 상자](media/vs2017modify-ChooseWorkload.png "Visual Studio 2017에서 작업 선택")

4. **수정**을 클릭합니다.  

5. 새 작업과 구성 요소가 설치된 후 **시작**을 클릭합니다.

## <a name="modify-individual-components"></a>개별 구성 요소 수정

편리한 작업 기능을 사용하여 Visual Studio 설치를 사용자 지정하지 않으려는 경우 Visual Studio 설치 관리자에서 **개별 구성 요소** 옵션을 선택하고 원하는 구성 요소를 선택한 다음 프롬프트를 따릅니다.  

> [!IMPORTANT]
> Visual Studio 2017 RC는 일반적으로 프로덕션 환경에서 사용할 수 있지만 설치 UI에서 "미리 보기"로 표시된 워크로드 및 구성 요소는 프로덕션 환경에서 사용할 수 없습니다.
