# Value(XElement 동적 속성)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

요소의 내용을 가져오거나 설정합니다.  
  
## 구문  
  
```  
elem.Value  
```  
  
## 속성 값\/반환 값  
 요소의 연결된 내용을 나타내는 <xref:System.String>입니다.  
  
## 설명  
 이 속성은 <xref:System.Xml.Linq.XElement?displayProperty=fullName> 클래스의 <xref:System.Xml.Linq.XElement.Value%2A> 속성과 동일하지만 이 동적 속성은 변경 알림도 지원합니다.
