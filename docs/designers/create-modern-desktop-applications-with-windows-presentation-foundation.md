# <a name="create-modern-desktop-applications-with-windows-presentation-foundation"></a>Windows Presentation Foundation으로 최신 데스크톱 응용 프로그램 만들기

Visual Studio의 WPF(Windows Presentation Foundation)는 Windows에서 최신 기간 업무 데스크톱 응용 프로그램을 빌드할 수 있는 통합된 프로그래밍 모델을 개발자에게 제공합니다.  

자세한 연습을 보려면 [WPF 시작](../designers/getting-started-with-wpf.md)에서 해당 보조 항목을 따르세요.

사용 가능한 도구 비교를 보려면 [Visual Studio 및 Blend for Visual Studio에서 XAML 디자인](../designers/designing-xaml-in-visual-studio.md)을 참조하세요.
