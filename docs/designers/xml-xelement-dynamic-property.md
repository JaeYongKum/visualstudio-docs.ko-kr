# Xml(XElement 동적 속성)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

요소의 서식이 지정되지 않은 XML 내용을 가져옵니다.  
  
## 구문  
  
```  
elem.Xml  
```  
  
## 속성 값\/반환 값  
 요소의 서식이 지정되지 않은 XML 내용을 나타내는 <xref:System.String>입니다.  
  
## 설명  
 이 속성은 `SaveOptions` 매개 변수가 <xref:System.Xml.Linq.SaveOptions>으로 설정된 <xref:System.Xml.Linq.XNode?displayProperty=fullName> 클래스의 <xref:System.Xml.Linq.XNode.ToString%28System.Xml.Linq.SaveOptions%29> 메서드와 동일합니다.
