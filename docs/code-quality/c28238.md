# C28238
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28238: 매개 변수의 주석에 value, stringValue 및 longValue가 두 개 이상 있습니다.  paramn\=xxx를 사용하십시오.
