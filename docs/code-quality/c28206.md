# C28206
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28206: \<expression\> : 남아 있는 피연산자가 구조체글 가리키고 `->` 사용  
  
 이 경고를 보고 구조체 포인터 표기법을 역참조 하는 경우 `->` 필요 합니다.
