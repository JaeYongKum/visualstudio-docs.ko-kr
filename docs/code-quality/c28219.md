# <a name="c28219"></a>C28219
C28219 경고: 주석에서 매개 변수 주석에 대 한 예상 열거형의 멤버  
  
 명명 된 멤버를 사용할 수는 주석의 매개 변수가 `enum` 유형과 다른 기호가 발견 되었습니다; 해당 멤버를 사용 하 여 `enum` 유형입니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다.
