# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28219: 주석에 있는 매개 변수 주석에 열거의 멤버가 필요합니다.  
  
 주석 매개가 `enum` 으로 명명된 멤버의 변수로 예상되고 몇몇 다른 유형 및 다른 기호가 발견 되었습니다.; `enum` 형식에 해당되는 멤버를 사용합니다.  일반적으로 잘못 코딩된 주석 매크로를 나타냅니다.
