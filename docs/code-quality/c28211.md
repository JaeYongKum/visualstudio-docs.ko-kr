# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경기 C28211: SAL\_context에 대한 정적 컨텍스트 이름이 필요합니다.  
  
 이 경고는 `_Static_context_` 주석에 피연산자 함수는 도구 정의 컨텍스트 이름이어야만 하는 것을 나타냅니다.  일반적인 사용에서는 이 경고가 발생 하지 않습니다.
