# <a name="c28211"></a>C28211
C28211 경고: 정적 컨텍스트 이름이 SAL_context에 대 한 필요 합니다.  
  
 이 경고를 나타냅니다에 있는 피연산자는 `_Static_context_` 주석 정의 도구 컨텍스트 이름 이어야 합니다. 일반적인 사용의 경우 이 경고가 발생하지 않습니다.
