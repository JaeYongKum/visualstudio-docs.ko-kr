# C28208
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28208: \<function\>는 이전에\<file\>\(\<line\>\)에서 다른 매개 변수 목록을 사용하여 정의되었습니다.  일부 분석 도구에서 잘못된 결과가 생성됩니다.  
  
 다른 항목을 검색 하지 않는 것으로 알려진 함수 정의를 하는 경우 이 경고가 보고 됩니다.
