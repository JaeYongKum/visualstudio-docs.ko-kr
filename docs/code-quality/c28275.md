# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28275: \_Macro\_value\_ 매개 변수가 null입니다.  
  
 이 경고는 분석 중인 코드, 모델 파일에 내부 오류가 있는 것을 나타냅니다.  *macroValue* 함수는 매개 변수 없이 호출되었습니다.
