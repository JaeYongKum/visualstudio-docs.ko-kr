# <a name="c28275"></a>C28275
C28275 경고: 매개 변수를 _Macro_value\_ null입니다  
  
 이 경고는 분석 중인 코드가 아니라 모델 파일에 내부 오류가 있음을 나타냅니다. *macroValue* 매개 변수 없이 함수가 호출 되었습니다.
