# <a name="c28163"></a>C28163
C28163 경고: 함수는 try / except 블록 내에서 호출 되지 해야  
  
 함수는 유형인 경우이 경고는 보고에 묶이면 안 하는 `try/except` 블록에서 발견 되는 `try/except` 블록입니다. 함수가 호출한 하나 이상의 경로에서 발견된 코드 분석 도구가 `try/except` 블록 안에 있습니다.
