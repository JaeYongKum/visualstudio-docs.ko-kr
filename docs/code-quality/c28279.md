# <a name="c28279"></a>C28279
C28279 경고: 기호에 대 한는 일치 하는 'end' 없이 발견 된 'begin'  
  
 주석 언어가 begin 및 end(C에서는 `{` 및 `}`) 구문을 지원하며, 이 쌍의 균형이 맞지 않습니다. 매크로를 사용하는 경우 이 상황을 방지할 수 있습니다.
