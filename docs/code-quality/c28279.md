# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28279: 기호의 경우 일치하는 'end'가 없는 'begin'이 있습니다.  
  
 주석 언어는 처음과 끝에서 \(C에서`{` 및 `}` \) 을 생성하는 것과 불균형인 구조를 지원합니다.  매크로 사용하는 경우 이러한 상황을 피할 수 있습니다.
