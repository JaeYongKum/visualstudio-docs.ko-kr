# <a name="c28305"></a>C28305
C28305 경고: 구문 분석 하는 동안 오류가 \<토큰 > 발견 되었습니다.

 이 경고는 지정된 토큰을 포함하는 식의 형식이 잘못되었을 때 보고됩니다.
