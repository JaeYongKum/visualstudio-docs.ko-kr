# <a name="c28285"></a>C28285
C28285 경고: 매개 변수에 구문 오류가 함수에 대 한

 코드 분석 도구는 모델 파일에서 오류가 발생할 때 이 경고를 보고합니다. 일부 소스 파일의 오류로 인해 이러한 오류가 발생할 수도 있습니다.
