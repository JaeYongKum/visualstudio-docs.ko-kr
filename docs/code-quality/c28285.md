# C28285
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28285: 함수의 경우 매개 변수에 구문 오류가 있습니다.  
  
 코드 분석 도구는 모델 파일에 가능한 오류가 발생 하면 이 경고를 보고 합니다.  일부 원본 파일 오류 이러한 오류가 발생할 수 있습니다.
