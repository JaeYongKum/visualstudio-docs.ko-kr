# C1256
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C1256: PCH를 쓰고 읽어야 합니다.  
  
 코드 분석 도구는 내부 도구에서 오류가 있고 분석되는 코드에 있지 않는 경우 이 경고를 보고 합니다.
