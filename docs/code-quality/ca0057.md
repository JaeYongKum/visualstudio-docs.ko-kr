 FxCopCmd.exe 프로젝트 파일을 사용 하 여 수동으로 호출 하는 경우이 오류가 발생 한 또는 위반이 대상 및 규칙의 현재 집합에 매핑할 수 없는 XML 보고서를 가져온 / 합니다.
