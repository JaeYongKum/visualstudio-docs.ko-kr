# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28300: \<parameter\_name\>: 연산자에 대한 호환되지 않는 형식의 식 피연산자 \<operator\_name\>입니다.  
  
 매개 변수에서 피연산자 형식 연산자와 호환 되지 않는 경우이 경고가 보고 됩니다.
