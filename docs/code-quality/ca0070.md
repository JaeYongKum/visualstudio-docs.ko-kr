# <a name="ca0070"></a>CA0070
다음 조건 중 하나에 적어도 및 코드 분석 어셈블리를 분석 하 고 권한 특성에서 발생 하는 경우이 오류가 발생 합니다.  
  
-   코드 분석의 특성이 포함 된 어셈블리를 찾을 수 없습니다.  
  
-   코드 분석에는 어셈블리를 로드할 수 있는 권한이 없습니다.  
  
-   코드 분석에 의해 로드 된 어셈블리에 특성이 없습니다.  
  
 대부분의 경우에서 코드 분석 도구의 최신 버전이 있는지 확인 하 여 문제를 해결할 수 있습니다.
