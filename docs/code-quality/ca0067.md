# <a name="ca0067"></a>CA0067
디렉터리를 찾을 수 없습니다.

 값은 **/directory** FxCopCmd 명령줄에서 옵션을 찾을 수 없습니다.

 이 경고는 사용 하는 경우에 발생할 수 있습니다는 **CodeAnalysisDependentAssemblyPaths** 속성 [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] 추가 참조 어셈블리 검색 경로 및 이러한 경로 중 하나를 지정할 수 없습니다.

-   와 함께 경고 c a 0067 나타나면 [c a 0058](ca0058.md) 오류 또는 [CA0060](ca0060.md) c a 0067을 해결 하는 경고, 일반적으로 다른 오류를 해결 합니다.

-   경고 c a 0067 다른 오류 또는 경고 없이 표시 되 면 디렉터리가 필요 하지 않음 때문에 일반적으로 경고를 무시할 수 있습니다.
