# <a name="c28229"></a>C28229
경고 C28229: 주석에서 매개 변수를 지원하지 않습니다.

 이 경고는 매개 변수 없이 주석이 선언되었지만 매개 변수와 함께 주석이 사용되었음을 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
