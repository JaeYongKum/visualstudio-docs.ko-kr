# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28229: 주석은 매개 변수를 지원하지 않습니다.  
  
 이 경고는 주석에 매개 변수 없이 선언할 때 매개 변수를 사용 하 여 주석을 사용 되었습니다 나타냅니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
