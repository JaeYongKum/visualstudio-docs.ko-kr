# <a name="c28227"></a>C28227
경고 C28227: 주석은 PrimOp 일 수 없습니다 (이전 선언 참조)

 이 경고는 명명 된 주석을 일반적인 주석으로 선언 되 고도 이전에 선언 되었으므로 PrimOp로 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
