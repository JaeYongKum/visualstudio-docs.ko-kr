# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28232: \_Pre\_, \_Post\_, 또는 \_Deref\_는 어떤 주석에도 적용되지 않습니다.  
  
 이 경고를 `_Pre_`, `_Post_`, 또는 `_Deref_` 이후의 기능 주석없이 주석 식에서 연산자가 표시 한정자를 무시 하지만 잘못 코딩된 주석을 나타냅니다.
