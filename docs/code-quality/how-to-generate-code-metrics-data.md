# 방법: 코드 메트릭 데이터 생성
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

전체 솔루션의 코드 메트릭 결과를 생성할지 또는 선택한 프로젝트의 코드 메트릭 결과를 생성할지 결정할 수 있습니다.  
  
 **요구 사항**  
  
 Visual Studio 2012 Ultimate 또는 Visual Studio 2012 Premium  
  
### 전체 솔루션에 대한 코드 메트릭 결과를 생성하려면  
  
-   **분석** 메뉴를 클릭한 다음 **솔루션에 대해 코드 메트릭 계산**을 클릭합니다.  
  
     \- 또는 \-  
  
-   **솔루션 탐색기**에서 솔루션을 마우스 오른쪽 단추로 클릭한 다음 **코드 메트릭 계산**을 클릭합니다.  
  
     \- 또는 \-  
  
-   **코드 메트릭 결과** 창에서 **솔루션에 대해 코드 메트릭 계산** 단추를 클릭합니다.  
  
     결과가 생성되고 **코드 메트릭 결과** 창이 표시됩니다.  
  
### 선택한 프로젝트 하나 이상에 대한 코드 메트릭 결과를 생성하려면  
  
1.  **솔루션 탐색기**에서 하나 이상의 프로젝트를 선택합니다.  
  
2.  **분석**을 클릭한 다음 **선택한 프로젝트에 대한 코드 메트릭 계산**을 클릭합니다.  
  
     \- 또는 \-  
  
     선택 항목을 마우스 오른쪽 단추로 클릭한 다음 **코드 메트릭 계산**을 클릭합니다.  
  
 결과가 생성되고 **코드 메트릭 결과** 창이 표시됩니다.  
  
### 자세한 결과를 보려면  
  
-   **계층 구조** 열에서 트리를 확장합니다.
