# <a name="ca0504"></a>CA0504
규칙 디렉터리는 존재 하지 않기 때문에 무시 되었습니다.

 경고 CA504 디렉터리에서 지정 된 경우 생성 되는 **CodeAnalysisRuleDirectories** 프로젝트 또는 빌드 구성 파일에서 속성을 찾을 수 없습니다. 이 문제를 해결 하려면 올바른 디렉터리 경로 입력 합니다.
