# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>관리 코드의 복잡성 및 유지 관리 용이성 측정
최신 소프트웨어 응용 프로그램의 복잡성 증가 유지 하 고 신뢰할 수 있는 코드를 만들기도 증가 합니다. 최근 몇 년 동안에서 코드 메트릭 이라는 많은 소프트웨어 측정 개발 되었습니다를 파악할 수 재작업이 나 더 많은 테스트가 코드에서 해야 합니다.  
  
 개발자가 사용할 수 [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] 복잡 하 고 관리 코드의 유지 관리 용이성 측정 하는 코드 메트릭 데이터를 생성 합니다. 전체 솔루션 또는 단일 프로젝트에 대해 코드 메트릭 데이터를 생성할 수 있습니다.
