# <a name="ca0052"></a>CA0052
분석 선택한 대상이 없습니다.  
  
 이 오류를 생성할 수 있는 두 가지 경우가 있습니다.  
  
-   CA0001 또는 CA0055 인의 근본 원인을 이유 때문에 대상이 분석 도구는 분석할 준비가 되었을 때 선택 된 경우 IDE 내에서 관리 코드 분석 도구를 사용 하 여이 오류가 발생할 수 있습니다.  
  
-   FxCopCmd.exe를 사용 하 여 명령줄에서 나열 된 대상이 FxCop 프로젝트 파일을 가리키는 /p 또는 /project 스위치와 함께 또는 /f 또는 /file 스위치 없이 호출 될 때이 오류가 발생할 수 있습니다. FxCopCmd 입력에 대 한 도움말 FxCopCmd.exe, /? 명령을 입력 하십시오.  
  
 다른 모든 오류는 오류의 근본 원인을 나타내는 다른 메시지와 함께 합니다.
