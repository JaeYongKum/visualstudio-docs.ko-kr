# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28289 : ReadableTo 또는 WritableTo에 limit\-spec가 매개 변수로 포함되지 않았습니다.  
  
 설명했듯이 코드 분석 도구는 함수\/매개 변수 주석이 miscode 된 경우 이 경고를 보고 합니다.
