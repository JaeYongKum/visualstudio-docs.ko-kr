# <a name="c28020"></a>C28020
경고 C28020: 식 \<expr >이 호출에 해당 되지 않습니다

 이 경고는 보고 때 합니다 \_Satisfies\_ 나열 하는 식이 true 아닙니다. 이 경고는 종종 잘못된 매개 변수를 나타냅니다.

 함수 선언에서 이 경고가 발생할 경우 주석은 불가능한 상태를 나타냅니다.
