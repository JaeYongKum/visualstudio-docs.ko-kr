# <a name="c28020"></a>C28020
C28020 경고: 식을 \<x p r >이 호출에 사실이 아닙니다

 이 경고는 보고 때는 _Satisfies\_ 나열 된 식에 사실이 아닙니다. 이 경고는 종종 잘못된 매개 변수를 나타냅니다.

 함수 선언에서 이 경고가 발생할 경우 주석은 불가능한 상태를 나타냅니다.
