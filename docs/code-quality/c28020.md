# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28020: 이 호출에서 식 \<expr\>이 true가 아닙니다.  
  
 목록에 나온 \_Satisfies\_ 식이 사실이 아닌 경우 이 경고가 보고 됩니다.  이 것은 자주 잘못된 매개 변수를 나타냅니다.  
  
 함수 선언에 이런 경우 주석은 불가능한 상태를 나타냅니다.
