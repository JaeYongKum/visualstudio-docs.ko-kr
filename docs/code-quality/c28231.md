# C28231
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28231: 주석은 배열에서만 유효합니다.  
  
 이 경고는 주석에 인수는 배열이어야 하고 몇 가지 다른 형식이 접해지는 것을 나타냅니다.
