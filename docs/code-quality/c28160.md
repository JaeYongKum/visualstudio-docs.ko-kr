# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28160: 주석 오류  
  
 `__drv_error` 주석이 발생했을 때 경고가 보고 됩니다.  이 주석은 고쳐져야만 하는 플래그 코딩에 사용되어야 하고 `__drv_when` 주석과 함께 매개 변수 조합을 나타내는 데에 사용될 수 있습니다.
