# <a name="c6506"></a>C6506
C6506 경고: 잘못 된 주석: \<이름 > 속성 값 포인터 또는 배열 형식에 사용할 수 있습니다  
  
 이 경고는 속성 포인터 또는 배열 형식이 아닌 형식에 사용 됨을 나타냅니다. 모든 데이터 형식에 액세스, Tainted, 및 유효한 속성을 사용할 수 수 있습니다. ValidBytesConst, ValidElementsConst, ElementSize, 및 NullTerminted 같은 다른 속성은 포인터, 멤버 또는 배열 형식에 대 한 포인터를 지원합니다. 속성 및 지원 되는 데이터 형식의 전체 목록은 참조 하십시오. [주석 속성](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d)합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 이 경고를 해결 하려면 다음 샘플 코드와 같이 포인터 또는 배열 형식를 사용 합니다.  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
