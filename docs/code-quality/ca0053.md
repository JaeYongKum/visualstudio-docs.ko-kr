# <a name="ca0053"></a>CA0053
규칙 어셈블리를 로드할 수 없습니다.

 이 리플렉션에 의해 규칙 어셈블리를 로드할 수 없음을 나타냅니다. 이 사용자에 게 사용자 지정 규칙을 제대로 빌드되지 않은 경우 발생할 수 또는 파일 시스템 액세스가 거부 되었습니다와 같은 분석 도구는 다른 이유로 어셈블리 로드할 수 없습니다.
