# C28226
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28226: 주석이 PrimOp일 수는 없습니다\(현재 선언에서\).  
  
 이 경고는 지정한 주석 일반적인 주석으로 선언하고도 미리 선언된 PrimOp으로 나타냅니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
