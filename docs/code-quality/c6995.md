# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C6995: XML 로그 파일을 저장 하지 못했습니다.  
  
 이 경고는 코드 분석 도구는 코드 분석 출력 되는 오류 로그를 만들 수 없다는 것을 나타냅니다.  
  
 이 오류가 디스크 오류일 수 있는 지정된 디렉터리에 파일을 만들 수 있는 권한이 있는지를 나타냅니다.
