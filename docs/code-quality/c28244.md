# <a name="c28244"></a>C28244
C28244 경고: 함수에 대 한 주석 구문 분석할 수 없는 매개 변수/외부 주석이 있는  
  
 주석에 표준 매크로를 사용하는 경우에는 현재 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
