# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28212: 주석에 대한 포인터 식이 있어야 합니다.  
  
 이 경고는 주석 \(하지 주석이 추가 되는 기능\)에 번호가 매겨진된 매개 변수 포인터 또는 배열 형식 대신 다른 형식을 나타냅니다.  주석을 수정 해야 합니다.
