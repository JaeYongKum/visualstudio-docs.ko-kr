# C28209
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28209: 기호의 선언과 충돌하는 선언이 있습니다.  
  
 이 경고는 잘못 생성 된 주석 선언을 나타냅니다.  일반적인 사용에서는 이 경고가 발생 하지 않습니다.
