# C28216
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28216: \_Check\_return\_ 주석은 특정 함수 매개 변수에 대한 사전 조건에만 적용됩니다.  
  
 `_Check_return_`  이외의 다른 컨텍스트에서 적용 된 주석  `__post` . 할 수 있는  `__post`  \(또는  `__drv_out` \) 한정자 또는 포함 될 수 있습니다.
