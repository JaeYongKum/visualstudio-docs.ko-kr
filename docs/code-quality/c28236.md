# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28236: 주석은 식에서 사용할 수 없습니다.  
  
 이 경고는 식 컨텍스트에서 주석으로 선언 된 함수를 사용 하려고 시도 했음을 나타냅니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
