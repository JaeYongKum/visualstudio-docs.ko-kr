# <a name="c28214"></a>C28214
경고 C28214: 특성 매개 변수 이름은 p1...p9여야 합니다.

 이 경고는 주석 선언을 생성할 때 매개 변수는 이름이 p1... 제한 나타냅니다 p9 합니다. 일반적인 사용의 경우 이 경고가 발생하지 않습니다.
