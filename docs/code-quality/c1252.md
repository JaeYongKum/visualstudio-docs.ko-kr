# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C1252: 플러그인 사이가 순환되거나 종속성이 없습니다. GUID가 필요합니다.  
  
 코드 분석 도구는 플러그인 종속성을 분석 하는 코드에 있는 내부 오류가 발생 하는 경우 이 경고를 보고 합니다.
