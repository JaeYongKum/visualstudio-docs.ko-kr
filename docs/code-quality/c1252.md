# <a name="c1252"></a>C1252
C1252 경고: 플러그 인 사이 원형 또는 누락 된 종속성: GUID 필요  
  
 코드 분석 도구는 분석 중인 코드가 아니라 플러그 인 종속성에 내부 오류가 있는 경우에 이 경고를 보고합니다.
