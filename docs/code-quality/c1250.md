[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C1250: 플러그 인을 로드할 수 없습니다.  
  
 코드 분석 도구는 내부 도구에서 오류가 있고 분석되는 코드에 있지 않는 경우 이 경고를 보고 합니다.
