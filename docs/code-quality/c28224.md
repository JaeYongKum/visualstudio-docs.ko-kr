# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28224: 주석에 매개 변수가 필요합니다.  
  
 이 경고는 지정한 주석 매개 변수 없이 사용하고 하나 이상의 매개 변수가 필요 나타냅니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
