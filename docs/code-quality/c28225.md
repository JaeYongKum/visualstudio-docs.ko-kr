# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28225: 주석에서 필요한 수의 매개 변수를 찾을 수 없습니다.  
  
 이 경고는 잘못 된 매개 변수 개수를 사용 하 여 지정한 주석 사용 됨을 나타냅니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
