# <a name="ca0059"></a>CA0059
명령줄 스위치 오류가 발생 했습니다.

 이 오류가 발생 하는 경우 필요한 명령줄 스위치가 없는 경우 예를 들어

 /out 또는 /console 스위치를 지정 하지 않으면 합니다. 이 오류는 잘못 된 제한 시간 값 등의 잘못 된 스위치, 없는 경우에 발생 합니다.
