# <a name="ca0059"></a>CA0059
명령줄 스위치 오류가 발생 했습니다.

 이 오류가 발생할 경우 필수 명령줄 스위치 없으면, 예를 들어,

 /out 또는 /console 스위치를 지정 하지 않습니다. 이 오류는 잘못 된 스위치, 잘못 된 시간 제한 값과 같은 경우에 발생 합니다.
