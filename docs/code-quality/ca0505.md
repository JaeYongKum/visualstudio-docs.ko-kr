# <a name="ca0505"></a>CA0505
대체 속성이 정의 되어 있으므로 사용 되지 않는 속성은 무시 됩니다.

 경고 CA505 모두 지정 된 사용 되지 않으며 속성 프로젝트에서 값을 포함 또는 구성 파일을 작성 하는 경우에 생성 됩니다. 이 문제를 해결 하려면 사용 되지 않는 속성을 제거 합니다.
