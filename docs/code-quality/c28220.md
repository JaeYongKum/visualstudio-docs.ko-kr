# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28220: 주석에 있는 매개 변수 주석에 정수 식이 필요합니다.  
  
 이 경고는 주석에 매개 변수 정수 식을 해야 하 고 몇 가지 다른 형식이 나타냅니다.  일반적으로 잘못 코딩된 주석 매크로를 나타냅니다.
