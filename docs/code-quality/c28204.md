# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28204: \<function\> : 이 오버로드와 \<filename\>\(\<line\>\)의 오버로드 중 하나에만 \<paramname\>에 대한 주석이 추가되었습니다. 모두 주석을 추가하거나 모두 주석을 추가하지 않아야 합니다.  
  
 이 경고는 주석에 오류가 있을 때 보고 됩니다.
