# <a name="c28228"></a>C28228
경고 C28228: 주석 매개 변수: 주석에서 형식을 사용할 수 없습니다.

 이 경고는 지원 되지 않는 형식의 매개 변수가 임을 나타냅니다. 주석 형식의 제한 된 집합을 매개 변수로 사용 하 여만 수 있습니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
