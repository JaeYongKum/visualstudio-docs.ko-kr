# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28228: 주석 매개 변수: 주석에서 형식을 사용할 수 없습니다.  
  
 이 경고는 지원 되지 않는 형식 매개 변수 인지를 나타냅니다.  주석 형식의 제한 된 집합을 매개 변수로 사용할 수 있습니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
