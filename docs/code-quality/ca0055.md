# <a name="ca0055"></a>CA0055
파일을 로드할 수 없습니다:\<경로 >  
  
 코드 분석 도구는 분석을 위해 지정 된 대상 파일을 로드할 수 없습니다. 이 파일을 찾을 수 없습니다 하거나 파일에 대 한 액세스를 거부 한 경우 발생 합니다. 사용자의 입력된 어셈블리 속성을 재정의 하도록 선택 하는 경우 일반적으로 발생이 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], 또는 fxcopcmd.exe를 직접 호출 합니다.
