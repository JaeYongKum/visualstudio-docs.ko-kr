# <a name="ca0055"></a>CA0055
파일을 로드할 수 없습니다:\<경로 >

 코드 분석 도구는 분석을 위해 지정 된 대상 파일을 로드할 수 없습니다. 이 경우 파일이 없거나 파일에 대 한 액세스가 거부 되었습니다. 일반적으로이 사용자 Visual Studio에서 입력된 어셈블리 속성을 재정의 하도록 선택 하거나 수동으로 fxcopcmd.exe를 호출 하는 경우 발생 합니다.
