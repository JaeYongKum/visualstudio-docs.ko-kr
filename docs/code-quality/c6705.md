# C6705
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C6705: 주석 오류입니다. \<expected\_number\>개 인수가 \<parameter\> 주석에 필요한데 \<actual\_number\>개가 있습니다.  
  
 이 경고는 주석에 오류가 있을 때 보고 됩니다.
