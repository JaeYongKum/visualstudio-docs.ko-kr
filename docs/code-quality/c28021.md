# C28021
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28021: 주석 \<anno\>을 추가하고 있는 매개 변수 \<param\>은 포인터여야 합니다.  
  
 주석이 추가 되는 개체 포인터 형식이 아닌 경우이 경고가 보고 됩니다.  이 주석은 `void` 또는 정수 계열 형식과 사용할 수 없습니다.
