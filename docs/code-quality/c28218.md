# <a name="c28218"></a>C28218
C28218 경고: 함수 매개 변수는 매개 변수와 일치 하지 않습니다 파일에 있는

 현재 줄에 한 메시지에 대 한 줄에 주석을 일치 하지 않습니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
