# <a name="c28230"></a>C28230
C28230 경고: 매개 변수 형식에 멤버가 없습니다.  
  
 이 경고는 주석에 대한 인수가 `struct`, `class` 또는 `union`에 엑세스하려고 하고 명명된 구성원이 없음을 나타냅니다.
