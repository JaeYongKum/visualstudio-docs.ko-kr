# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28230: 매개 변수의 형식에 멤버가 없습니다.  
  
 이 경고는 주석에 대한 인수가 `struct`, `class`, 또는 `union` 에 엑세스 하려 하고 명명 된 구성원이 존재 하지 않음을 나타냅니다.
