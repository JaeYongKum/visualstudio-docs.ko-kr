# <a name="ca0056"></a>CA0056
프로젝트 파일에는 분석 도구 버전이 잘못 되었습니다.  
  
 FxCopCmd.exe 잘못 된 코드 분석 도구 버전을 보유 하는 프로젝트 파일을 사용 하 여 수동으로 호출 하는 경우이 경고가 발생 합니다.
