# C28245
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28245: 함수에 대한 주석에서 멤버가 아닌 함수에 'this'를 주석으로 추가합니다.  
  
 PREfast 모델 파일에 내부 오류가 발생 했습니다.  일반적인 사용법에서 이 경고가 발생하지 않아야 합니다.
