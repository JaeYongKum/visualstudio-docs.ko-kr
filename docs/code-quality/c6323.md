# <a name="c6323"></a>C6323
경고-C6323 부울 형식에 산술 연산자 사용  
  
 부울 데이터 형식에 산술 연산자를 사용 하는 경우이 경고가 발생 합니다. 잘못 된 연산자를 사용 하 여 잘못 된 결과가 발생할 수 있습니다. 또한 만들려는 프로그래머의 의도 코드에 반영 되지 않았음을 나타냅니다.  
  
## <a name="example"></a>예  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 이 경고를 해결 하려면 올바른 데이터 형식 및 연산자를 사용 합니다.  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
