# C28196
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28196: 충족하지 않은 요구 사항입니다. \(이 식은 true가 아닙니다.\)  
  
 이 경고는 분석 중인 함수에 대한 함수 프로토타입을 나타내는 것은 `__notnull`, `__null` 또는 `__drv_valueIs` 에 `_Out_` 주석의 일치 하지 않는 매개 변수 또는 반환 값은 있지만 값을 반환합니다.
