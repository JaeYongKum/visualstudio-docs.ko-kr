# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

28222 경고: 주석에 대 한 예상 \_No\_, \_Yes\_, \_Maybe\_  
  
 이 경고는 `_Yes_`, `_No_`, or `_Maybe_`을 나타내고 주석의 매개 변수는 문자열이 아닌 다른 형식이 발생 했음을 나타냅니다.  일반적으로 잘못 코딩된 주석 매크로를 나타냅니다.
