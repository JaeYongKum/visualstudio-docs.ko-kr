# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28223: 주석, 매개 변수에 대해 토큰\/식별자가 필요한데 없습니다.  
  
 이 경고는 주석의 매개 변수는 문자열이 아닌 다른 형식이 발생 했음을 나타냅니다.  일반적으로 잘못 코딩된 주석 매크로를 나타냅니다.  이 위치에서 C 또는 c \+ \+ 키워드를 사용 하 여가이 오류가 발생 합니다.
