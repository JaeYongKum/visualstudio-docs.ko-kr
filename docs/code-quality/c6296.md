# <a name="c6296"></a>C6296
경고 C6296: 루프가 잘못 정의 되었습니다: 루프 본문이 한 번만 실행  
  
 이 경고는 for 루프 의도 한 대로 작동 하지 않을 수를 나타냅니다. 인덱스 서명 되지 않은 루프를 0부터 아래로 계산 하는 경우 해당 본문 한 번만 실행 됩니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 이 경고를 해결 하려면 다음 코드를 사용 합니다.  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
