# <a name="c28311"></a>C28311
28311 경고: < annotation_name > 주석에 \<함수 > \<매개 변수 > SAL의 더 버전입니다.  
  
 주석이 이전 버전 및 해당 하는 업데이트 되어 [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)합니다. 이전의 비일관적 주석 경고가 표시된 경우에는 이 경고가 표시되지 않으며 선언당 한 번만 보고됩니다. 나머지 선언에서 사용하지 않는 SAL이 더 있는지 검사합니다.
