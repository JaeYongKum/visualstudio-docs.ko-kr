# C28311
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 28311: \<함수\> \<매개 변수\>의 \<annotation\_name\> 주석이 SAL의 사용되지 않는 버전입니다.  
  
 이 주석은 이전 버전이며 동급의 [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)로 업데이트해야 합니다.  이전의 비일관적 주석 경고가 표시된 경우에는 이 경고가 표시되지 않으며 선언당 한 번만 보고됩니다.  나머지 선언에서 사용하지 않는 SAL이 더 있는지 검사합니다.
