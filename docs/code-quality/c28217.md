# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 28217: 함수의 경우 주석에 대한 매개 변수 개수가 파일에 있는 개수와 일치하지 않습니다.  
  
 현재 줄에서 한 줄 메시지에 주석을 일치 하지 않습니다.  표준 매크로를 주석에 사용하는 경우 일반적인 사용에 이 경고가 나타나지 않습니다.
