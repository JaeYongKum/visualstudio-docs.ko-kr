# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28239: 매개 변수의 주석에 value, stringValue 또는 longValue와 paramn\=xxx가 모두 있습니다.  paramn\=xxx만 사용하십시오.  
  
 PREfast 모델 파일에 내부 오류가 발생 했습니다.  일반적인 사용법에서 이 경고가 발생하지 않아야 합니다.
