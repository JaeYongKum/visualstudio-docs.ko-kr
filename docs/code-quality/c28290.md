# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28290: 함수에 대한 주석에 실제 매개 변수 개수보다 많은 외부 참조가 있습니다.  
  
 코드 분석 도구 함수에 대 한 주석 실제 매개 변수 개수 보다 더 많은 외부 참조가 포함 되어 있는 경우이 경고를 보고 합니다.
