# <a name="c6292"></a>C6292
C6292 경고: 루프가 잘못 정의 되었습니다: 최대값부터 위로 계산  
  
 이 경고는 for 루프 의도 한 대로 작동 하지 않을 수를 나타냅니다.  
  
 루프가 최대값부터 위로 계산 되지만 낮은 종료 조건이 발생 합니다. 정수 오버플로가 발생 한 후에이 루프 종료 됩니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
void f( )  
{  
   int i;  
  
   for (i = 100; i >= 0; i++)  
   {  
      // code ...  
   }  
}  
```  
  
 이 경고를 해결 하려면 다음 코드를 사용 합니다.  
  
```  
void f( )  
{  
   int i;  
  
  for (i = 100; i >= 0; i--)  
   {  
      // code ...  
   }  
}  
```
