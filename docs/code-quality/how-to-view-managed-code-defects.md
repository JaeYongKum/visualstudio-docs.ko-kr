# 방법: 관리 코드 오류 보기
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

코드 분석 오류, 경고 및 메시지 코드 분석 창에 보고 됩니다.  프로젝트 또는 솔루션에 코드 분석을 실행할 코드 분석 창이 자동으로 나타납니다.  
  
 분석 코드 창에서 다음을 수행할 수 있습니다.  
  
-   **경고 제목을 선택 합니다.**  
  
     경고에 대 한 자세한 내용은 코드 분석 창에 표시 됩니다.  경고가 발생 하는 소스 파일의 줄에 표시 되 고 소스 코드 편집기에서 강조 표시 됩니다.  
  
-   **경고 id를 선택 합니다.** 경고 id가 옵니다  **CA** 4 자리 숫자를 포함 하 고 있습니다.  예를 들면 CA1234과 같습니다.  
  
     경고에 대 한 MSDN Library 도움말 항목이 표시 됩니다.  도움말 항목 경고의 원인에 대 한 자세한 정보가 들어 및 경고 문제 발생 및 해결 방법을 보여 주는 예제가 포함 되어 있습니다.
