# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28301: 함수 \<function\>.\<note1\>함수의 첫 번째 선언에 대한 주석이 없습니다. \<filename\>\(\<line\>\). \<note2\>를 참조하십시오.  
  
 주어진 기능의 첫 번째 선언에서 주석을 찾을 수 없는 경우 이 경고가 보고 됩니다.
