# C28310
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 28310: \<함수\> \<매개 변수\>의 \<annotation\_name\> 주석에 SAL 버전이 없습니다.  
  
 소스 코드에 사용된 모든 SAL 주석은 SAL\_name 용도로 적용된 주석 버전이 있어야 합니다.  매크로 정의에서 수정해야 합니다.  
  
 이 경고는 선언당 한 번만 보고됩니다.  나머지 선언에서 사용하지 않는 SAL이 더 있는지 검사합니다.
