# C28210
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 28210: `_On_failure_` 컨텍스트에 대한 주석이 명시적 사전 컨텍스트에 없어야 합니다.  
  
 주석  `_On_failure_`  명시적 또는 암시적으로 표시 되고  `__post`  컨텍스트, 즉, 함수가 반환 된 후에 적용 될 수 있습니다.   `_drv_out`  이 되도록 사용합니다.
