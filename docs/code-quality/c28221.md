# <a name="c28221"></a>C28221
C28221 경고: 문자열 식이 주석에 있는 매개 변수에 대해 필요 합니다.  
  
 이 경고는 주석의 매개 변수가 문자열이어야 하는데 다른 형식이 발견되었음을 나타냅니다. 이 경고는 대개 잘못 코딩된 주석 매크로를 나타내며 일반적인 사용의 경우 표시되지 않습니다.
