# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28221: 주석에 있는 매개 변수에 문자열 식이 필요합니다.  
  
 이 경고는 주석의 매개 변수는 문자열이 아닌 다른 형식이 발생 했음을 나타냅니다.  이것은 일반적으로 잘못 코딩된 주석 매크로를 나타내고 일반적인 사용에서 볼 수 없습니다.
