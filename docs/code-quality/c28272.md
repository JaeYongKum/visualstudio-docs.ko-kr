# C28272
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28272: 함수, 매개 변수에 대한 주석이 검사 시 함수 선언과 일치하지 않습니다.  
  
 이 경고는 분석되지않은, 코드에서가 아닌 주석에서의 오류를 나타냅니다.  함수 정의에 나타나는 주석은 선언에 나타나는 내용과 일치 하지 않습니다.  두 주석은 일치 되야 합니다.
