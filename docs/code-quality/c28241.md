# <a name="c28241"></a>C28241
warning C28241: 매개 변수에 대한 함수 주석이 인식되지 않습니다.

 인식할 수 없는 주석 이름을 사용 했습니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
