# <a name="c28241"></a>C28241
C28241 경고: 매개 변수에 대 한 함수 주석이 인식 되지 않습니다

 주석을 인식할 수 없는 이름을 사용 했습니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
