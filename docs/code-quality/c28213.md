# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28213: 이전 선언을 수정하지 않고 참조하려면 `_Use_decl_annotations_` 주석을 사용해야 합니다. \<why\>  
  
 요구된 `_Use_decl_annotations_` 주석 없이 이전 선언이 참조될 때 경고는 보고됩니다.
