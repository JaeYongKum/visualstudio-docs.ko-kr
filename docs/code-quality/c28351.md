# C28351
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28351: \<annotation\> 주석은 동적 값\(변수\)을 조건에 사용할 수 없는 위치를 설명합니다.  
  
 C\/c \+ \+ 형식을 검사 하 고 주석의 적용 될 때이 경고가 보고 됩니다.  
  
 식에는  `_When_`  상수를 평가 해야 합니다.  `_When_`가 무시됩니다.
