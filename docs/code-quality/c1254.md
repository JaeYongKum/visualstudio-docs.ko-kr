# C1254
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C1254: 플러그 인 버전 불일치: 버전이 PREfast 드라이버의 버전과 일치하지 않습니다.  
  
 코드 분석 도구는 코드가 아닌 플러그인 버전에서 내부오류가 발생하는 경우 이 경고를 보고 합니다.
