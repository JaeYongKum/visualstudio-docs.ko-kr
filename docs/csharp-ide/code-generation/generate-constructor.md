# <a name="generate-a-constructor-in-c"></a>C#에 생성자를 생성 합니다. #
**:** 즉시 클래스에 새로운 생성자에 대 한 코드를 생성할 수 있습니다. 

**경우:** 제대로 자동으로 선언 하 고 새 생성자를 추가 하거나 기존 생성자를 수정 합니다. 

**이유:** 이 기능에서, 적절 한 매개 변수를 자동으로 생성 되지만 생성자를 사용 하기 전에 선언할 수 있습니다. 또한 기존 생성자를 수정 하려면이 기능을 사용 하 여 자동으로 업데이트 하지 않으면 모든 callsite를 업데이트 합니다.

**방법:** 생성자를 생성 하는 방법은 여러 가지가 있습니다.
- [생성자를 생성 하 고 멤버를 선택 합니다.](#pick)
- [선택한 필드에서 생성자를 생성 합니다.](#selection)
- [새 사용에서 생성자를 생성 합니다.](#usage)
- [기존 생성자에 매개 변수를 추가 합니다.](#addparameter)
- [만들기 및 필드/속성은 생성자 매개 변수에서 초기화](#create)

## <a id = "pick"></a>생성자를 생성 하 고 멤버를 선택 합니다.
1. 클래스에서 모든 빈 줄에 커서를 놓습니다.

   ![빈 줄에 커서](media/constructor1_highlight.png)

1. 다음으로, 다음 중 하나를 수행 합니다.
   * **키보드**
     * 키를 눌러 **Ctrl +.** 트리거에 **빠른 작업 및 리팩터링** 메뉴와 선택 **생성 생성자 중...**  미리 보기 창 팝업에서 합니다.
   * **마우스**
     * 마우스 오른쪽 단추로 클릭 하 고 선택 된 **빠른 작업 및 리팩터링** 메뉴와 선택 **생성 생성자 중...**  미리 보기 창 팝업에서 합니다.
     * 클릭 하 고 ![전구](media/bulb.png) 커서가 있는 경우 텍스트 이미 클래스에 빈 줄에 왼쪽된 여백에 표시 되는 아이콘입니다.

   ![생성자 미리 보기를 생성 합니다.](media/constructor1_preview.png)

1. 생성자 매개 변수로 사용할 수 있을 뿐만 아니라 (위쪽 또는 아래쪽 화살표)를 정렬 하려면 멤버를 선택할 수 있도록 대화 상자가 나타납니다. 키를 눌러 **확인** 변경 내용을 적용 하려면.
  
   ![멤버 대화 선택](media/constructor1_dialog.png)

   >[!TIP] 
   >확인할 수 있습니다는 **null 검사 추가** 생성자 매개 변수를 확인 합니다. null을 자동으로 생성 하려면 확인란을 선택 합니다.

1. 지정 된 순서로 선택한 매개 변수를 가진 생성자 만들어질 수 있습니다.
   ![생성자 결과 생성 합니다.](media/constructor1_result.png)

## <a id="selection"></a>선택한 필드에서 생성자를 생성 합니다.
1. 생성 된 생성자에서을 두려는 구성원을 강조 표시: ![구성원을 강조 표시](media/constructor2_highlight.png)

1. 다음으로, 다음 중 하나를 수행 합니다.
   * **키보드**
     * 키를 눌러 **Ctrl +.** 트리거에 **빠른 작업 및 리팩터링** 메뉴와 선택 **생성자 생성 'TypeName(...)'**  미리 보기 창 팝업에서 합니다.
   * **마우스**
     * 마우스 오른쪽 단추로 클릭 하 고 선택 된 **빠른 작업 및 리팩터링** 메뉴와 선택 **생성자 생성 'TypeName(...)'**  미리 보기 창 팝업에서 합니다.
     * 클릭 하 고 ![전구](media/bulb.png) 커서가 있는 경우 텍스트 이미 선택 항목으로 줄에 왼쪽된 여백에 표시 되는 아이콘입니다.

     ![생성자 미리 보기를 생성 합니다.](media/constructor2_preview.png)

1. 선택한 매개 변수를 가진 생성자를 만들어질 수 있습니다.
     ![생성자 결과 생성 합니다.](media/constructor2_result.png)

   ![강조 표시 된 코드](media/constructor_highlight.png)

1. 다음으로, 다음 중 하나를 수행 합니다.
   * **키보드**
     * 키를 눌러 **Ctrl +.** 트리거에 **빠른 작업 및 리팩터링** 메뉴와 선택  **에 생성자가 생성 '*TypeName*' * * 미리 보기 창 팝업에서 합니다.

     * 빨간색 물결 기호 위로 마우스를 클릭 하 고 ![전구](media/bulb.png) 표시 되는 아이콘입니다.
     * 클릭 하 고 ![전구](media/bulb.png) 커서가 있는 경우 텍스트 이미 빨간색 물결 기호는 줄에 왼쪽된 여백에 표시 되는 아이콘입니다.

   >[!TIP]
   >사용 하 여는 [ **변경 내용 미리 보기가** ](../../ide/preview-changes.md) 모두 선택 하기 전에 적용 될 변경 내용을 보려면 미리 보기 창의 맨 아래에 링크 합니다.

1. 생성자의 사용법에서 유추 된 모든 매개 변수와 함께 자동으로 만들어집니다.

## <a id="create"></a>만들기 및 필드/속성은 생성자 매개 변수에서 초기화
1. 기존 생성자에서 매개 변수를 추가 합니다.

   ![생성자 강조 표시를 생성 합니다.](media/constructor5_highlight.png)

1. 새로 추가한 매개 변수 안에 커서를 놓습니다.

1. 다음으로, 다음 중 하나를 수행 합니다.
   * **키보드**
     * 키를 눌러 **Ctrl +.** 트리거에 **빠른 작업 및 리팩터링** 메뉴와 선택 **만들기 및 초기화 속성 'YourProperty'** 또는 **만들기 및 초기화 필드 'YourField'** 에서 미리 보기 창이 팝업 됩니다.
   * **마우스**
     * 마우스 오른쪽 단추로 클릭 하 고 선택 된 **빠른 작업 및 리팩터링** 메뉴와 선택 **만들기 및 초기화 속성 'YourProperty'** 또는 **'YourField'필드만들기및초기화**미리 보기 창 팝업에서 합니다.
     * 클릭 하 고 ![전구](media/bulb.png) 커서가 있는 경우 텍스트 이미 추가 된 매개 변수와 함께 줄에 왼쪽된 여백에 표시 되는 아이콘입니다.

   ![생성자 미리 보기를 생성 합니다.](media/constructor5_preview.png)

1. 필드/속성 만들어지고 프로그램 종류와 일치 하도록 자동으로 이름이 지정 됩니다.

   ![생성자 결과 생성 합니다.](media/constructor5_result.png)
