## <a name="deployment-editions"></a>배포 버전  
 [!INCLUDE[dsl](../modeling/includes/dsl_md.md)]에서는 작성하는 DSL(Domain-Specific Language)을 배포하기 위한 다음 구성을 지원합니다.  
  
-   Visual Studio Enterprise  
  
-   Visual Studio Professional  
  
-   Visual Studio Shell (통합된 모드) 재배포 가능 패키지  
  
-   Visual Studio Shell(격리 모드) 재배포 가능 패키지  
  
> [!NOTE]
>  DSL Shell 제품에서 실행할 수 있도록 설정 해야는 **지원 되는 VS 버전** 확장 매니페스트에서 필드입니다. 자세한 내용은 참조 [도메인별 언어 솔루션 배포](../modeling/deploying-domain-specific-language-solutions.md)합니다.
