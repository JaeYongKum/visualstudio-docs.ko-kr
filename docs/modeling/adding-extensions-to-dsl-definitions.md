# <a name="adding-extensions-to-dsl-definitions"></a>DSL 정의에 확장 추가
DSL 정의 확장을 사용 하면 도메인별 언어 (DSL)에 대 한 확장의 패키지를 만들 수 있습니다. 에 Visual Studio 통합 확장 (VSIX) 포함 된 DSL 확장 DSL로 같은 방식으로 사용자의 컴퓨터에 설치할 수 있습니다. 추가 기능은 동적으로 활성화 하 고 수에서 런타임에 사용할 수 없습니다. Dsl 확장을 위해 명시적으로 디자인 될 필요가 없습니다 되며 확장 확장된 DSL을 변경 하지 않고에 나중에 또는 제&3; 자가 설계할 수 있습니다.  
  
 추가 기능은 다음과 같습니다.  
  
-   모델 및 프레젠테이션 요소에 대 한 속성  
  
-   모양 및 연결선에 대 한 데코레이터  
  
-   클래스, 관계, 모양 및 연결선  
  
-   유효성 검사 제약 조건  
  
-   탭 및 도구 상자 항목  
  
 확장된 하는 DSL의 사용자를 만들어 추가 기능 인스턴스를 포함 하는 모델을 저장 하 고 적절 한 확장을 설치한 다른 사용자가 읽을 수 있습니다. 확장을 설치 하지 않은 사용자가 추가 기능을를 사용할 수 없지만 업데이트 하 고 추가 기능을 손실 하지 않고 모델을 저장할 수 있습니다.
