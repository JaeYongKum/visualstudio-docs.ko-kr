# <a name="api-reference-for-t4-text-templates"></a>T4 텍스트 템플릿에 대한 API 참조
텍스트 템플릿 API를 사용 하면 호출 하 고의 변환을 사용자 지정할 [텍스트 템플릿](../modeling/code-generation-and-t4-text-templates.md)합니다.  
  
## <a name="namespaces"></a>네임스페이스  
  
|네임스페이스|용도|  
|---------------|-------------|  
|<xref:Microsoft.VisualStudio.TextTemplating>|텍스트 템플릿 변환 기능에 대 한 클래스를 포함합니다. 텍스트 템플릿 변환 엔진은 Visual Studio에 통합 하 고 생성 된 텍스트 출력 파일에 텍스트 템플릿 파일을 변환 합니다.|  
|<xref:Microsoft.VisualStudio.TextTemplating.Modeling>|텍스트를 UML 모델 및 도메인 특정 언어에 대 한 액세스와 관련 된 변환 기능 제공 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ModelBus 합니다.|  
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost>|텍스트 템플릿 서비스에 대 한 액세스를 제공 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]합니다.|
