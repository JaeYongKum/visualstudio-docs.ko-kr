# <a name="understanding-coauthoring-and-add-ins"></a>공동 이해 및 추가 기능

다른 사용자와 동시에 (즉, OneDrive, 비즈니스용 OneDrive, 또는 SharePoint Online)는 클라우드에서 호스팅되는 통합 문서를 편집할 수 있는 Excel for Windows desktop 공동 기능은입니다. 이 기능을 추가 하 여 작동 하는지 확인 하는 동안에 사용자는 공동 하기 위해 Excel 솔루션을 조정 해야 합니다. 참조 [Excel에서 공동 작업에 대 한](https://msdn.microsoft.com/vba/excel-vba/articles/about-coauthoring-in-excel)합니다.

참고: Windows 데스크톱 Excel에 대 한 공동 2017에 도입 된 및 office 365 구독자에 대해 현재 사용할 수 있습니다. Office 2016 또는 이전 정품 라이선스를 구입한 사용자 현재 없는 공동 기능에 액세스할 수 있습니다. (Excel Online Excel for Android, ios의 경우 Excel 및 Excel Mobile Windows 저장소에도 지원 공동).
