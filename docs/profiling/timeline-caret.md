# 시간 표시 막대 캐럿
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

이 캐럿은 시간 표시 막대에서 실행 중인 스레드 세그먼트를 선택할 경우 이 세그먼트 위에 표시됩니다.  현재 스택 탭에 표시된 호출 스택은 해당 세그먼트를 클릭한 지점에서 시간적으로 가장 가까운 호출 스택입니다.  캐럿은 샘플링 된때에 인스턴스와 함께 **현재** 탭 아래에 표시 되는 호출 스택에 연결하기 위해 사용됩니다.  캐럿은 사용자가 선택한 위치에 가장 가까운 호출 스택의 정확한 위치를 보여 줍니다.
