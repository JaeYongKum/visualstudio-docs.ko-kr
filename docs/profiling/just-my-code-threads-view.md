# 내 코드만(스레드 뷰)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

이 옵션을 선택하면 사용자 코드와 한 수준의 호출된 함수만 표시되도록 호출 스택이 필터링됩니다.  
  
 이 옵션을 활성화하면 호출 스택의 복잡성을 상당히 줄일 수 있으므로 특정 문제를 보다 쉽게 진단할 수 있습니다.  
  
 경우에 따라 이 옵션을 선택하면 차단 호출이 필터링될 수도 있습니다.  결정하는 데 전체 호출 스택 정보가 필요한 경우에는 이 옵션의 선택을 취소하여 전체 호출 스택을 표시합니다.
