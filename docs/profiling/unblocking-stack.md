# 차단 해제 스택
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

현재 선택된 스레드 요소가 차단된 세그먼트를 나타내며 이 세그먼트가 나중에 현재 프로세스에서 다른 스레드에 의해 차단 해제된 후 실행되기 시작한 경우, 차단 해제를 수행한 스레드의 호출 스택이 이 탭에 표시됩니다.
