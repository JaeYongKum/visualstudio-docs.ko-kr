# <a name="unblock-stack"></a>스택 차단 해제
현재 선택된 스레드 요소가 현재 프로세스의 다른 스레드에 의해 차단이 해제된 후 나중에 실행되기 시작한 차단된 세그먼트를 나타내는 경우, 차단 해제를 수행한 스레드의 호출 스택이 이 탭에 표시됩니다.
