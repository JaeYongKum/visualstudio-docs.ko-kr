# 코어 뷰 확대/축소 도구
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

확대\/축소 컨트롤은 관심 있는 특정 영역에 초점을 맞출 수 있도록 시간 표시 막대를 확대 및 축소하는 데 사용하는 슬라이더입니다.  이 컨트롤은 현재 뷰의 가운데를 확대하므로 확대하기 전에 뷰를 조정합니다.  
  
## 다른 확대\/축소 방법  
  
-   확대: 시간 표시 막대 뷰의 영역을 마우스로 끌어 선택합니다.  마우스 단추를 놓으면 시간 표시 막대 뷰에서 선택한 범위가 확대됩니다.  
  
-   확대 및 축소: 시간 표시 막대를 마우스로 가리킨 다음 Ctrl 키를 누른 채 마우스 휠을 움직입니다.
