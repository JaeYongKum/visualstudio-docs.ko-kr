# <a name="markerserieswritealert-method"></a>marker_series::write_alert 메서드
동시성 시각화 도우미 추적 파일에 경고를 씁니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
void write_alert(  
   _In_ LPCTSTR _Format,  
   ...  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `_Format`  
 인수 목록의 개체에 해당하는 0개 이상의 서식 항목과 결합된 텍스트를 포함하는 합성 서식 문자열입니다.
