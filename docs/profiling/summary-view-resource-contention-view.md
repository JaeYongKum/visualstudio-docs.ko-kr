# 요약 뷰 - 프로파일러 리소스 경합 뷰
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

요약 뷰에는 스레드 또는 프로세스가 리소스에 액세스할 수 있기를 기다리면서 중지되었던 응용 프로그램 이벤트에 대한 정보가 표시됩니다.  
  
 알림 링크 및 보고서 목록에 대한 설명을 비롯한 자세한 내용은 [요약 뷰](../profiling/summary-view.md)를 참조하십시오.  
  
## 시간 표시 그래프  
 요약 뷰의 시간 표시 그래프에서는 프로파일링이 발생한 시간 동안 프로파일링된 응용 프로그램에서의 경합 이벤트 수를 보여 줍니다.  시간 표시 그래프를 사용하여 선택한 시간 범위로 뷰를 필터링할 수 있습니다.  자세한 내용은 [방법: 요약 시간 표시 막대에서 보고서 뷰 필터링](../Topic/How%20to:%20Filter%20Report%20Views%20from%20the%20Summary%20Timeline.md)을 참조하십시오.  
  
## 경합이 가장 많은 리소스  
 **경합이 가장 많은 리소스**는 응용 프로그램에서 경합 이벤트를 가장 많이 발생시킨 리소스를 나열합니다.  리소스 이름을 클릭하여 경합 뷰를 표시할 수 있습니다.  경합 뷰에서는 스레드에 의한 리소스 경합의 자세한 시간 표시 막대를 제공합니다.  
  
 **경합이 가장 많은 리소스**에는 각 리소스에 대한 다음 데이터가 포함됩니다.  
  
|열|설명|  
|-------|--------|  
|**Name**|리소스의 이름입니다.|  
|**경합 비율\(%\)**|프로파일링 데이터의 전체 경합 이벤트 중 해당 리소스에 대한 경합이었던 이벤트의 백분율입니다.|  
  
## 경합이 가장 많은 스레드  
 **경합이 가장 많은 스레드**는 응용 프로그램에서 경합 이벤트를 가장 많이 발생시킨 스레드를 나열합니다.  스레드 이름을 클릭하면 해당 스레드에 의한 리소스 경합의 자세한 시간 표시 막대를 제공하는 경합 뷰가 표시됩니다.  
  
 **경합이 가장 많은 스레드**에는 각 스레드에 대한 다음 데이터가 포함됩니다.  
  
|열|설명|  
|-------|--------|  
|**ID**|스레드 식별자입니다.|  
|**Name**|해당 스레드를 소유하는 프로세스의 이름입니다.|  
|**경합 비율\(%\)**|프로파일링 데이터의 전체 경합 이벤트 중 해당 리소스에 대한 경합이었던 이벤트의 백분율입니다.|
