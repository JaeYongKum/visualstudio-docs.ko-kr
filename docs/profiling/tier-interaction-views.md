# 계층 상호 작용 뷰
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

계층 상호 작용 프로파일링에서는 [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)]을 통해 데이터베이스와 통신하는 다중 계층 응용 프로그램의 함수 실행 시간에 대한 추가 정보를 제공합니다.  동기 함수 호출에 대한 데이터만 수집됩니다.  모든 프로파일링 방법에 계층 상호 작용 데이터를 포함할 수 있습니다.
