# 코어 뷰 시간 표시 막대
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

시간 표시 막대의 각 행은 프로파일링된 시스템의 논리 프로세서 코어를 나타냅니다.  각 행에 대해 가로 축은 특정 시점에 논리 코어에서 실행 중이었던 스레드를 보여 줍니다.  시간 표시 막대에서 관심 있는 색을 마우스로 가리키면 해당 스레드를 식별하는 도구 설명이 표시됩니다.  스레드 식별에 유용하도록 창의 맨 아래에 있는 범례에는 각 색이 나타내는 스레드가 표시됩니다.  클릭하여 끌거나 Ctrl 키를 누르고 마우스 휠을 움직이는 방법으로 확대\/축소 도구를 사용하여 확대\/축소할 수 있습니다.  코드 뷰와 스레드 뷰 사이를 전환할 때도 확대\/축소 일관성이 유지됩니다.
