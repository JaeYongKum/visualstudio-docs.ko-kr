# 범주가 여러 개 있는 경우
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

특정 시간 범위 동안 둘 이상의 작업 유형이 포함된 구역이 선택되었습니다.  이때 두 가지 색이 세로로 나란히 표시됩니다.  이러한 색은 특정 시간 범위 동안 가장 빈도가 높은 상위 두 개의 작업만을 나타냅니다.  이 두 작업의 빈도는 상대적으로 세로 막대의 1\/3 또는 1\/2를 차지하는 것으로 표시됩니다.  보다 자세히 보려면 스레드 뷰의 이 구역을 확대해 봅니다.
