# 표시되는 시간 범위를 기반으로 하는 보고서
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

프로필 뷰에는 현재 표시된 시간 범위 및 채널을 기반으로 하는 보고서가 표시됩니다.  데이터의 다른 하위 집합에 대한 세부 정보를 보려면 범례에서 항목을 클릭합니다.  
  
 그러면 테이블의 데이터에 대한 세부 정보를 볼 수 있습니다.
