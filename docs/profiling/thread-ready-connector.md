# 스레드 준비 커넥터
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

차단 세그먼트를 클릭하여 호출 스택과 해당 차단 해제 스택을 볼 경우 스레드 준비 커넥터도 표시될 수 있습니다.  차단 해제 이벤트가 현재 프로세스의 다른 스레드에서 발생한 경우, 스레드 준비 커넥터는 차단된 스레드에서 실행을 다시 시작할 수 있도록 한 스레드 및 실행 세그먼트를 시각적으로 식별합니다.
