# 현재 탭
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**현재** 탭을 클릭하면 시간 표시 막대에서 현재 선택 포인트에 가장 가까이 있는 호출 스택\(사용 가능한 경우\)을 볼 수 있습니다.  이 경우에 선택 포인트는 시간 표시 막대 위에 검정 화살표나 캐럿으로 표시됩니다.  차단되는 세그먼트가 선택되어 있으면 이 세그먼트에는 실행 항목이 없었으므로 캐럿이 표시되지 않습니다.  그러나, 세그먼트는 여전히 강조 표시 하 고 호출 스택이 표시 됩니다.  
  
 **현재** 탭 DirectX 활동 세그먼트, 마커 및 I\/O 액세스 하는 방법에 대 한 정보도 표시 합니다.  DirectX 활동 세그먼트에 대 한 DMA 패킷이 하드웨어 큐에서 처리 하는 방법에 대 한 정보가 표시 됩니다.  표식에 대 한 설명 및 표식 유형에 대 한 정보가 표시 됩니다.  I\/O 액세스를 위해 파일을 읽거나 쓴 바이트 수에 대 한 정보가 표시 됩니다.
