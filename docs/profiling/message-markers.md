# 메시지 표식
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

메시지 표식 로그 출력을 나타냅니다.  메시지를 특정 시간에 특정 스레드를 생성 하는 문자열입니다.  다른 도구와 함께 사용 하기 위해 텍스트 파일에 메시지를 내보낼 수 있습니다.  메시지 문자열을 보려면 동시성 시각화 도우미에서 메시지에 포인터를 놓으면 수 있습니다.  메시지 마커를 볼 수 있는 [표식 보고서](../profiling/markers-report.md)을 볼수 있습니다.  다음 그림은 유효성 검사 오류 메시지를 보여 줍니다.  
  
## 메시지 집계 마커  
 플래그는 개별적으로 되지 않는 시각화 도우미에서 다른 하나와 가깝게 발생합니다.  이 경우는  *메시지 집계 마커* 를 나타내는 내부 메시지 표시 됩니다.  이러한 아이콘 중 하나에 포인터를 놓으면 도구 설명이 표시 되는 기본 메세지 수를 나타냅니다.  메시지를 보려면 확대 하십시오. 내부에서 메시지를 볼 수까지 확대하고 [표식 보고서](../profiling/markers-report.md)을 볼 수 있습니다.
