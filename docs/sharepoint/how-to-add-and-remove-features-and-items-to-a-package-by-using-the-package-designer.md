# 방법: 패키지 디자이너를 사용하여 패키지에 기능과 항목 추가 및 제거
  SharePoint 솔루션을 만들면 기본 SharePoint 기능이 솔루션의 패키지에 추가됩니다.  최종 배포하기 전에 SharePoint 프로젝트 항목과 기능을 추가 및 제거하여 SharePoint 패키지를 수정할 수 있습니다.  
  
 또는 패키징 탐색기를 사용하여 SharePoint 프로젝트 항목을 추가 및 제거할 수 있습니다.  패키지\(.wsp\)에 포함된 SharePoint 프로젝트 항목 및 기능의 계층 구조를 보고 변경할 수도 있습니다.  자세한 내용은 [방법: 패키징 탐색기를 사용하여 패키지에 기능과 항목 추가 및 제거](../sharepoint/how-to-add-and-remove-features-and-items-to-a-package-by-using-the-packaging-explorer.md)을 참조하십시오.  
  
## SharePoint 패키지에 기능 추가  
 패키지 디자이너를 사용하여 SharePoint 패키지에 기능을 추가할 수 있습니다.  
  
#### 패키지 디자이너를 사용하여 SharePoint 기능을 추가하려면  
  
1.  **패키지 디자이너**를 엽니다.  
  
     자세한 내용은 [방법: SharePoint 솔루션 패키지 사용자 지정](../sharepoint/how-to-customize-a-sharepoint-solution-package.md)을 참조하십시오.  
  
2.  다음 단계 중 하나 이상을 수행하여 하나 이상의 SharePoint 기능을 추가 합니다.  
  
    1.  **솔루션에서의 항목** 목록에서 추가할 각 항목을 더블 클릭합니다.  
  
    2.  추가할 항목을 선택한 다음 **추가** 버튼 \(\>\) 을 선택합니다.  
  
    3.  **모두 추가** 버튼 \(\>\>\) 을 선택하여 한 번에 모든 항목을 추가합니다.  
  
     예를 들어 **솔루션에 있는 항목** 목록에서 항목을 두 번 클릭하여 **패키지에 있는 항목** 목록으로 추가할 수 있습니다.  
  
     SharePoint 프로젝트 항목과 기능은 **패키지에 있는 항목** 목록에 표시됩니다.  
  
## SharePoint 패키지에서 기능 제거  
 패키지 디자이너를 사용하여 SharePoint 패키지에서 기능을 제거할 수 있습니다.  
  
#### 패키지 디자이너를 사용하여 SharePoint 기능을 제거하려면  
  
1.  **패키지의 항목** 목록에서, 제거할 항목을 선택한 다음 **제거** \(\<\) 버튼을 선택하거나, 또는 **모두 제거** 버튼 \(\<\<\) 을 선택하여 모든 항목을 제거합니다.  
  
     SharePoint 항목은 **솔루션에 있는 항목** 목록에 표시됩니다.
