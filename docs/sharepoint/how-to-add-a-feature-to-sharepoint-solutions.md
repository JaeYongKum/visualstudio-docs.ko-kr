# 방법: SharePoint 솔루션에 기능 추가
  SharePoint 솔루션을 만들면 각각 하나의 기능과 패키지가 솔루션에 추가됩니다.  기능 항목을 더 추가하여 여러 형식의 SharePoint 파일을 구성할 수 있습니다.  
  
### 솔루션 탐색기를 통해 기능을 추가하려면  
  
1.  **솔루션 탐색기**에서 **기능**의 바로 가기 메뉴를 열고 **기능 추가**를 선택합니다.
