# 방법: 사용자 지정 속성 추가
  **속성 편집기**를 사용하여 사용자 지정 속성을 모델에 추가할 수 있습니다.  런타임에 연결 문자열, 기타 데이터 등의 정보를 검색하기 위해 코드에서 이러한 속성에 액세스할 수 있습니다.  
  
### 사용자 지정 속성을 추가하려면  
  
1.  **BDC 탐색기**에서 사용자 지정 속성을 적용할 모델 요소를 나타내는 노드를 선택합니다.  
  
2.  메뉴 모음에서 **보기**, **속성 창**을 선택합니다.  
  
3.  **속성** 창에서 **사용자 지정 속성** 속성을 선택하고 줄임표 버튼\(![ASP.NET 모바일 디자이너 줄임표](../sharepoint/media/mwellipsis.png "ASP.NET 모바일 디자이너 줄임표")\)을 선택합니다.  
  
     **속성 편집기** 대화 상자가 나타납니다.  
  
4.  텍스트 상자의 **이름** 열에서 속성의 이름을 지정 합니다.  
  
5.  사용자 지정 속성의 **형식** 필드에서 적합한 데이터 형식을 선택합니다.  
  
6.  사용자 지정 속성의 **값** 필드에서 값을 지정한 다음 **확인** 버튼을 선택합니다.
