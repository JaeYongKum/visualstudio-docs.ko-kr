# 방법: SharePoint 연결 추가 또는 제거
  서버 탐색기를 사용하여 SharePoint 사이트와 데이터 연결을 찾아볼 수 있습니다.  하지만 SharePoint 사이트의 내용을 찾아보려면 먼저 **SharePoint 연결** 노드에 추가해야 합니다.  
  
### SharePoint 연결 노드에 SharePoint 사이트를 추가하려면  
  
1.  메뉴 모음에서 **보기**, **서버 탐색기**를 선택합니다.  
  
2.  **서버 탐색기** 에서 **SharePoint 연결** 노드를 선택한 다음, 메뉴 표시줄에서 **도구**, **SharePoint 연결 추가** 를 선택합니다.  
  
3.  **SharePoint 연결 추가** 상자에서 SharePoint 사이트로 [!INCLUDE[TLA2#tla_url](../sharepoint/includes/tla2sharptla-url-md.md)] 을 입력합니다. \(예: http:\/\/testserver\/sites\/unittests\)  
  
### SharePoint 연결 노드에서 SharePoint 사이트를 삭제하려면  
  
1.  메뉴 모음에서 **보기**, **서버 탐색기** 를 선택하여 **서버 탐색기** 를 엽니다.  
  
2.  **SharePoint 연결** 노드를 확장하여 SharePoint **서버 탐색기** 에서 삭제할 SharePoint 사이트를 표시합니다.  
  
3.  사이트를 선택한 다음 메뉴 모음에서  **편집**, **삭제** 를 선택합니다.  
  
    > [!NOTE]  
    >  이 단계에서 내부 사이트는 삭제되지 않고 연결만 **서버 탐색기** 에서 삭제됩니다.
