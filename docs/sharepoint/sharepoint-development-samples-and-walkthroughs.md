# SharePoint 개발 샘플 및 연습
  이 단원에는 SharePoint 솔루션을 빌드하는 데 사용되는 구문, 구조 및 기술을 보여 주는 샘플 응용 프로그램, 예제 코드 및 단계별 연습이 포함되어 있습니다.  
  
 샘플 코드는 내용 설명을 위한 것이며 실제 배포 솔루션에서 샘플 코드를 사용하려면 수정해야 합니다.  특히 보안 관련 샘플 코드를 사용하려는 경우에는 더욱 주의해야 합니다.
