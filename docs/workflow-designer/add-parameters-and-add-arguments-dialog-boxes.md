# 매개 변수 추가 및 인수 추가 대화 상자
다음 표에서는 <xref:System.Activities.Statements.InvokeMethod> 디자이너에서 사용하는 **매개 변수 추가** 대화 상자의 사용자 인터페이스\(UI\) 요소에 대해 설명합니다.  
  
|||  
|-|-|  
|**방향**|매개 변수가 메서드 안으로, 메서드에서 밖으로 또는 둘 모두의 데이터 흐름을 나타내는지 여부를 지정합니다.|  
|**형식**|새 매개 변수 형식의 형식 이름입니다.|  
|**Value**|기본값을 새 매개 변수에 할당하는 데 사용되는 VB 식입니다.|  
  
 다음 표에서는 <xref:System.Activities.Statements.InvokeDelegate> 디자이너에서 사용하는 **인수 추가** 대화 상자의 사용자 인터페이스\(UI\) 요소에 대해 설명합니다.  
  
|||  
|-|-|  
|**이름**|인수의 이름입니다.|  
|**방향**|인수가 대리자 안으로, 대리자에서 밖으로 또는 둘 모두의 데이터 흐름을 나타내는지 여부를 지정합니다.|  
|**형식**|새 인수의 형식 이름입니다.|  
|**Value**|대리자 인수의 이 인스턴스에 사용할 값입니다.|
