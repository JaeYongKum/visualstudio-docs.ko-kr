# <a name="transaction-activity-designers"></a>트랜잭션 활동 디자이너

트랜잭션 활동 디자이너는 보정 및 확인과 같은 트랜잭션 활동을 만들고 구성하는 데 사용됩니다. 이 섹션에 포함 하 고 다음 표에 나열 된 항목에는 워크플로 디자이너 활동 및 템플릿 디자이너를 사용 하는 방법에 지침을 제공 합니다.
