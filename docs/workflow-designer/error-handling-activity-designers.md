# <a name="error-handling-activity-designers"></a>오류 처리 활동 디자이너

Error Handling 활동 디자이너는 워크플로 디자이너에서 오류 처리를 관리 하는 데 사용 됩니다. 이 단원의 항목에서는 워크플로 디자이너 활동 및 템플릿 디자이너를 사용 하는 방법에 지침을 제공 합니다.
