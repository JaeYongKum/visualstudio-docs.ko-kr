# <a name="workflow-designer-ui-help"></a>워크플로 디자이너 UI 도움말

이 섹션에서는 Workflow designer 상황에 맞는 도움말 항목을 포함합니다. Visual Studio에서 작업에 액세스할 수 있습니다 이러한 항목 사용자 인터페이스에서 키를 눌러 **F1** 워크플로 디자이너 대화 상자가 열려 있습니다.

또한 이 단원에는 탐색 및 오류 메시지에 관한 항목도 나와 있습니다.
