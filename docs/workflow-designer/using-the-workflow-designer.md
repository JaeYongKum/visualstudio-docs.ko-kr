# Workflow Designer 사용
이 단원에서는 기본 제공 디자이너와 몇몇 활동 디자이너에서 호스팅되는 식 편집기를 사용하여 새로운 활동 및 워크플로 프로젝트를 만드는 방법에 대해 설명합니다.

 [방법: 이동 경로 탐색 사용](../workflow-designer/how-to-use-breadcrumb-navigation.md)
