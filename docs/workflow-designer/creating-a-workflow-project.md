# <a name="creating-a-workflow-project"></a>워크플로 프로젝트 만들기

Visual Studio 프로젝트 템플릿을 사용 하 여 워크플로, Windows Communication Foundation (WCF) 워크플로 서비스, 사용자 지정 활동 및 사용자 지정 활동 디자이너를 만들 수 있습니다. 이 섹션에서는 Visual Studio에서 사용할 수 있는 프로젝트 템플릿이 함께 제공 라이브러리 및 응용 프로그램을 만드는 방법을 설명 합니다.

아래의 처음 네 항목에서 찾을 수 있는 워크플로 템플릿을 사용 하는 방법에 설명 된 **새 프로젝트** 창. Visual Studio 2010을 열고 선택 하 여 이러한 서식 파일을 액세스할 수 있습니다 **파일** > **새로** > **프로젝트**, 하면  **워크플로** 에서 범주는 **설치 된 템플릿** 섹션 아래의 왼쪽에는 **Visual Basic** 또는 **Visual C#** 노드.

마지막 항목에서는 워크플로 활동, 디자이너 및 서비스 또는 기타 Visual Studio 항목에는 기존 워크플로 또는 다른 프로젝트를 추가 하는 방법에 설명 합니다.
