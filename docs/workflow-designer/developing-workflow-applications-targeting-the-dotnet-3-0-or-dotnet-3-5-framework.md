# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>.NET 3.0 또는 .NET 3.5 Framework를 대상으로 하는 워크플로 응용 프로그램 개발

Visual Studio 2010은.NET Framework 4를 대상으로 하는 Windows WF (Workflow Foundation) 응용 프로그램을 만드는 Windows 워크플로 디자이너를 제공 합니다. 디자이너는 그래픽을 생성 하 고 디버깅할 워크플로 응용 프로그램에 대 한 친숙 한 Visual Studio 사용자 인터페이스를 사용 합니다.

Visual Studio 2010의 WinFX 또는.NET Framework 버전 3.5 대상으로 사용할 수 있는 레거시 워크플로 디자이너에 액세스할 수도 있습니다. 이 단원에는 이 레거시 디자이너의 사용에 대한 지침이 포함되어 있습니다.
