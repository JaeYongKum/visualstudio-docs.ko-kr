# <a name="additional-resources-for-msbuild"></a>MSBuild를 위한 추가 리소스

## <a name="microsoft-resources"></a>Microsoft 리소스

 [MSBuild Channel 9 웹 사이트](http://go.microsoft.com/fwlink/?LinkId=243092)

 [MSBuild 팀 블로그](http://go.microsoft.com/fwlink/?LinkId=65846)
