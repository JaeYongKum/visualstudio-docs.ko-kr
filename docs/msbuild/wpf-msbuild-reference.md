# <a name="wpf-msbuild-reference"></a>WPF MSBuild 참조
WPF(Windows Presentation Foundation)는 이 섹션에서 설명되는 추가 빌드 지원을 사용하여 Microsoft Build Engine(MSBuild)을 확장합니다.
