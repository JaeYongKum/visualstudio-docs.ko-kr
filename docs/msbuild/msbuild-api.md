# <a name="use-the-msbuild-api"></a>MSBuild API 사용

MSBuild는 프로그램이 빌드를 수행하고 프로젝트를 검사할 수 있도록 공용 API 화면을 제공합니다.

MSBuild API에 대한 설명서는 [Microsoft.Build 네임스페이스](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx)에서 찾을 수 있습니다.
