# How to: Visualize a Collection Association (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

다른 형식의 컬렉션인 속성과 필드는 클래스 다이어그램에서 컬렉션 형식 연결로 표시할 수 있습니다.  일반적인 연결에서는 필드나 속성이 소유 클래스에서 필드 형식으로의 연결선으로 나타나지만 컬렉션 형식 연결에서는 소유 클래스에서 수집한 형식으로의 연결선으로 나타납니다.  
  
### 컬렉션 형식 연결을 만들려면  
  
1.  코드에서 형식 자체가 강력한 형식의 컬렉션인 속성이나 필드를 만듭니다.  
  
2.  클래스 다이어그램에서 클래스를 확장하여 속성과 필드를 표시합니다.  
  
3.  클래스에서 필드나 속성을 마우스 오른쪽 단추로 클릭한 다음 **컬렉션 형식 연결로 표시**를 선택합니다.  
  
     속성 또는 필드가 수집된 형식으로 연결된 연결선으로 나타납니다.
