# 옵션, 텍스트 편집기, 모든 언어
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

이 대화 상자에서는 코드 편집기의 기본 동작을 변경할 수 있습니다.  이러한 설정은 HTML 디자이너의 소스 뷰와 같은 코드 편집기 기반의 다른 편집기에도 영향을 미칩니다.  이 대화 상자를 열려면 **도구** 메뉴에서 **옵션**을 선택합니다.  **텍스트 편집기** 폴더 내에서 **모든 언어** 하위 폴더를 확장한 다음 **일반**을 선택합니다.  
  
> [!CAUTION]
>  이 페이지에서는 모든 개발 언어에 대해 기본 옵션을 설정합니다.  이 대화 상자에서 옵션을 다시 설정하면 모든 언어의 일반 옵션이 여기서 선택한 항목으로 설정됩니다.  한 언어에 대한 텍스트 편집기 옵션을 변경하려면 해당 언어의 하위 폴더를 확장한 다음 해당 옵션 페이지를 선택합니다.  
  
 일부 프로그래밍 언어의 경우 일반 옵션 페이지에서 옵션이 선택된 경우 회색 확인 표시가 나타나지만 다른 언어의 경우 그렇지 않습니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
## 문 완성  
 멤버 목록 자동 표시  
 이 옵션을 선택하면 사용자가 편집기에서 입력할 때 IntelliSense는 사용 가능한 멤버, 속성, 값 또는 메서드의 팝업 목록을 표시합니다.  팝업 목록에서 항목을 선택하여 코드에 삽입합니다.  이 옵션을 선택하면 **고급 멤버 숨기기** 옵션이 활성화됩니다.  
  
 고급 멤버 숨기기  
 이 옵션을 선택하면 팝업 문 완성 목록이 축소되어 가장 일반적으로 사용하는 항목만 표시됩니다.  다른 항목은 목록에서 필터링됩니다.  
  
 매개 변수 정보  
 이 옵션을 선택하면 현재 선언이나 프로시저의 완전한 구문이 모든 사용 가능한 매개 변수와 함께 편집기의 삽입 지점 아래 표시됩니다.  다음에 할당할 수 있는 매개 변수는 굵게 표시됩니다.  
  
## 설정  
 가상 공간 활성화  
 이 옵션을 선택하고 **자동 줄 바꿈**의 선택을 취소하면 코드 편집기에서 줄 끝 뒤에 원하는 위치를 클릭하여 내용을 입력할 수 있습니다.  이 기능을 사용하여 코드 옆의 일정한 지점에 주석을 배치할 수 있습니다.  
  
 자동 줄 바꿈  
 이 옵션을 선택하면 편집기의 표시 가능 영역을 넘어 가로로 연장되는 줄의 모든 부분이 자동으로 다음 줄에 표시됩니다.  또한 이 옵션을 선택하면 **자동 줄 바꿈 시각 문자 모양** 옵션을 사용할 수 있게 됩니다.  
  
> [!NOTE]
>  **자동 줄 바꿈**이 설정된 동안에는 **가상 공간** 기능이 해제됩니다.  
  
 자동 줄 바꿈 시각 문자 표시  
 이 옵션을 선택하면 긴 줄이 두 번째 줄로 넘어가는 곳에 리턴 화살표 표시기가 표시됩니다.  
  
 이러한 표시기를 표시하지 않으려면 이 옵션의 선택을 취소합니다.  
  
> [!NOTE]
>  이러한 알림 화살표는 코드에 추가되거나 인쇄되지 않습니다.  단지 참조하기 위한 것입니다.  
  
 선택 영역이 없는 경우 잘라내기 또는 복사 명령을 빈 줄에 적용  
 이 옵션은 삽입 지점을 빈 줄에 배치하고 아무 것도 선택하지 않은 다음 복사 또는 잘라내기를 선택한 경우 편집기의 동작을 설정합니다.  
  
-   이 옵션을 선택하면 빈 줄이 복사되거나 잘립니다.  그런 다음 붙여넣기를 선택하면 새로운 빈 줄이 삽입됩니다.  
  
-   이 옵션을 해제하면 잘라내기 명령을 통해 빈 줄이 제거됩니다.  그러나 클립보드의 데이터는 보존됩니다.  따라서 이후에 붙여넣기 명령을 사용하면 클립보드로 복사된 최신 내용을 붙여넣게 됩니다.  전에 아무 것도 복사하지 않았으면 아무 것도 붙여넣지 않습니다.  
  
 빈 줄이 아닌 경우에는 이 설정이 복사 또는 잘라내기에 영향을 주지 않습니다.  선택한 내용이 없는 경우에는 줄 전체가 복사되거나 잘립니다.  그런 다음 붙여넣기를 선택하면 전체 텍스트 줄과 줄 끝 문자를 붙여넣습니다.  
  
> [!TIP]
>  공백, 탭 및 줄 끝에 대한 표시기를 표시하여 전체적으로 비어 있는 줄과 들여쓴 줄을 구별하려면 **편집** 메뉴에서 **고급**을 선택한 다음 **공백 보기**를 선택합니다.  
  
## 표시  
 줄 번호  
 이 옵션을 선택하면 각 코드 줄 옆에 줄 번호가 표시됩니다.  
  
> [!NOTE]
>  이러한 줄 번호는 코드에 추가되거나 인쇄되지 않으며  단지 참조하기 위한 것입니다.  
  
 한 번 클릭으로 URL 탐색  
 이 옵션을 선택하면 편집기에서 마우스 커서가 URL을 통과할 때 지시하는 손으로 바뀝니다.  URL을 클릭하면 표시된 페이지가 웹 브라우저에서 열립니다.  
  
 탐색 모음  
 이 옵션을 선택하면 코드 편집기 맨 위에 **탐색 모음**이 표시됩니다.  드롭다운 **개체** 및 **멤버** 목록을 사용하여 코드의 특정 개체를 선택하여 해당 멤버에서 선택한 다음 코드 편집기의 선택된 멤버 선언으로 이동할 수 있습니다.
