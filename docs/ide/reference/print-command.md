# <a name="print-command"></a>인쇄 명령
식을 계산하거나 지정된 텍스트를 표시합니다.

## <a name="syntax"></a>구문

```cmd
Debug.Print text
```

## <a name="arguments"></a>인수
 `text`

 필수. 계산할 식 또는 표시할 텍스트입니다.

## <a name="remarks"></a>설명
 이 명령에 대한 별칭으로 물음표(?)를 사용할 수 있습니다. 따라서 예를 들면

```cmd
>Debug.Print expA
```

 명령을 작성할 수도 있습니다.

```cmd
>? expA
```

 이 명령의 두 버전은 모두 `expA` 식의 현재 값을 반환합니다.

## <a name="example"></a>예

```cmd
>Debug.Print varA
```
