# 방법: 편집기에서 줄 바꿈 관리
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

**자동 줄 바꿈** 옵션을 설정하고 해제할 수 있습니다.  이 옵션을 설정하면 코드 편집기 창의 현재 너비를 넘어가는 긴 줄은 그 일부가 다음 줄에 표시됩니다.  예를 들어, 줄 번호를 효율적으로 사용하기 위해 이 옵션을 해제한 경우에는 오른쪽으로 스크롤해야 긴 줄의 끝을 볼 수 있습니다.  
  
 자세한 내용은 [How to: Set General Editor Options](http://msdn.microsoft.com/ko-kr/704e4a7b-2162-4bed-8a47-f4f6ffec98c2)을 참조하십시오.  
  
> [!NOTE]
>  대화 상자 및 메뉴 명령은 실제 설정이나 버전에 따라 **도움말**에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
## 절차  
  
#### 자동 줄 바꿈의 기본 설정을 설정하려면  
  
1.  **도구** 메뉴에서 **옵션**을 선택합니다.  
  
2.  **텍스트 편집기** 폴더의 **모든 언어** 하위 폴더에서 **일반** 옵션을 선택하여 이 옵션을 전체적으로 설정합니다.  
  
     —또는—  
  
     프로그래밍하고 있는 언어의 하위 폴더에서 **일반** 옵션을 선택합니다.  
  
3.  **설정**에서 **자동 줄 바꿈** 옵션을 선택하거나 해제합니다.  
  
     **자동 줄 바꿈** 옵션을 선택하면 **자동 줄 바꿈 시각 문자 모양** 옵션이 활성화됩니다.  
  
4.  **자동 줄 바꿈 시각 문자 모양** 옵션을 선택하면 긴 줄이 두 번째 줄로 넘어가는 곳에 줄 바꿈 화살표 표시기가 표시됩니다.  표시기 화살표를 표시하지 않으려면 이 옵션의 선택을 취소합니다.  
  
    > [!NOTE]
    >  이러한 알림 화살표는 코드에 추가되지 않으며 표시 용도로만 사용됩니다.
