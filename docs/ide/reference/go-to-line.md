# <a name="go-to-line"></a>줄 이동
이 대화 상자를 사용하면 활성 문서의 특정 줄로 이동할 수 있습니다. 이 대화 상자에 액세스하려면 편집할 문서를 연 후 **편집** 메뉴에서 **이동**을 선택합니다.  
  
 줄 번호(1 - `n`)  
 이동할 활성 문서의 줄 번호를 입력할 수 있습니다. 입력한 숫자는 1과 `n`(현재 문서의 줄 수) 사이여야 합니다.
