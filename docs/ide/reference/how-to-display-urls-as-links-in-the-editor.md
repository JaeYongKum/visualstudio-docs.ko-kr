# 방법: 편집기에서 URL을 링크로 표시
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

코드에 있는 URL이 코드 편집기에서 활성 링크로 처리되도록 할 수 있습니다.  이 기능을 사용하면 URL이 다음과 같이 처리됩니다.  
  
-   밑줄이 표시됩니다.  
  
-   위에 마우스를 올리면 **도구 설명**이 표시됩니다.  
  
-   링크를 **Ctrl** 키를 누른 채 마우스 왼쪽 단추로 클릭하면 지정된 웹 사이트를 열려고 시도합니다.  기본적으로 웹 사이트는 내부 웹 브라우저에 표시됩니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
## 절차  
  
#### URL을 링크로 표시하려면  
  
1.  **도구** 메뉴에서 **옵션**을 클릭합니다.  
  
2.  **텍스트 편집기**를 클릭합니다.  
  
3.  한 언어의 옵션만 변경하려면 해당 언어의 폴더를 확장한 다음 **일반**을 선택합니다.  
  
     — 또는 —  
  
     모든 언어의 옵션을 변경하려면 **모든 언어** 폴더를 확장한 다음 **일반**을 선택합니다.  
  
4.  **표시**에서 **한 번 클릭으로 URL 탐색**을 선택합니다.
