# Resources for Troubleshooting Integrated Development Environment Errors
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

모든 오류 메시지에 구체적인 관련 도움말 항목이 있는 것은 아닙니다.  오류 메시지의 정보가 문제 해결에 도움이 되지 않는 경우 기술 자료 문서, 개발자 커뮤니티 또는 기술 지원 같은 다른 리소스를 참조할 수 있습니다.  
  
## 기술 자료 문서  
 온라인으로 제품 문제와 관련한 KB\(기술 자료\) 문서를 검색할 수 있습니다.  모든 문제에 대해 해당하는 기술 자료 문서가 있는 것은 아니지만 상당수의 고객이 경험하는 오류는 대개 문서화되어 있습니다.  KB 문서에 액세스할 수 있는 [Microsoft 기술 지원 서비스](http://go.microsoft.com/fwlink/?linkid=1782) 웹 사이트.  
  
## 개발자 커뮤니티  
 포럼을 통해 Microsoft 직원은 물론 다른 개발자와도 의견을 교환할 수 있습니다.  해결 방법을 찾을 수 없는 오류가 발생하면 해당 문제에 대한 질문을 포럼에 게시할 수 있습니다.  또한 뉴스 그룹을 검색하여 다른 개발자가 동일한 문제에 대한 내용을 게시했는지 확인할 수 있습니다.  
  
 포럼, 블로그, 채팅 및 기타 리소스에 액세스할 수 있는 [Microsoft 기술 커뮤니티](란?LinkID%20=%20144922) 웹 사이트.  
  
## 기술 지원  
 다른 리소스를 시도한 후에 여전히 질문이 있는 경우 방문 하 여 Microsoft 지원 서비스에 문의할 수 있습니다는 [Microsoft 지원](http://go.microsoft.com/fwlink/?linkid=1782) 웹 사이트.  해당 지역에서 사용 가능한 기술 지원에 대한 자세한 내용은 [의견 보내기](../../ide/talk-to-us.md)를 참조하십시오.
