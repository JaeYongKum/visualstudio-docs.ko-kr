# <a name="set-current-stack-frame-command"></a>현재 스택 프레임 설정 명령
특정 스택 프레임을 설정할 수 있습니다.

## <a name="syntax"></a>구문

```cmd
Debug.SetCurrentStackFrame index
```

## <a name="arguments"></a>인수
 `index`

 필수. 해당 인덱스로 스택 프레임을 선택합니다.

## <a name="example"></a>예

```cmd
>Debug.SetCurrentStackFrame 1
```
