# <a name="useenv-devenvexe"></a>/UseEnv(devenv.exe)
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]를 시작하고 환경 변수를 **VC++ 디렉터리** 대화 상자에 로드합니다.  
  
## <a name="syntax"></a>구문  
  
```  
Devenv /useenv  
```  
  
## <a name="example"></a>예  
 다음 예제에서는 [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]를 시작하고 환경 변수를 **VC++ 디렉터리** 대화 상자에 로드합니다.  
  
```  
Devenv.exe /useenv  
```
