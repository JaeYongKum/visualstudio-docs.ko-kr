# /UseEnv(devenv.exe)
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]를 시작하고 환경 변수를 **VC\+\+ 디렉터리** 대화 상자로 로드합니다.  
  
## 구문  
  
```  
Devenv /useenv  
```  
  
## 예제  
 다음 예제에서는 [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]를 시작하고 환경 변수를 **VC\+\+ 디렉터리** 대화 상자로 로드합니다.  
  
```  
Devenv.exe /useenv  
```
