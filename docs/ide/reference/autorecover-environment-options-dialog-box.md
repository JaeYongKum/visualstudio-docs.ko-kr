# 옵션 대화 상자, 환경, 자동 저장
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

옵션 대화 상자의 이 페이지를 사용하면 파일을 자동으로 백업할지 여부와  예기치 않게 IDE\(통합 개발 환경\)가 종료될 경우 수정된 파일을 복구할지 여부를 지정할 수 있습니다.  **도구** 메뉴, **옵션**, **환경** 폴더, **자동 저장** 페이지를 차례로 선택하여 이 대화 상자에 액세스할 수 있습니다.  이 페이지가 목록에 나타나지 않으면 **옵션** 대화 상자에서 **모든 설정 표시**를 선택합니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 도구 메뉴에서 설정 가져오기 및 내보내기를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
 **자동 저장 간격**  
 편집기에서 파일을 자동으로 저장할 간격을 사용자 지정하려면 이 옵션을 사용합니다.  이전에 저장한 파일의 경우 파일 복사본이 \\...  \\My Documents\\Visual Studio \<*version*\>\\Backup Files\\\<*projectname*\>에 저장됩니다.  파일이 새 파일이고 직접 저장하지 않은 경우에는 임의로 생성된 파일 이름을 사용하여 자동으로 저장됩니다.  
  
 **자동 보관 기간**  
 자동 저장을 위해 만들어진 파일을 보관할 기간을 지정하려면 이 옵션을 사용합니다.
