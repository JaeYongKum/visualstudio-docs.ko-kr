# 옵션, 텍스트 편집기, 파일 확장명
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

이 옵션 대화 상자를 사용하면 특정 파일 확장명을 갖는 모든 파일이 Visual Studio IDE\(통합 개발 환경\)에서 처리되는 방법을 지정할 수 있습니다.  입력하는 각 **확장명**에 대해 편집 환경을 선택할 수 있습니다.  이 대화 상자를 통해 특정 형식의 문서가 열릴 IDE 편집기나 디자이너를 선택할 수 있습니다.  이러한 옵션을 표시하려면 **도구** 메뉴에서 **옵션**을 선택하고 **텍스트 편집기** 노드를 확장한 다음 **파일 확장명**을 선택합니다.  
  
 "인코딩 사용" 옵션을 선택하면 해당 형식의 문서를 열 때마다 해당 문서의 인코딩 체계를 선택할 수 있는 대화 상자가 표시됩니다.  다른 플랫폼에서 사용하거나 다른 대상 언어로 된 프로젝트 문서의 버전을 준비하는 경우 이 대화 상자가 도움이 될 수 있습니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
## UI 요소 목록  
 **확장명**  
 IDE에서 편집 환경을 정의할 파일 확장명을 입력합니다.  
  
 **편집기**  
 이 파일 확장명을 갖는 문서가 열릴 IDE 편집기 또는 디자이너를 선택합니다.  "인코딩 사용" 옵션을 선택하면 해당 문서를 열 때마다 인코딩 체계를 선택할 수 있는 대화 상자가 표시됩니다.  
  
 **Add**  
 지정된 **확장명** 및 **편집 환경**이 포함된 항목을 확장명 목록에 추가합니다.  
  
 **Remove**  
 선택한 항목을 확장명 목록에서 삭제합니다.  
  
 확장명 목록  
 편집 환경이 지정된 모든 확장명 목록을 표시합니다.  
  
 **확장명 없는 파일을 다음에 매핑**  
 확장명 없는 파일이 IDE에서 처리되는 방법을 지정하려는 경우 이 옵션을 선택합니다.  
  
 **확장명 없는 파일 옵션**  
 **편집기**와 같은 목록을 제공합니다.  파일 확장명이 없는 문서가 열릴 IDE 편집기 또는 디자이너를 선택합니다.
