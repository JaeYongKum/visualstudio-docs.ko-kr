# 빌드 플랫폼 이해
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

여러 대상 플랫폼에 적용 되는 솔루션과 프로젝트 속성의 여러 버전을 저장할 수 있습니다.  예를 들어, x86 플랫폼을 대상으로 하는 디버그 구성을 만들고 x64 플랫폼을 대상으로 하는 디버그 구성을 만들 수 있습니다.  여러 구성을 쉽게 빌드할 수 있습니다 있도록 활성 플랫폼을 신속 하 게 변경할 수 있습니다.
