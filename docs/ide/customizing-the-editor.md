# <a name="customize-the-editor"></a>편집기 사용자 지정

텍스트 편집기의 서식, 탭, 글꼴 및 색, IntelliSense 및 기타 여러 기능을 모든 언어 또는 특정 프로그래밍 언어에 맞게 사용자 지정할 수 있습니다. 이 섹션의 항목에서는 이러한 사용자 지정 중 일부를 수행하는 방법을 보여줍니다.

## <a name="options-dialog-box"></a>옵션 대화 상자

텍스트 편집기 옵션에 대한 자세한 내용은 [텍스트 편집기 옵션 대화 상자](../ide/reference/text-editor-options-dialog-box.md)를 참조하세요.
