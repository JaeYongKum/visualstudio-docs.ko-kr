# 도움말 뷰어 창 관리
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

도움말 뷰어에서 창 레이아웃을 사용자 지정할 수 있습니다.  
  
## 도킹 탭  
 도움말 뷰어는 표준 도킹 기능을 지원합니다.  도움말 뷰어에 있는 모든 탭이 도킹돼 있도록 기본 설정이 돼 있지만 다른 위치로 직접 이동하거나 크기를 조정하여 "고정 해제"함으로써 마치 독립된 자식 창처럼 보이게 할 수 있다.  
  
 도움말 뷰어 창의 기본 레이아웃을 복원하려면 **뷰어 옵션** 대화 상자를 연 다음 **재설정** 단추를 선택합니다.  
  
## 새 탭에서 항목 열기  
 탐색 탭에서 항목을 선택한 후 Ctrl\+Enter 키를 선택합니다.  
  
## 탐색 탭 최소화  
 탐색 탭에서 고정 아이콘을 선택하여 항목을 볼 수 있는 공간을 늘립니다.  이러한 탭이 최소화되어 있는 경우 창의 가장 가까운 가장자리에 레이블만 표시됩니다.  탭을 복원하려면 탭의 레이블을 선택하고 고정 아이콘을 다시 선택합니다.
