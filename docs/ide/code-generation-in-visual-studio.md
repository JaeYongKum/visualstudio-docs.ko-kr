# <a name="code-generation-in-visual-studio"></a>Visual Studio에서 코드 생성
Visual Studio는 소스 코드 생성을 지원하여 실제 작업에 집중할 수 있도록 합니다.  몇 가지 일반적인 코드 생성 기능은 다음과 같습니다.  

* 클래스, 메서드, 형식 등 생성
* XML 문서 주석 생성
* 추상 클래스 또는 인터페이스 구현
* 복잡한 식에 지역 변수 도입

코드 생성 논리는 언어 구문과 밀접하게 연관되어 있으므로 Visual Studio의 각 언어 서비스에서는 고유한 코드 생성 기능을 제공합니다. 자세한 내용은 다음 페이지를 참조하세요.  
  
* [코드 생성(C#)](../csharp-ide/code-generation-csharp.md)
* [코드 생성(Visual Basic)](../vb-ide/code-generation-vb.md)

다른 언어에 대한 설명서도 곧 제공될 예정입니다.
