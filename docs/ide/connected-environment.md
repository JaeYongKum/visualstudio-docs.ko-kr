# <a name="internet-resources-used-by-visual-studio"></a>Visual Studio에 사용되는 인터넷 리소스

Visual Studio는 연결된 환경입니다.

- Visual Studio에 [로그인](../ide/signing-in-to-visual-studio.md)하면 사용자 계정 및 이 계정에 링크된 모든 계정에 연결된 모든 서비스에 액세스할 수 있습니다. 계정 설정은 Visual Studio를 사용하는 곳이라면 어디든지 사용자와 로밍됩니다.  
- Visual Studio에서는 SDK처럼 설치의 어떤 구성 요소에든 사용 가능한 업데이트가 있으면 [알려줍니다](../ide/visual-studio-notifications.md).  
- [NuGet](/nuget/) 패키지를 다운로드하여 라이브러리를 얻을 수 있습니다.  
- Visual Studio [확장](../ide/finding-and-using-visual-studio-extensions.md)을 다운로드할 수 있습니다.  
- Azure에 연결할 수 있습니다.  
- 시작 페이지는 다양한 웹 사이트의 최신 개발자 뉴스로 업데이트됩니다.  
- 기타  

이 섹션의 항목은 Visual Studio에서 인터넷 리소스를 사용하는 다양한 방법과 관련이 있습니다.
