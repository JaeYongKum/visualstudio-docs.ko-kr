# <a name="connected-environment"></a>연결된 환경
Visual Studio는 연결된 환경입니다.  

- Visual Studio에 로그인하면 사용자 계정 및 이 계정에 링크된 모든 계정에 연결된 모든 서비스에 액세스할 수 있습니다. 계정 설정은 Visual Studio를 사용하는 곳이라면 어디든지 사용자와 로밍됩니다.  
- Visual Studio에서는 SDK처럼 설치의 어떤 구성 요소에든 사용 가능한 업데이트가 있으면 알려줍니다.  
- NuGet 패키지 및 확장을 다운로드할 수 있습니다.  
- 기타...
