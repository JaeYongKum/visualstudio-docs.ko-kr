# 방법: 편집기에서 텍스트의 대/소문자 변경
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

메뉴 명령이나 바로 가기 키를 사용하여 선택된 텍스트의 대\/소문자를 모두 대문자로 또는 모두 소문자로 변환할 수 있습니다.  
  
> [!NOTE]
>  대화 상자 및 메뉴 명령은 실제 설정이나 버전에 따라 **도움말**에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
## 절차  
  
#### 텍스트를 대문자로 변환하려면  
  
1.  변환하려는 텍스트를 선택합니다.  
  
2.  **편집** 메뉴에서 **고급**을 선택합니다.  
  
3.  텍스트를 모두 대문자로 변환하려면 **대문자로**를 선택하거나 **Ctrl**\+**Shift**\+**U**를 누릅니다.  
  
     —또는—  
  
     텍스트를 모두 소문자로 변환하려면 **소문자로**를 선택하거나 **Ctrl**\+**U**를 누릅니다.  
  
    > [!TIP]
    >  이 변경 작업 이전의 대\/소문자 서식으로 되돌리려면 **편집** 메뉴에서 **실행 취소**를 선택합니다.
