# How to: Add Comments to Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

메모 모양을 사용하여 클래스 다이어그램에 메모를 추가할 수 있습니다.  메모 모양의 속성은 **Text** 하나뿐이며 여기에 텍스트를 입력할 수 있습니다.  메모 모양은 다이어그램 화면에만 있으며 코드에는 없습니다.  
  
 메모는 **클래스 디자이너**의 클래스 다이어그램뷰에 있습니다. 동일한 프로젝트에서 두 번째 클래스 다이어그램을 열면 첫 번째 뷰에서 만든 메모가 표시되지 않습니다.  다이어그램을 삭제할 경우 포함된 모든 메모도 함께 삭제됩니다.  
  
 메모 모양의 크기는 변경할 수 있으나 배경색, 글꼴 또는 글꼴 크기 등은 변경할 수 없습니다.  
  
### 메모를 추가하려면  
  
1.  **클래스 디자이너 도구 상자**에서 메모를 클래스 다이어그램으로 끌어서 놓습니다.  
  
2.  다이어그램에서 새 메모 모양을 클릭하고 원하는 텍스트를 입력합니다.
