# <a name="idiasectioncontribgetaddressoffset"></a>IDiaSectionContrib::get_addressOffset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [idiasectioncontrib:: Get_addressoffset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasectioncontrib-get-addressoffset)합니다.  
  
오프셋된 부분에 대 한 기여도 주소를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp#  
HRESULT get_addressOffset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pRetVal`  
 [out] 오프셋된에 대 한 기여도 주소 부분을 반환합니다.  
  
## <a name="return-value"></a>반환 값  
 성공 하면 반환 `S_OK`합니다. 반환 `S_FALSE` 경우이 속성이 지원 되지 않습니다. 그러지 않으면 오류 코드가 반환됩니다.
