# <a name="idiasymbolgetnumberofregisterindices"></a>IDiaSymbol::get_numberOfRegisterIndices
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [IDiaSymbol::get_numberOfRegisterIndices](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-numberofregisterindices)합니다.  
  
등록 하는 인덱스 수를 검색 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT get_numberOfRegisterIndices(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pRetVal`  
 [out] 에 대 한 포인터를 `DWORD` 등록 인덱스의 수를 보유 하는 합니다.
