# <a name="idiasegmentgetaddresssection"></a>IDiaSegment::get_addressSection

이 항목의 최신 버전에서 찾을 수 있습니다 [idiasegment:: Get_addresssection](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasegment-get-addresssection)합니다.  
  
이 세그먼트에 매핑되는 섹션 번호를 검색 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp#  
HRESULT get_addressSection (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pRetVal`  
 [out] 이 세그먼트에 매핑되는 섹션 번호를 반환 합니다.  
  
## <a name="return-value"></a>반환 값  
 성공 하면 반환 `S_OK`합니다. 반환 `S_FALSE` 경우이 속성이 지원 되지 않습니다. 그러지 않으면 오류 코드가 반환됩니다.
