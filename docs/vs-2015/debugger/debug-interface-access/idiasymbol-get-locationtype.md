# <a name="idiasymbolgetlocationtype"></a>IDiaSymbol::get_locationType
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [idiasymbol:: Get_locationtype](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-locationtype)합니다.  
  
데이터 기호 위치 형식을 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp#  
HRESULT get_locationType (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pRetVal`  
 [out] 값을 반환 합니다 [LocationType 열거형](../../debugger/debug-interface-access/locationtype.md) 와 같은 위치에 대 한 형태의 데이터 기호를 지정 하는 열거형 `static` 또는 `local`합니다.  
  
## <a name="return-value"></a>반환 값  
 성공 하면 반환 `S_OK`이 고, 그렇지 않으면 반환 `S_FALSE` 또는 오류 코드입니다.  
  
> [!NOTE]
>  반환 값이 `S_FALSE` 속성 기호를 사용할 수 없는 것을 의미 합니다.
