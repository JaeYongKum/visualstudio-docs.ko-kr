# <a name="c6993"></a>C6993
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 6993: 코드 분석에서 OpenMP 구문을 무시합니다. 단일 스레드 코드를 분석합니다.  
  
 이 경고는 코드 분석기에서 분석할 수 없는 OpenMP pragma가 발생한 것을 나타냅니다.
