이 항목의 최신 버전에서 찾을 수 있습니다 [C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227)합니다.  
  
경고 C28227: 주석이 PrimOp 일 수 없습니다 (이전 선언 참조)  
  
 이 경고는 명명 된 주석을 일반를 주석으로 선언 되도 이전에 선언 되었습니다 PrimOp를 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
