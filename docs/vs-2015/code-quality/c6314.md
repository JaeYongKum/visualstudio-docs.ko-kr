# <a name="c6314"></a>C6314
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C6314: 연산 순서가 잘못 되었습니다: 비트-했거나 조건식 연산자 보다 우선 순위가 높습니다. 의도 명확히 하려면 괄호를 추가 합니다.  
  
 이 메시지는 나타냅니다 비트를 포함 하는 식-or 연산자 (`|`)의 조건부 작업을 테스트 식 검색 되었습니다 (`?:`).  
  
 조건부 연산자는 비트 연산자 보다 우선 순위가 낮습니다. 테스트 식에 비트 포함 해야 하는 경우-또는 연산자를 괄호로 조건식 주위에 추가 해야 합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
int SystemState();  
  
int f(int SignalValue)  
{  
  return SystemState() | (SignalValue != 0) ? 1 : 0;  
}  
```  
  
 이 경고를 해결 하려면 다음 코드를 사용 합니다.  
  
```  
int SystemState();  
  
int f(int SignalValue)  
{  
  return SystemState() | ((SignalValue != 0) ? 1 : 0);  
}   
```
