# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

c28211: SAL_context에 대해 정적 컨텍스트 이름이  
  
 이 경고의 피연산자는 `_Static_context_` 주석 도구 정의한 컨텍스트 이름 이어야 합니다. 일반적인 사용의 경우 이 경고가 발생하지 않습니다.
