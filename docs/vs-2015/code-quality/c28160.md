# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28160: 주석 오류  
  
 이 경고는 `__drv_error` 주석이 발생했을 때 보고됩니다. 이 주석은 수정이 필요한 코딩 방식을 표시하는 데 사용되며, `__drv_when` 주석과 함께 사용해서 매개 변수의 특정 조합을 나타낼 수 있습니다.
