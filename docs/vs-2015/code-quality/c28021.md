# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C28021](https://docs.microsoft.com/visualstudio/code-quality/c28021)합니다.  
  
경고 C28021: 매개 변수 \<매개 변수 >으로 주석이 달린 \<anno >는 포인터 여야 합니다.  
  
 이 경고는 주석 처리 중인 개체가 포인터 형식이 아닐 때 보고됩니다. 이 주석은 `void` 또는 정수 계열 형식과 함께 사용할 수 없습니다.
