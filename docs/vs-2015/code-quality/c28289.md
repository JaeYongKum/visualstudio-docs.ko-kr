# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28289: 함수의 경우: ReadableTo 또는 WritableTo에 limit-spec가 매개 변수로 포함되지 않았습니다.  
  
 코드 분석 도구는 함수/매개 변수 주석이 언급된 것처럼 잘못 코딩된 경우 이 경고를 보고합니다.
