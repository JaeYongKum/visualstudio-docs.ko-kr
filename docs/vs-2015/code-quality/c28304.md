경고 C28304: c + + 참조 매개 변수 < 매개 변수 _ 이름 >, 부적절 하 게 배치 `_Notref_` 연산자를 적용할 찾을 \<토큰 >.  
  
 `_Notref_` 연산자는 C++ 참조 매개 변수를 포함하는 특수한 경우 및 시스템에서 제공하는 매크로에서만 사용해야 합니다. 이 연산자 바로 다음에는 `_Deref_` 연산자 또는 기능 주석이 와야 합니다.
