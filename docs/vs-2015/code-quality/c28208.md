# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28208: 함수 \<함수 >에서 다른 매개 변수 목록을 사용 하 여 이전에 정의한 \<파일 > (\<줄 >). 일부 분석 도구는 잘못 된 결과가 생성 됩니다.  
  
 이 경고는 함수의 알려진 정의가 다른 정의와 일치하지 않을 때 보고됩니다.
