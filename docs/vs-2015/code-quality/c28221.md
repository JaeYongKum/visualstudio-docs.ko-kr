# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C28221](https://docs.microsoft.com/visualstudio/code-quality/c28221)합니다.  
  
경고 C28221: 주석에 있는 매개 변수에 문자열 식이 필요  
  
 이 경고는 주석의 매개 변수가 문자열이어야 하는데 다른 형식이 발견되었음을 나타냅니다. 이 경고는 대개 잘못 코딩된 주석 매크로를 나타내며 일반적인 사용의 경우 표시되지 않습니다.
