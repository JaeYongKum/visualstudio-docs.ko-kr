# <a name="c6295"></a>C6295
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C6295](https://docs.microsoft.com/visualstudio/code-quality/c6295)합니다.  
  
경고 C6295: for 루프가: \<변수 >의 "max" "min" 범위 값은입니다. 무한 루프가 발생  
  
 이 경고는 for 루프 의도 한 대로 작동 하지 않을 수 있습니다를 나타냅니다. for 루프는 >=를 사용하여 부호 없는 값을 0에 대해 테스트합니다. 결과 항상 true, 이므로 루프 한정 되지 않습니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
void f( )  
{  
  for (unsigned int i = 100; i >= 0; i--)   
  {  
    // code ...  
  }  
}  
```  
  
 이 경고를 해결 하려면 다음 코드를 사용 합니다.  
  
```  
void f( )  
{  
   for (unsigned int i = 100; i > 0; i--)  
   {  
      // code ...  
   }  
}  
```
