# <a name="ca0053"></a>CA0053

규칙 어셈블리를 로드할 수 없습니다.

이 나타냅니다 리플렉션에 의해 규칙 어셈블리를 로드할 수 없습니다. 사용자에 게 사용자 지정 규칙을 올바르게 빌드되지 않은 경우 발생할 수 있습니다 또는 파일 시스템 액세스 거부와 같은 분석 도구는 다른 이유로 어셈블리를 로드할 수 없습니다.
