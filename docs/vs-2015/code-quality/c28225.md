# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C28225](https://docs.microsoft.com/visualstudio/code-quality/c28225)합니다.  
  
경고 C28225: 주석에서 올바른 필수 매개 변수 개수를 찾지 못했습니다  
  
 이 경고는 잘못 된 개수의 매개 변수를 사용 하 여 명명 된 주석을 사용 됨을 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
