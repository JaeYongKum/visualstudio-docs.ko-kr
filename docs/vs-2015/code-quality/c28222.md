# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C28222](https://docs.microsoft.com/visualstudio/code-quality/c28222)합니다.  
  
경고 28222: (_y).\_, (_n)\_, 또는 _maybe가\_ 주석에 대 한 필요 합니다.  
  
 이 경고는 주석의 매개 변수로 `_Yes_`, `_No_` 또는 `_Maybe_` 기호 중 하나가 필요한데 다른 기호가 발견되었음을 나타냅니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다.
