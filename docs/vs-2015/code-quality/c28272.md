# <a name="c28272"></a>C28272
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C28272](https://docs.microsoft.com/visualstudio/code-quality/c28272)합니다.  
  
경고 C28272: 함수, 매개 변수에 대 한 주석이 함수 선언과 일치 하지 않습니다.  
  
 이 경고는 분석되고 있는 코드의 오류가 아니라 주석의 오류를 나타냅니다. 함수 정의에 나타나는 주석은 선언에 나타나는 주석과 일치하지 않습니다. 두 주석은 일치를 위해 확인되어야 합니다.
