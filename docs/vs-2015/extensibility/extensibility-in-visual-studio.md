Visual Studio에서는 다양한 확장성 옵션이 제공됩니다. 사용자 고유의 Sdk를 만드는 하 고, Windows, Windows Phone 및 Azure Sdk (Visual Studio의 일부로 설치 되어 있음)를 사용 하 고, Visual Studio 자체를 확장할 수 있습니다.
