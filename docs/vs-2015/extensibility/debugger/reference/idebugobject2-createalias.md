# <a name="idebugobject2createalias"></a>IDebugObject2::CreateAlias
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [IDebugObject2::CreateAlias](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugobject2-createalias)합니다.  
  
고유 ID 또는이 개체에 대 한 별칭을 만들거나 기존 별칭을 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT CreateAlias(  
   IDebugAlias** ppAlias  
);  
```  
  
```csharp  
int CreateAlias(  
   out IDebugAlias ppAlias  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `ppAlias`  
 [out] 새로운 (또는 기존) 별칭입니다.  
  
## <a name="return-value"></a>반환 값  
 성공 하면 S_OK를 반환 합니다. 그렇지 않으면 오류 코드를 반환합니다.  
  
## <a name="remarks"></a>설명  
 별칭은 메모리에 개체는 특정 개체를 나타내는 레이블입니다.
