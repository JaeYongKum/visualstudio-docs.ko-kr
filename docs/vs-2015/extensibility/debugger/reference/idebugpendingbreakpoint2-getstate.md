# <a name="idebugpendingbreakpoint2getstate"></a>IDebugPendingBreakpoint2::GetState

이 항목의 최신 버전에서 찾을 수 있습니다 [IDebugPendingBreakpoint2::GetState](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugpendingbreakpoint2-getstate)합니다.  
  
보류 중인 중단점의 상태를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp#  
HRESULT GetState(   
   PENDING_BP_STATE_INFO* pState  
);  
```  
  
```csharp  
int GetState(   
   PENDING_BP_STATE_INFO[] pState  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pState`
