# <a name="idebugarrayfieldgetelementtype"></a>IDebugArrayField::GetElementType

이 항목의 최신 버전에서 찾을 수 있습니다 [IDebugArrayField::GetElementType](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugarrayfield-getelementtype)합니다.  
  
배열의 요소 형식을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp#  
HRESULT GetElementType(   
   IDebugField** ppType  
);  
```  
  
```csharp  
int GetElementType(  
   out IDebugField ppType  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `ppType`
