# <a name="idebugaliasdispose"></a>IDebugAlias::Dispose

이 항목의 최신 버전에서 찾을 수 있습니다 [IDebugAlias::Dispose](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugalias-dispose)합니다.  
  
제거에 대 한이 별칭을 표시합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Dispose();  
```  
  
```csharp  
int Dispose();  
```  
  
#### <a name="parameters"></a>매개 변수  
 없음

 성공 하면 S_OK를 반환 합니다. 그렇지 않으면 오류 코드를 반환합니다.
