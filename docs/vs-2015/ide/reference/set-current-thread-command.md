# <a name="set-current-thread-command"></a>현재 스레드 설정 명령
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [현재 스레드 명령을 설정](https://docs.microsoft.com/visualstudio/ide/reference/set-current-thread-command)합니다.  
  
  
지정한 스레드를 현재 스레드로 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
Debug.SetCurrentThread index  
```  
  
## <a name="arguments"></a>인수  
 `index`  
 필수. 해당 인덱스로 스레드를 선택합니다.  
  
## <a name="example"></a>예제  
  
```  
>Debug.SetCurrentThread 1  
```
