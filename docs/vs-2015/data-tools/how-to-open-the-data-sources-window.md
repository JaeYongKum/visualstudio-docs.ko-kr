# <a name="how-to-open-the-data-sources-window"></a>방법: 데이터 소스 창 열기
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

합니다 **데이터 원본** 프로젝트에서 데이터 원본 창에 표시 됩니다. 사용할 합니다 **데이터 원본** 창에서 프로젝트의 폼으로 항목을 끌어 데이터 바인딩된 컨트롤의 사용자 인터페이스를 만들 창입니다. 참조 [Visual Studio에서 데이터 바인딩 Windows Forms 컨트롤](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md)합니다.  
  
 채울 수 있습니다 합니다 **데이터 원본** 실행 하 여 창 합니다 **데이터 소스 구성** 마법사. 자세한 내용은 참조 [데이터 소스 구성 마법사](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f)합니다.  
  
### <a name="to-open-the-data-sources-window"></a>데이터 소스 창을 열려면  
  
-   메뉴 모음에서 **뷰**, **기타 Windows**를 **데이터 원본을** (또는 Shift + Alt + D 키 선택).  
  
    > [!IMPORTANT]
    >  프로젝트가 데이터 소스 만들기 및 사용을 지원할 경우에만 데이터 소스를 프로젝트에 추가할 수 있습니다. 예를 들어, 열 수 없습니다는 **데이터 원본** 창 Windows 스토어 앱 용 프로젝트에서입니다.
