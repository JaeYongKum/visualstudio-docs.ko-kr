# <a name="how-to-start-the-tableadapter-configuration-wizard"></a>방법: TableAdapter 구성 마법사 시작
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

합니다 **TableAdapter 구성 마법사** 만들고 강력한 형식의 데이터 집합에서 Tableadapter를 편집 합니다. 이 마법사는 사용자가 마법사에 입력하는 SQL 문 또는 데이터베이스의 기존 저장 프로시저를 기반으로 TableAdapter를 만듭니다. 또한 사용자가 마법사에 입력하는 SQL 문을 기반으로 데이터베이스에 새 저장 프로시저를 만들 수도 있습니다.  
  
### <a name="to-start-the-tableadapter-configuration-wizard-to-create-a-new-tableadapter"></a>TableAdapter 구성 마법사를 시작하여 새 TableAdapter를 만들려면  
  
1.  데이터 집합을 엽니다는 **데이터 집합 디자이너**합니다. 자세한 내용은 [방법: 데이터 집합 디자이너에서 데이터 집합 열기](http://msdn.microsoft.com/library/36fc266f-365b-42cb-aebb-c993dc2c47c3)합니다.  
  
    > [!NOTE]
    >  참조 프로젝트에서 데이터 집합 없으면 [만들기 및 데이터 집합 구성](../data-tools/create-and-configure-datasets-in-visual-studio.md)합니다.  
  
2.  끌어 새 TableAdapter를 만드는 경우를 **TableAdapter** 에서 개체를 **데이터 집합** 탭의 **도구 상자** 에 **데이터 집합 디자이너**.  
  
3.  에 **데이터 연결 선택** 페이지에서 현재 사용 가능한 연결 목록에서 데이터 연결을 선택 하거나 선택 **새 연결** 새 연결을 만듭니다.  
  
### <a name="to-start-the-tableadapter-configuration-wizard-to-edit-an-existing-tableadapter"></a>TableAdapter 구성 마법사를 시작하여 기존 TableAdapter를 편집하려면  
  
1.  데이터 집합을 엽니다는 **데이터 집합 디자이너**합니다. 자세한 내용은 [방법: 데이터 집합 디자이너에서 데이터 집합 열기](http://msdn.microsoft.com/library/36fc266f-365b-42cb-aebb-c993dc2c47c3)합니다.  
  
2.  TableAdapter를 마우스 오른쪽 단추로 클릭 합니다 **데이터 집합 디자이너** 선택한 **구성**합니다. 마법사가 열립니다는 **SQL 문 생성** 페이지 또는 **기존 저장 프로시저에 명령 바인딩** TableAdapter를 원래 구성한 방식에 따라 페이지입니다.  
  
3.  마법사를 완료합니다.
