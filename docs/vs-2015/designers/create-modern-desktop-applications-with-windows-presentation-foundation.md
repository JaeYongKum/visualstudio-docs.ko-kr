# <a name="create-modern-desktop-applications-with-windows-presentation-foundation"></a>Windows Presentation Foundation으로 최신 데스크톱 응용 프로그램 만들기
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 2015의 WPF(Windows Presentation Foundation)는 Windows에서 최신 LOB(기간 업무) 데스크톱 응용 프로그램을 빌드할 수 있는 통합된 프로그래밍 모델을 개발자에게 제공합니다.

 [Visual Studio 및 Blend for Visual Studio에서 XAML 디자인](../designers/designing-xaml-in-visual-studio.md)  
  
 최신 버전의 .NET Framework에 포함된 WPF는 기본적으로 버전 4.0에서 변경되지 않았습니다. 버전 4.0에 대한 설명서가 현재 버전에도 적용됩니다.
