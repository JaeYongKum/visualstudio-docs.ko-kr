# <a name="activating-the-pluralsight-training-benefit-in-visual-studio-subscriptions"></a>Visual Studio 구독에서 Pluralsight 교육 혜택 활성화

Pluralsight 및 Code School 카탈로그 전체를 활용하여 자신의 실력을 높이세요!  전문가가 만든 과정, 학습 경로, 적응형 기술 평가 등을 이용할 수 있습니다.  혜택은 보유한 구독 유형에 따라 다릅니다.  

| 구독                                                     | 이점                              |
|------------------------------------------------------------------|--------------------------------------|
| Visual Studio Enterprise 표준 및 연간 클라우드 구독   | 연간 구독 전체             |
| Visual Studio Professional 표준 및 연간 클라우드 구독 | 40개 선택 과정 및 12개월 액세스 |
| Visual Studio Test Professional                                  | 40개 선택 과정 및 12개월 액세스 |
| MSDN 플랫폼                                                   | 40개 선택 과정 및 12개월 액세스 |
| Visual Studio Dev Essentials                                     | 전체 카탈로그 및 3개월 액세스  |  

  

1. 구독을 활성화하려면 Pluralsight 혜택 타일에서 **코드 가져오기** 링크를 클릭합니다.

![Pluralsight 혜택 타일](_img\vs-pluralsight\vs-pluralsight-tile.png)

2. 쿠폰 코드가 성공적으로 검색되었다는 알림을 받게 됩니다.  **활성화**를 클릭합니다.   

![Pluralsight 혜택 방문 페이지](_img\vs-pluralsight\vs-pluralsight-landing-crop.png)

3.  활성화 페이지에서
- 양식에 요청받은 연락처 정보를 제공합니다. 
- 확인란을 클릭하여 사용 약관 및 개인정보처리방침에 동의합니다.
- **혜택 활성화에 동의함**을 클릭합니다. 

![Pluralsight 혜택 활성화](_img\vs-pluralsight\vs-pluralsight-registration-crop.png)

4.  정말 간단하죠.  학습하려면 언제든지 [Pluralsight](https://app.pluralsight.com/id?)에 연결하고 로그인하세요.  

![Pluralsight 혜택 로그인](_img\vs-pluralsight\vs-pluralsight-sign-in.png)
