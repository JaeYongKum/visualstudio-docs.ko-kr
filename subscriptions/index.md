# <a name="visual-studio-subscriptions"></a>Visual Studio 구독
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>개발자 도구, 클라우드 서비스, 소프트웨어, 지원 및 교육

Visual Studio 구독은 Windows, iOS, Android, Linux 등을 비롯한 기본 플랫폼 및 장치에서 유용한 응용 프로그램을 만들고 배포하고 관리하는 데 필요한 가장 포괄적인 리소스 집합입니다.

# <a name="welcome-to-subscriptions"></a>구독에 오신 것을 환영합니다.

구독자이든 회사를 위해 구독을 관리하든, Visual Studio 구독을 최대한 활용하는 데 도움이 되는 정보를 찾을 수 있습니다.
