# <a name="azure-devops-services-benefits-for-visual-studio-subscribers"></a>Visual Studio 구독자용 Azure DevOps Services 혜택

활동적인 Visual Studio 구독자로서 구독 혜택에 포함된 Azure DevOps Services의 다양한 기능을 사용할 수 있습니다. 계정을 만들었든 또는 다른 사용자에 의해 추가되었든 멤버로서 각 Azure DevOps Services 계정에서 이러한 동일한 기능을 사용할 수 있습니다.

Visual Studio 구독을 활성화하는 데 사용한 것과 동일한 ID 또는 [대체 ID](vs-alternate-identity.md)를 사용하여 Azure DevOps Services에 로그인하면 자동으로 인식됩니다.

[무료 Azure DevOps Services 계정에 등록](https://visualstudio.microsoft.com/team-services/)

## <a name="eligibility"></a>자격

| 구독 수준                                                 |     채널                                            | 이점                                                          | 갱신 가능?    |
|--------------------------------------------------------------------|---------------------------------------------------------|------------------------------------------------------------------|---------------|
| Visual Studio Enterprise(표준, 연간 클라우드, 월간 클라우드)   | VL, Azure, 일반 정품, 선택한 NFR<sup>1</sup>, FTE  | 기본, 테스트 관리자, 패키지 관리, 자체 호스트된 CI/CD       |  예          |
| Visual Studio Professional(표준, 연간 클라우드, 월간 클라우드) | VL, Azure, 일반 정품,                                        | Basic                                                            |  예          |
| Visual Studio Test Professional(표준)                         | VL, 일반 정품                                              | 기본, 테스트 관리자                                              |  예          |
| MSDN 플랫폼(표준)                                          | VL, 일반 정품                                              | 기본, 테스트 관리자                                              |  예          |

<sup>1</sup> *포함: NFR(전매금지), MVP(Microsoft Valued Partner), RD(Region Director), VSIP(Visual Studio 업계 파트너), Microsoft 파트너 네트워크(Enterprise), BizSpark, MCT Software & Services Developer. 제외: MCT Software & Services, Imagine.*

다음과 같은 Azure DevOps Services 기능에 대해 자세히 알아봅니다.

- [기본](https://visualstudio.microsoft.com/team-services/compare-features/)
- [테스트 관리자](https://marketplace.visualstudio.com/items?itemName=ms.vss-testmanager-web)
- [패키지 관리](https://marketplace.visualstudio.com/items?itemName=ms.feed)

어떤 구독을 사용하고 있는지 확실하지 않나요?  자신에게 할당된 모든 구독을 보려면 [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs)에 연결합니다.
일부 구독이 표시되지 않으면 하나 이상이 다른 전자 메일 주소에 할당되어 있을 수 있습니다.  해당 구독을 보려면 해당 전자 메일 주소로 로그인해야 합니다.

## <a name="support-resources"></a>지원 리소스

- Visual Studio 구독에 대한 판매, 구독, 계정 및 요금 청구에 대한 지원을 받으려면 Visual Studio [구독 지원](https://visualstudio.microsoft.com/subscriptions/support/)에 문의하세요.
- Visual Studio IDE, Azure DevOps 또는 기타 Visual Studio 제품이나 서비스와 관련하여 궁금한 점이 있나요?  [Visual Studio 지원](https://visualstudio.microsoft.com/support/)을 참조하세요.
- [Azure DevOps 설명서](/azure/devops/)를 참조하세요.
