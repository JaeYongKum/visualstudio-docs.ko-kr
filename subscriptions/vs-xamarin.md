# <a name="xamarin-university-training-benefit-in-visual-studio-subscriptions"></a>Visual Studio 구독의 Xamarin University 교육 혜택

녹화된 과정 및 게스트 강의를 사용하여 C#에서 네이티브 iOS 및 Android 앱을 빌드합니다.  
-  Visual Studio Dev Essentials용 Xamarin University 모바일 교육에서는 12개의 과정 및 3개의 게스트 강의에 액세스할 수 있습니다. 
-  Visual Studio Professional 및 Visual Studio Enterprise 구독용 Xamarin University 모바일 교육에는 Visual Studio Dev Essentials가 제공하는 것과 동일한 과정 및 강의와 함께 18개의 추가 과정 및 6개의 추가 강의가 포함됩니다.

## <a name="activation-steps"></a>활성화 단계
1.  [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)에 로그인합니다. 
2.  전문적인 개발 섹션에서 Xamarin University 교육 타일을 찾고 혜택 타일의 왼쪽 아래에 있는 **코드 얻기** 링크를 클릭합니다.   쿠폰 코드가 성공적으로 검색되었다는 알림을 받게 됩니다.  **활성화**를 클릭하여 계속합니다.

    ![Xamarin University 교육 혜택 타일](_img\vs-xamarin\vs-xamarin-tile.png)

3.  등록 페이지에서 연락처 정보를 완성하고, 확인란을 클릭하여 사용 약관 및 개인정보처리방침에 동의한 다음, **혜택 활성화**를 클릭합니다. 

    ![Xamarin University 교육 혜택 등록](_img\vs-xamarin\vs-xamarin-registration-resized.png)
4.  이것이 전부입니다!  이제 Xamarin University 교육 혜택을 활성화했으므로 Microsoft Visual Studio용 Xamarin University 모바일 교육 페이지로 리디렉션되고 학습을 시작할 수 있습니다.  

    ![Xamarin University 교육 혜택 홈 페이지](_img\vs-xamarin\vs-xamarin-home-resized.png)

어떤 구독을 사용하고 있는지 확실하지 않나요?  자신의 이메일 주소에 할당된 모든 구독을 보려면 [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs)에 연결합니다. 일부 구독이 표시되지 않으면 하나 이상이 다른 전자 메일 주소에 할당되어 있을 수 있습니다.  해당 구독을 보려면 해당 전자 메일 주소로 로그인해야 합니다. 


## <a name="frequently-asked-questions"></a>질문과 대답
### <a name="q--what-topics-are-covered-in-the-classes-and-lectures"></a>Q: 수업 및 강의에서 다루는 주제는 무엇인가요?
A: https://www.xamarin.com/university/mobile-training에 두 제안에 대한 강의 및 강좌의 전체 목록이 있습니다. 

### <a name="q-im-already-a-xamarin-university-subscriber-does-this-affect-my-xamarin-university-mobile-training-benefit"></a>Q: 이미 Xamarin University 구독자입니다. 내 Xamarin University 모바일 교육 혜택에 영향을 주나요?
A: 아니요. 기존 Xamarin University 구독자는 university.xamarin.com에서 로그인하여 라이브 과정, 자료 및 게스트 강의를 포함하여 Xamarin University 연간 구독에 완전히 액세스할 수 있습니다.  
Visual Studio Dev Essentials 프로그램 및 Visual Studio 구독의 혜택으로 포함된 과정은 Xamarin University 구독과 구분되고 university.xamarin.com이 아니라 Visual Studio Dev Essentials 또는 Visual Studio 포털에서 액세스합니다.  Visual Studio Dev Essentials 프로그램 및 Visual Studio 구독에 포함된 Xamarin University 모바일 교육 과정 녹화 자료는 연간 Xamarin University 구독에 포함되지만 프로그램 멤버 자격 또는 Visual Studio 구독이 유효하면 계속 사용할 수 있습니다. Xamarin University 구독 상태와 무관하게 Visual Studio Dev Essentials 또는 Visual Studio 포털에서 계속 액세스할 수 있습니다.

### <a name="q--can-i-become-a-certified-xamarin-developer-with-the-xamarin-university-included-in-visual-studio-dev-essentials-what-about-through-visual-studio-subscriptions"></a>Q: Visual Studio Dev Essentials에 포함된 Xamarin University를 사용하여 인증된 Xamarin 개발자가 될 수 있나요? Visual Studio 구독을 사용하는 것은 어떨까요?
A: Visual Studio Dev Essentials 및 Visual Studio/MSDN 구독에 Xamarin University와 함께 포함된 과정 및 자료는 인증에 유효하지 않습니다. Xamarin 인증을 획득하는 데 필요한 자료, 과정 및 녹화 자료는 전체 Xamarin University 구독에만 포함됩니다.
